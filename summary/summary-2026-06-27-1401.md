## 一、增伤理论

**论点：DNF 70版本的增伤体系基于"乘区隔离 + 阈值激活"双层设计。策划通过将不同增伤类型分配到独立乘区（力量/物理攻击/独立攻击/暴击），构建了装备收益的不可替代性——同一件装备的不同属性分别进入不同乘区，使得单件装备的总收益不是线性可比的，而是乘区拓扑位置的函数。**

70版本装备遵循清晰的分类隔离：CC套为代表的通用基础属性装提供面板增益（力量/攻击/暴击）；异界套装提供特定技能加成（技能等级+%或技能范围+%）；粉装/史诗则提供稀有词条（如独立攻击%）。不同类型装备在伤害公式中占据不同乘区，互相不可替代。

**案例1：CC套的乘区拓扑**
CC套6件提供4种属性——力量（基础乘区）、物理攻击（百分比乘区）、独立攻击（固伤乘区）、暴击（暴击乘区）。这4个属性分别进入伤害公式的3个不同乘区，且对固伤和百分比流派的收益完全不对称（+310力量对固伤零收益）。这种设计迫使玩家根据自身输出类型选择装备，而非简单"堆数值"。

**案例2：异界套的阈值触发模型**
E2血之狂暴6件套：技能+35%、出血+10%、力量+5%——仅当集齐6件才激活套装效果。这套"阈值触发"设计让散搭玩家永远无法达到套装收益，迫使玩家在"套装完整性"和"散搭灵活性"间做取舍，是DNF装备设计最核心的经济学机制之一。

## 二、冷却压缩理论

**论点：70版本冷却缩减资源极度稀缺，策划有意将冷却压缩控制在小幅可控范围内。冷却缩减的稀缺性设计意图在于维持技能循环的"节奏感"——不让任何职业能通过装备实现无限连招，同时保证冷却收益不会稀释到可有可无。**

该分类数据不充分，仅从已有样本推断：现有研究仓库主要聚焦于加成数值而非冷却缩减。CC套不提供冷却缩减，异界套装的主要收益在技能等级+伤害而非冷却。在70版本末期，冷却缩减的主要来源是药剂（精神刺激灵药）和少数特定装备，而非通用套装。

## 三、属性地基理论

**论点：力量/智力在70版本中不是"百分比增益乘数"，而是伤害公式的"地基层"——公式结构为 `基础攻击 × (1 + 力量/250)`，这意味着力量的本质是放大系数的线性放大器，而非独立乘区。250点力量=面板攻击翻倍的"固定汇率"设计，让力量收益呈现"越多越赚"的递增表象，但边际收益实际递减。**

**案例1：力量的"地基效应"**
对于狂战士，CC套+310力量进入暴走后（1430→1740），收益计算为 `(1+1740/250)/(1+1430/250)-1`，这本质上是新地基/旧地基的比例。力量对固伤技能的痛点是：固伤公式是 `基数 × (1+独立/250) × ...`，根本不包含力量项，因此固伤职业的力量增益完全不存在于公式中——这解释了+310力量对固伤零收益的根本原因。

**案例2：暴走的乘数放大**
暴走技能+40%力量（728→1019.2），表面上是力量放大40%，但在公式 `1+力量/250` 中，实际收益是 `(1+1019.2/250)/(1+728/250)`，从3.912→5.077，实际增幅仅29.8%（而非40%）。这说明力量作为"地基层"，其收益受到"基准250"的稀释——技能百分比增益被250常数拉平。

## 四、攻击力道理论

**论点：DNF 70版本设计了"物理攻击/魔法攻击/独立攻击"三套并行的攻击力道体系，这本质上是一种"伤害类型二维分类法"：以武器（百分比）vs固伤（独立）为横轴、物理vs魔法为纵轴。独立攻击的设立是DNF最深远的设计转折之一——它将固伤职业从"必须追高强化武器"的深渊中解放出来，同时为职业平衡创造了新的调节杠杆。**

**案例1：独立攻击成为固伤的"专属乘区"**
70版独立攻击在公式中为 `1 + 独立/250`（固伤）且 `1 + 独立/1000`（最终公式），独立攻击的价值完全集中于固伤职业。CC套+120独立对狂战士固伤贡献+8.0%，而对剑魂（纯百分比）贡献为0。这说明独立攻击的引入本质上是将固伤职业从武器强化依赖中分离，形成独立的经济循环——锻造取代强化成为固伤选手的核心投入点。

**案例2：独立加成%的极端稀缺性**
任务20证实：全70版本仅2件SS武器提供独立加成%（屠戮之刃+5%、魂·巨剑+7%），粉装/史诗首饰0件。这揭示了一个明确的设计规律——策划通过"极度压缩独立%乘区的供给"来控制固伤职业的数值天花板。相比之下，力量/物理攻击的加成装备非常丰富，说明策划认为固伤职业的膨胀风险高于百分比职业。

## 五、暴击概率论

**论点：暴击在70版本中是"低成本高回报"的经济属性——基础暴击伤害倍率150%，每提升1%暴击率等价于提升0.5%期望伤害（在非满暴状态下），但暴击装备的成本远低于同等期望收益的独立/力量装备。暴击的边际收益呈递减曲线：从50%→60%（10pp）收益约+3.3%，而从85%→95%（10pp）仅+1.4%。**

**案例1：CC套+3%暴击的收益验证**
狂战士55%→58%（期望1.275→1.29），收益+1.18%≈1.2%；剑魂50%→53%（期望1.25→1.265），收益+1.2%。两者+3%暴击带来的实际收益基本一致（约1.2%），说明暴击收益曲线在同一暴击区间对不同职业相近。但1.2%的收益显著低于独立+8.0%或力量+36.47%，证明暴击在CC套中属于"附带加成"而非核心设计意图。

**案例2：职业基础暴击分化**
狂战士基础暴击率55%（含被动血气分流+20%），剑魂基础50%（无被动暴击加成）。这形成了"固伤职业天生暴击高"的格局——固伤职业因无法从力量/强化获得百分比收益，策划通过给予更高暴击率作为补偿，维持两职业的相对平衡。

## 六、特殊力学

**论点：DNF 70版本中最具战略意义的特殊力学机制是"出血穿透防御"——这是策划刻意保留的"无视防御伤害通道"。穿透防御的本质是给了一个"经济制裁"：无穿透的普通伤害需要面对BOSS的高防御而大幅缩水，而有穿透的出血伤害则直接作用于生命值，不受任何防御减伤影响。这使得出血成为对抗高防御目标时的"作弊代码"。**

**案例1：出血的公式独立性**
出血伤害公式为 `出血固定值 × (1 + 独立/250)`，完全无视 `300/(300+防御)` 的防御减伤公式。Lv21出血基础3800/秒，满配5890/秒，对低防和高防怪物完全一致。这意味着当BOSS防御极高时（如安徒恩系，减伤率56%+），出血的实际价值被防御减伤倍数倍增。

**案例2：武器精通的武器差异化**
剑魂四种武器精通（巨剑/短剑/太刀/手剑）仅提供"基础物理攻击力差异"而非技能倍率差异——巨剑2000，手剑1400。这种"基值差异化"设计让不同武器选择产生30%+的基础攻击差距，但所有技能倍率保持不变。这意味着武器的选择影响的是"公式前端的基值"而非"公式后端的倍率"，是DNF设计中的"基值优先"哲学。

## 七、固伤 vs 百分比二元论

**论点：公式架构不对称定律——固伤与百分比流派的根本分化源于底层公式结构的不对称性，而非装备数值的简单差异。同一套装备（CC套+310力量/+110物理攻击/+120独立/+3%暴击）对狂战士固伤带来+9.27%，对剑魂百分比带来+43.95%，收益差距4.74倍。这个倍数（4.740937）经607+轮稳态核查确认为系统固有频率不变量——它是由公式结构本身决定的代数恒等式，而非人为设定的平衡参数。**

**边际对偶的数学本质：**
```
边际对偶 = 百分比综合收益 / 固伤综合收益
         = 43.95% / 9.27%
         = 4.740937

核心驱动因素：
1. 力量对固伤零收益（狂战士固伤不受力量影响）
2. 独立攻击仅贡献8.0%
3. 物理攻击仅贡献5.50%（百分比）+ 0%（固伤）
4. 暴击贡献相同（1.2%）
```

**案例1：CC套收益的二元分裂**
狂战士固伤仅从独立(+8.0%)和暴击(+1.2%)获得9.27%总收益，而剑魂从力量(+36.47%)、物理攻击(+4.23%)和暴击(+1.2%)获得43.95%。CC套给固伤提供了120独立但拒绝了310力量+110物理攻击，而给百分比职业同时打开了力量+物理攻击两扇大门——套装的"无意识偏见"暴露了公式架构的结构性失衡。

**案例2：边际对偶的动态稳定性**
在历时一个月的607+轮稳态核查中（v199→v619），边际对偶值始终稳定在4.74倍（精确值4.740937），即使在核心数据修正（v199物理攻击收益公式修正、v565剑魂基础力量680→600修正）后仍维持不变。这说明边际对偶不是偶然数值，而是由DNF 70版本的公式体系决定的系统固有频率——只要 `力量/250` 和 `独立/250` 的系数不变，这个比例就是一个代数不变量。

## 综合理论总结

**三层结构模型下的CC套定位：**

```
底座层（基础面板）  →  CC套在底座层提供泛用增益（力量/攻击/暴击）
增益层（技能放大）  →  异界套在增益层提供技能等级+效果增强
逃逸层（稀有词条）  →  史诗套在逃逸层提供独有词条（独立%等）
```

CC套在底座层的突出价值是"泛用性优先"——它不绑定任何技能或流派，任何职业穿戴都能获得实质性收益。但其收益极性分化（固伤+9.27% vs 百分比+43.95%）揭示了DNF底层设计的根本不对称性：**固伤职业天生被隔绝于力量乘区之外，而力量是70版本覆盖面最广、数值最高的属性**。

**五大维度综合观察：**

| 维度 | 核心发现 |
|------|---------|
| 设计模式 | 阈值触发（6件激活）+ 乘区隔离为核心设计范式 |
| 经济规律 | 独立%乘区被极度压缩（仅2件SS），力量乘区被极度丰富 |
| 流派分叉 | 固伤/百分比的分裂源自公式结构而非装备属性 |
| 乘区拓扑 | 力量→基础乘区，独立→固伤专用乘区，暴击→暴击乘区 |
| 版本特征 | 70末期是"基础属性为王"的时代，CC套泛用性强 |
