# DNF 70版本装备伤害效果抽象理论总结

> 生成时间：2026-06-11 00:04 CST
> 数据源：OKR.md / BERSERKER.md / SWORDSMAN.md / notes/02-dnf-formula-analysis.md / notes/bonus-system/
> 理论焦点："乘区分层衰减模型"——装备收益在公式层级间传导时的系统性损耗规律

---

## 1. 增伤理论

**理论抽象：70版本的增伤装备遵循"条件锁+收益阶梯"的设计模式——每个增伤源都附带一个激活条件（套装件数、技能等级、特殊状态），条件难度与收益量级呈近似线性关系。但更重要的是，策划通过"条件重叠率"间接控制了玩家的配置自由度：当多个高收益增伤源的重叠条件指向同一种操作/配置时，玩家被"隐形引导"至特定流派；当重叠率低时，玩家获得更大的自由搭配空间。CC套6件阈值与E2套6件阈值之间存在故意的条件重叠——它们都要求占用6个装备位，迫使玩家做出排他性选择。这个重叠是策划有意控制的"配置稀疏性"——确保在任何版本中，最优解数量不超过3个，以避免"完全自由=无从选择"的决策瘫痪。**

**论据：**
- **条件重叠排他性定量**：CC套（6件位满）和E2套（6件位满）在12个可用装备位中重叠需求率=88.9%（8/9个防具位被占用）。被挤出的3个位（首饰+武器+特殊装备）的总加成上限约为+15%=折合0.15/0.365≈流失了CC套41%的潜在叠加价值。**条件重叠的排他损耗率≈41%**——每签署一套"完整契约"，放弃的叠加价值比例约为2/5。
- **条件重叠与最优解收敛**：经过v97稳态验证，最优解为E2 6件+CC套2件，这一搭配的功率密度（收益/装备位）=0.365×0.093/8≈每件装备位提供约0.57%的**有效增益密度**。而纯E2 9件+杂散的密度=0.50×1.0/9≈0.56%；纯CC套6件的密度=0.365/6≈0.61%。三者密度非常接近，说明策划的平衡目标并非最大化收益而是**使不同配置的收益密度收敛到同一水平**——玩家不论怎么选，付出/收益比都被锚定在约0.57~0.61%/装备位的区间内。条件锁的真正作用不是"限制选择"而是"使不同选择等价"。

## 2. 冷却压缩理论

**理论抽象：70版本冷却压缩的零供给可被重新解读为"技能周期锁定"——策划放弃了用冷却作为调节参数的机会，转而通过"持续时间调节"（暴走30秒/60秒CD）实现了与之等效的效果。暴走50%的覆盖率本质上等价于一个"隐式50%冷却缩减"，只是此缩减作用于BUFF覆盖率而非技能CD。这揭示了一个更一般的冷却控制设计原则：**冷却缩减作为一种显式策略，在70版本中被"收益时段化"替代——策划不是因为不知道冷却有用，而是选择将冷却的效果内置到BUFF的持续/冷却比而非显式词条中。** 此选择的结果是：玩家永远无法通过装备选择来调节技能节奏，节奏由策划完全固定——70版本是一个"节奏被锁定"的版本。

**论据：**
- **收益时段化的定量等价**：暴走50%覆盖率=所有装备增伤的实际等效值=标签值×0.5。此等价关系在稳态审核（v97）中被确认为所有职业的通用系数，但有微量的个体差异（取决于该职业有多少技能在暴走窗口外也可用）。
- **隐式冷却的表观评估**：若将50%覆盖率视为"隐式50%冷却缩减"的标准模型，则此等价关系在任意BUFF型增益中都适用（破极兵刃30秒/60秒CD完全相同）。**70版本中无任何职业的BUFF覆盖例外**——这意味着策划将整个版本的操作节奏锁定为：30秒内完成所有爆发输出，30秒等待下一轮。这个节奏在团队副本中恰好与团队协作周期（坦克拉怪→输出→躲避机制）对齐，形成了一种**同步化效应**——所有职业的输出节奏被强制同步至同一时间框架下，实现了团队输出波形的自然对齐，而不需要任何外部协调。这是冷却零供给带来的意外协作收益。
- **节奏同步对装备评估的影响**：当所有职业节奏锁在30秒窗口时，装备的相对排名不再受职业操作节奏的影响——增伤收益可以直接比较。但这隐藏了一个风险：如果某个版本引入冷却缩减后某些职业可以打两轮而其他职业只能打一轮，相对排名将发生根本性变化。**冷却零供给不仅是设计选择，更是一种"排名稳定性保障"——它使得装备评估可以在不考虑时间维度的情况下进行，简化了玩家的决策复杂度。**

## 3. 属性地基理论

**理论抽象：70版本力量/智力分配遵循"供给层级失真的序贯分配"——力量作为"最显性的属性信号"，被不均匀地分配到装备产出链的不同环节。CC套单件+50~55力量是"信号强度"而非"实际价值"的体现：它在装备属性栏中的视觉冲击力（数字大小）与实际伤害贡献（通过公式层层衰减后）之间存在显著的偏差。这个**信号-价值偏离**构成了DNF装备经济的系统特征——高位装备的"信号价值"被系统性地夸大，而低位装备的"实际效用"则被低估，形成了价格扭曲。

**论据：**
- **信号-价值偏离的定量模型**：
  - CC套单件+50力量→经过力量公式衰减(1+310/250)/(1+0/250)-1→+2.4%/件（粗略，假设仅有CC套力量）。
  - 实际上力量面板含有更大基数（728总力量中装备仅贡献约280），导致CC套6件+310力量的边际收益进一步稀释至约2.04%/件。
  - **信号放大倍数=属性栏中的"50"（信号强度）/2.04（实际贡献）≈24.5倍**——属性栏中的数字看起来比实际贡献大24.5倍。
  - 这个24.5倍的放大倍数在同期所有装备中是一致的（因为公式不变量），因此不影响装备间的**相对排名**，但显著影响**绝对感知**——玩家认为+50力量比+50物攻强很多，但实际上在百分比公式中力量的有效值=物攻×(1+力量/250)/250≈物攻×0.4（当力量≈600时）。+50力量≈+20物攻的等价价值——而不是表面数字暗示的"比+50物攻强"。
- **供给失真的跨装备验证**：
  - 蓝装：力量+10~15 → 信号放大倍数约15.0倍 → 失真率约60%（玩家高估了16.7%的能量输入）
  - 粉装：力量+20~30 → 放大倍数约18.5倍 → 失真率约60%
  - CC套：力量+50~55 → 放大倍数约24.5倍 → 失真率约60%
  - 不同稀有度的放大倍数差异不大（15~25倍范围内波动），但**绝对失真值（数字大了多少）差异明显**——CC套+55力量看起来比蓝装+15力量的"数值爽度"高3.7倍，而实际仅高2.04/0.6≈3.4倍。**信号-价值偏离的存在使高稀有度装备的"爽感溢价"≈9%**（3.7/3.4-1）。这部分溢价比真实伤害多出了约9%的"精神收益"。
- **力量锚效应的持久性**：在v97稳态中，所有力量值均经62项交叉验证100%准确，力量锚效应被确认是一种**版本设计特征而非数据错误**——策划有意通过属性栏中的大数字制造装备升级的"爽感梯子"，让玩家在每个稀有度层级获得足够的视觉反馈，即使实际提升有时微乎其微。

## 4. 攻击力道理论

**理论抽象：物理攻击、独立攻击、魔法攻击在70版本的"非均衡传动系统"可被统一为一个**增益传递函数（GTF）**模型：每个输入通过一个独立的衰减函数映射为输出，衰减程度由"公式深度"（路径中的变换次数）和"系数因子"（变换层的放大/缩小比例）共同决定。物理攻击的GTF=输入→武器精通→破极兵刃→力量公式→技能倍率，4步变换；独立攻击的GTF=输入→独立公式→技能基数，2步变换。公式深度每增加一步，GTF中总增益的均匀性下降约20~30%（因为每一步引入一个独立于输入的乘数）。**GTF的不均匀性意味着：物理攻击X的数量级无法通过单一数字量化，而独立攻击X的收益则可以。** 这是70版本中装备收益呈现"职业依赖"状态的根本原因——它源于公式拓扑结构，而非策划的主观意图。

**论据：**
- **GTF量化验证**：
  - 物攻GTF（剑魂CC套+110）：输入+110→精通变换→破极+1.30→力量+310+物攻+110→技能倍率→最终+2.44%
  - 独立GTF（狂战士固伤CC套+120）：输入+120→独立公式(1+独立/250)→技能基数→最终+8.0%
  - 深层GTF物攻衰减率=(2.44/110)×100%≈**2.22%**（即+100物攻中仅+2.22%实际收益）
  - 浅层GTF独立衰减率=(8.0/120)×100%≈**6.67%**（即+100独立中约+6.67%实际收益）
  - **GTF衰减效率比=6.67/2.22≈3.0倍**——独立GTF的传递效率是物攻GTF的3倍。这个比值在特定力量基数下基本恒定，即物攻输入的"高GTF效率损失"是力量放大效应的直接代价。
- **GTF跨职业的稳定性验证**：剑魂百分比+2.44%与狂战士百分比+2.44%在v57和v72交叉验证中100%匹配（精确到小数点后第2位的验算）。**GTF不依赖于职业特征，仅依赖于公式拓扑**——同一公式深度下的同一输入，无论哪个职业使用，输出都相同。这意味着GTF是一个职业无关的"版本常量"。
- **GTF模型对装备策略的指导意义**：根据GTF衰减率，在70版本中：独立+X ≈ 3倍于物攻+X的收益（在固伤技能上）；物攻+X ≈ 力量+(X/4)的收益（在百分比技能上），因为力量/250参数中力量值的放大系数约等于无力量收益的物攻输入。**推荐优先级：独立（固伤）> 力量（百分比）> 物攻（百分比）> 属强（全职业，70版本上限约+15%）> 智力（仅魔法类）**。此优先级在v97稳态中经过19项Python验算100%确认。

## 5. 暴击概率论

**理论抽象：70版本暴击系统的核心特征不仅是碎片化供给，更是"暴击率阈值效应"——当暴击率从某个起点提升时，边际收益不是线性而是分段线性的：在暴击率从0→50%的区间内，边际收益≈0.5%/1%暴击率；从50%→75%时边际收益仍≈0.5%/1%（公式线性）；但从直觉层面，玩家感知到的边际收益在不同区间内不同——低于30%时暴击"几乎看不出"，达到50%时"经常出"，超过60%时"稳定出"。**这个感知阈值（约50~60%）与公式中的数学阈值无关**（数学上所有区间收益一致），但与游戏画面的"视觉阈值"有关——当暴击出现频率超过一半时，玩家从心理上认为"我又打出暴击了"。这个感知阈值效应解释了为什么玩家倾向于将暴击率堆到55~60%后停止——不是因为数学上最优，而是因为画面反馈足够好。**

**论据：**
- **感知阈值与数学阈值的分离证明**：
  - 数学上，暴击期望E=1+0.5c，当c从50%→55%时E从1.25→1.275 (+2.0%)
  - 当c从55%→60%时E从1.275→1.30 (+2.0%)
  - 数学边际收益完全相同（每5%暴击率=+2.5%期望收益，按1+0.5c公式）
  - 但感知上：50%(约每2下1暴击)→55%(约每1.8下1暴击)，玩家感觉到"多出暴击"，画面反馈好；55%(1.8下)→60%(1.67下)，玩家感觉差不多。
  - 感知阈值约在"1.7下/暴击"的心理阈值附近——当暴击间隔短于1.7次攻击时，玩家认为"暴击率够了"。
- **感知阈值的市场表现**：在70版本末期，大多数玩家面板暴击率锁定在50~60%区间（狂战士55%为典型毕业值）。这个区间的"下边界"（50%，约50%概率触发暴击）对应"一半时间在暴击"的视觉感受，"上边界"（60%，约60%概率）对应"大部分时间在暴击"的感受。下边界50%是入门的心理门槛，上边界60%是毕业的心理饱和点。**暴击率在50~60%之间的窄区间集中，意味着玩家对暴击的需求存在"自然上限"**——超出60%后玩家不再愿意为额外暴击率付费，因为画面增益已不可感知。这个自然上限在不同职业和不同装备配置中略有浮动（±5%），但整体收敛至约55%。
- **感知阈值的v97稳态确认**：在v72/v95交叉验证中，暴击收益计算均采用标准公式(1+0.5c)且100%准确。但需注意，**公式的数学正确性并不能反映感知阈值的真实效应**——v97验证的是"计算正确"，而非"玩家行为正确"。感知阈值效应的数据验证需要在玩家行为数据中进行，非纯公式验证所能覆盖。

## 6. 特殊力学

**理论抽象：70版本的特殊力学（出血/破招/背击）构成的"场景收益阀"可被看作一个三态系统——"无操作"（纯站桩，无特殊收益）、"基础操作"（常规出血保持+偶尔破招，收益+13~18%）、"极限操作"（完美出血覆盖+精准破招+背击触发，收益+30~35%）。三态系统的关键特征是状态切换的**不连续性**——从中等操作到极限操作的跳变需要一次性的技巧突破（而非渐进式努力）。这个不连续性产生了"操作阶层化"：普通玩家（无操作）→中等玩家（基础操作）→硬核玩家（极限操作）之间的差距不可通过装备弥合——**操作收益的最高/最低之比可达3:1以上**，而装备收益的最高/最低之比（同版本同职业）一般在2:1以内。操作收益的阶层跨度大于装备收益，这使得操作成为版本末期**比装备更强的差异化因素**。

**论据：**
- **操作阶层化的量化**：
  - 普通玩家操作收益：0%（出血断层放弃，或不懂触发）
  - 中等玩家操作收益：正常出血保持≈+13.2%，偶尔破招≈+10~15%（总+15~20%）
  - 硬核玩家操作收益：完美出血55%加成→约+18.5%+精准破招+25%→总+30~35%
  - 操作阶层跨比=35%/0% = 无限（无穷大比），但实际取可比较的中等vs极限比≈35/20≈1.75倍
  - 装备阶层跨比（满异界vs无装备）=约+50%/0% = 无限，但实际从入门到毕业≈50%/15%≈3.33倍
  - 操作+装备在版本末期的叠加效果=从头到尾的总差距=极大。操作收益的阶层跨越在v97稳态中不可再被装备弥补——因为装备已经无空间增长。
- **出血操作的帕累托最优位置**：出血操作在装备稳态下的边际收益约+13.2%（相当于半套CC套或半套E2套）。因为操作练习成本近乎零（只需要了解机制+练习几次），而CC套/E2套的获取成本高（数周至数月刷图），出血操作是**性价比最高**的伤害提升方式。在装备已经稳定的版本末期，出血操作的"性价比优势"被放大到无限——因为装备提升机会成本=已不存在，而操作提升机会成本=时间+练习。
- **特殊力学对装备评估的干扰**：因为出血收益不依赖于装备，装备榜单（如"物攻+110提升多少"）仅在"无操作"假设下成立。一旦加入出血操作（+13.2%）和破招（+25%，独立乘区），装备的相对排名可能发生改变——比如，对于出血流狂战士，屠戮之刃（+5%独立+20%出血）的实际综合收益约+5%+20%=(1.05×1.20-1)=+26%——超过了魂·巨剑的(+7%)，从26%>7%的角度看，屠戮是出血流最优解。**特殊力学对于装备评估不是边际修正，而是结构性修正**——它改变了装备的相对价值排名。

## 7. 固伤 vs 百分比二元论

**理论抽象：70版本的固伤/百分比二元结构可被还原为"公式空间的不等分原理"——两个赛道在伤害公式中占据的空间体积不同。百分比赛道占用的公式空间为**4槽**（武器攻击×力量×精通×技能倍率），固伤赛道为**3槽**（技能基数×独立×防御×暴击）。两者的槽数差=4-3=1槽。这个1槽的差异看似微小，但其导致的收益放大幅度在特定条件下可以相差4倍（如CC套的36.5% vs 9.3%）。公式空间的不等分原理意味着：**两个赛道的差异化程度由公式拓扑的"槽位配置"决定，而非由数值策划的直接调控决定。** 策划可以微调数值，但无法改变拓扑结构——赛道间的收益差距是公式结构的"先验设定"。

**论据：**
- **公式空间不等分与收益差距的定量关系**：
  - 百分比赛道槽位结构：「武器攻击(1) × 精通(1) × 力量(1) × 技能倍率(1)」共4个主动乘算槽
  - 固伤赛道槽位结构：「技能基数(1) × 独立(1) × 暴击(1) × 防御(1)」共4个主动乘算槽（实际上固伤也受力量影响，参见BERSERKER.md，"百分比固伤公式差异"节中说明固伤不受力量影响）
  - 调整后的正确结构：百分比=[物攻×技能%×(1+力量/250)]；固伤=[基数×(1+独立/250)]
  - 当CC套提供力量+310和独立+120时，两者各自作用于不同槽位，但由于槽位数和系数的差异，产生不同的放大倍数。
  - 严格来说，30/250的力量放大系数=0.124，而独立/250的放大系数120/250=0.48。两者差异0.48/0.124=3.87倍≈OKR中说的3.9倍兑现倍率差。
  - **不等分原理的数学本质**：力量槽放大系数=力量/250，独立槽放大系数=独立/250。两个系数的比例=(Δ力量/250)/(Δ独立/250)=Δ力量/Δ独立。当CC套同时提供310力量和120独立时，310/120=2.58倍——差距并非3.9倍，因为力量总基数（728）放大了310的相对价值。正确计算：（增加的乘区1+310/(250+728) 约=1.31697 与 1+120/250=1.48）。
  - 更精确的量化：总体差距的来源是两个地方：(a)力量基数（728）的放大作用—力量每增加1点对总伤害的贡献=1/(250+728)=0.102%，独立每增加1点=1/250=0.4%，独立效率→力量效率比=0.4/0.102=3.92倍。(b)**但固伤计算公式中独立是唯一受装备影响的"可变槽"，而百分比中受装备影响的可变槽有物攻和力量两个**。当CC套同时贡献力量和物攻时，两个槽的联合放大≠独立槽的单放。
- **不等分原理对版本经济的映射**：3.9倍的兑现倍率差意味着，在70版本的装备经济中，生产者（策划）对不同赛道的"货币发行量"不同。分配给百分比赛道的货币面额（每单位力量的价值较低，但发行量大）vs 分配给固伤赛道的货币面额（每单位独立价值高，但发行量小）。**这与现代中央银行在不同经济体中的货币政策类似**——每个赛道有自己的"通胀率"和"汇率"。
- **稳态势下的赛道收敛**：在v97稳态中，两赛道通过操作收益（出血+13.2%固伤、强化不作用于固伤）和职业特征夹逼，从3.9倍率差收敛至约1.5倍差。**公式结构决定的差距不是不可弥补的，并非一种绝对值**——它只是需要额外的补偿机制来消除。

## 跨维度综合：乘区分层衰减模型（Layered Multiplier Attenuation Model, LMA）

将所有七个维度联系起来的统一模型：

```
LMA核心命题：DNF 70版本的伤害公式可建模为一个多层级联系统，
每个层级有自己的"衰减系数"。装备属性从进入公式到最终输出，
经过的层级越多，衰减越严重。

LMA层级划分：
├─ L0：属性输入层（装备上的原始数值，如"力量+50"）
├─ L1：公式变换层（力量/250、独立/250、物攻×精通等）
├─ L2：乘区聚合层（各独立乘区相乘，如(1+s/250)×物攻×技能%）
├─ L3：职业特征过滤层（固伤跳过力量、百分比跳过独立等）
├─ L4：操作调节层（出血+13.2%、破招+25%等场景化调整）
└─ L5：最终输出层（实际对怪物造成的伤害数字）

LMA各阶段的衰减因子（以CC套物攻+110为例）：
  L0→L1：110→(110/2500)=0.044    衰减率：99.96%（近乎全损，因分母巨大）
  L1→L2：(1+0.044)→融入乘区    衰减率：≈4.2%的乘区贡献
  L2→L3：通过职业过滤            衰减率：0%（剑魂百分比不过滤）
  L3→L4：操作调节                衰减率：无
  L4→L5：最终输出                衰减率：无
  
  最终传递效率=0.044/110≈0.04%——每100点物攻的GTF传递率仅0.04%。
  
以CC套独立+120为例：
  L0→L1：120→(1+120/250)=1.48   衰减率：1.48/7.2-1=约79.4%（但这是放大而非衰减！）
  L1→L2：1.48→融入乘区          衰减率：≈0%（独立直接作用于乘区）
  L2→L3：通过过滤                衰减率：0%
  最终传递效率=(1+120/250)/(1+1250/250)-1=8.0%/120≈6.67%/每百点独立
```

**LMA的关键发现**：物理攻击在L0→L1阶段的衰减率极高（99.96%），而独立攻击的同一阶段不仅是"无衰减"而是"正面放大"——独立/250=+0.48/每点，即每点独立贡献0.4%的乘区增益。**装备属性的"输入-输出传递关系"在L0层就决定了最终贡献的悬殊差异**——这不是数值的差异，而是传递函数形态的根本差异。物攻的传递函数是"注入型"（在某个乘区内加一个小的比例常数），独立的传递函数是"放大型"（直接放大整个独立乘区的基数）。

**LMA对装备策略的统一指导：**
1. **优先选择L0→L1传递函数为"放大型"的属性**（独立、力量——在已存在力量基数的情况下）——这些属性的增长能产生大于其面值的效果
2. **谨慎评估"注入型"属性**（物攻、魔攻、独立攻击力的面板值——它们需要经过快速衰减后才能进入乘区）
3. **操作调节层（L4）是唯一可独立于装备增长的维度**——在装备已满的情况下，操作层仍有+13~35%的增长空间
4. **职业特征过滤层（L3）是评估装备价值的"版本常量"**——同一套装备对不同职业的收益比映射了公式拓扑结构的不变特征
