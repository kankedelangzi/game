## 一、增伤理论：条件触发模型与乘区拓扑结构

DNF 70版本的伤害加成构建于"条件触发 + 阈值激活"的双层设计范式之上。加成模式呈现显著的非连续性——大多数套装的完整收益依附于明确的激活条件（如异界套6件阈值、CC套6件阈值），而非线性递增。这种设计迫使玩家在"套装完整收益"与"散搭灵活性"之间做出权衡，形成了70版本独具特色的装备搭配博弈。

**核心规律：加成来源与收益激活呈正相关，但激活条件越严苛，单件平均加成越高。**

**案例1：CC套6件阈值机制** — 1~5件仅享受单件属性（力量约260、物理攻击约90），第6件才激活完整的套装效果（额外增加力量+310、物理攻击+110、独立攻击+120、暴击+3%）。6件阈值意味着玩家必须牺牲6个部位换取套装收益，迫使与其他套装（如E2 6件）形成"择一"关系。

**案例2：异界E2/E3套装** — E2侧重技能等级加成（如狂战士血之狂暴6件：技能+35%、出血+10%、力量+5%），E3侧重伤害加成（技能+30%、出血+15%）。两者同为6件阈值机制，但定位分化：E2强化特定技能流派，E3提供通用输出提升。

**案例3：独立加成极稀缺** — 全70版本仅2件SS武器提供独立加成%（屠戮之刃+5%、魂·巨剑+7%），证明策划采用了"乘区隔离"策略——通过有意限制某个乘区的装备供给来控制职业平衡。

该分类数据不充分，现有研究仓库未系统收录冷却缩减装备的数值分布。基于已有样本的初步观察：

70版本冷却缩减装备数量极为有限，主要集中在异界套装的特定部位加成（如E2狂战士暴走+2级通过提升技能等级间接缩短CD间隔）。冷却压缩在70版本属于"隐性乘区"——主要通过技能等级提升间接影响冷却，而非直接标注"冷却缩减+X%"的词条。这与此后版本（80/85）引入的显式冷却缩减装备形成对比。

## 三、属性地基理论：力量/智力作为收益基底的三层不对称结构

DNF 70版本的属性地基系统存在一项根本性的结构性不对称：力量/智力通过公式 `(1 + 主属性/250)` 作为所有伤害类型的乘区基底，但固伤技能实际上切断了对这一基底的依赖——看似公式中存在力量乘区，实际固伤技能的"伤害 = 基数 × (1 + 独立/250) × 防御 × 暴击"中，力量被公式结构性地排除在外。这一发现构成了"公式架构不对称理论"的核心：固伤公式实际上是一个"降维"的公式，它退去了力量这一维度，只保留独立攻击作为面板输入。

**三层结构模型（底座层→增益层→逃逸层）：**
- **底座层**：力量/智力作为全局属性基底，对所有职业统一作用
- **增益层**：技能倍率、装备加成在此层叠加，产生流派分叉
- **逃逸层**：异常状态（出血）穿透防御，直接作用于HP，实现"逃逸"机制

**案例1：CC套力量+310** — 对狂战士固伤技能收益为0%，对剑魂百分比技能收益+36.5%。同一件装备、同一数值，因目标职业的伤害公式结构不同而产生从0%到36.5%的巨大差异。这证明了力量/智力不是"普适乘数"，而是"条件乘数"——它仅作用于百分比技能，对固伤技能是结构性零收益。

**案例2：力量收益边际递减曲线** — 力量从600→910收益+36.5%，而从728→1038（暴走后1019.2→1329.2）收益仅+24.42%。在同一乘区内，基底越高则边际递减越明显，这是 `(1 + new/250)/(1 + old/250) - 1` 公式的固有数学特性。

## 四、攻击力道理论：三轨并行的攻击力分工体系

物理攻击、魔法攻击、独立攻击在70版本构成了三条并行的"伤害通道"——这三者的分工逻辑根植于职业底层设计：物理/魔法攻击对应百分比技能（依赖武器面板），独立攻击对应固伤技能（依赖独立攻击数值）。这种"双系统并行"的设计哲学确保了百分比职业和固伤职业拥有不同的装备追求路线，而策划通过独立加成乘区的高度稀缺性（全版本仅2件）来控制固伤职业的数值天花板。

**攻击力道三通道模型：**
| 通道 | 对应技能类型 | 依赖属性 | 收益公式特征 |
|------|------------|---------|------------|
| 物理/魔法攻击 | 百分比技能 | 武器面板×强化 | 直接乘数（无系数分母） |
| 独立攻击 | 固伤技能 | 独立面板 | 除以250后有收益 |
| 混合通道 | 混伤技能 | 两者 | 分开计算后叠加 |

**案例1：物理攻击作为直接乘数** — CC套物理攻击+110，收益比 = phy_new/phy_old - 1，这是直接乘数，非 `(1+phy/2500)`。这一修正（v199）使狂战士百分比综合从28.97%→32.81%，剑魂综合从41.1%→43.95%。物理攻击公式不存在系数分母，意味着其边际收益线性递减（单纯因为分母增长），但无结构性衰减。

**案例2：独立攻击与固伤关系的系数结构** — 独立攻击/250的系数意味着每250点独立攻击翻倍固伤基础值。CC套独立+120/250 = 0.48倍提升，基数为1250时的收益为(1+1370/250)/(1+1250/250)-1 = 8.0%。与力量乘区同分母（250）的设计暗示了策划试图统一两个乘区的"刻度"，但独立攻击的绝对数值规模远低于力量（独立约1250 vs 暴走后力量约1019），导致独立攻击每一"增量刻度"的边际价值更高。

## 五、暴击概率论：暴击作为收益稳定器的设计意图

70版本的暴击系统采用"命中期权"模型——暴击不是必发事件，而是概率性触发1.5倍基础伤害。期望公式为 `E(dmg) = 1 + 暴击率 × 0.5`。这一设计的精妙之处在于：暴击率提升带来的收益呈现线性（每1%暴击率带来约0.5%的期望伤害提升），但实战中暴击的实际价值高于期望——因为在70版本的连招输出模式中，关键技能的一次暴击可能决定击杀窗口。策划通过限制暴击率的常见阈值（物理职业约50%~55%、固伤职业更低）来维持暴击的"惊喜获得感"而不至于成为确定性收益。

**案例1：CC套暴击+3%的期望收益** — 狂战士55%→58%，期望系数1.275→1.29，收益+1.2%；剑魂50%→53%，期望系数1.25→1.265，收益+1.2%。两个职业的暴击收益完全相同（均+1.2%），因为3%暴击率提升的期望公式增量是固定的（0.03 × 0.5 = 0.015，即+1.5%——但由于期望系数比值的缘故会略低于理论值）。

**案例2：爆伤系数1.5倍** — 70版本暴击伤害固定为1.5倍，没有暴击伤害加成装备（暴伤加成乘区为80/85版本引入）。这意味着70版本的暴击系统比后续版本更"纯净"——它是纯粹的概率加成，不受装备词条的稀释或冲突影响。

## 六、特殊力学：出血穿透防御的设计意图与破招奖励

70版本的特殊力学体系中最显著的特征是"出血穿透防御"机制——出血伤害直接作用于HP，无视怪物防御减伤公式 `300/(300+defense)`。这一设计创造了一条"逃逸通道"：当怪物防御极高导致百分比/固伤技能伤害大幅衰减时，出血提供了无视防御的固定数值伤害。这是一条"结构性逃逸路径"——它不是数值层面上的"高VS低"问题，而是从防御公式体系本身的逃逸。

**乘区拓扑中的逃逸通道定位：**
- 常规伤害路径：攻击力 → 属性乘区 → 防御减伤 → 暴击 → ... → 最终伤害
- 出血逃逸路径：出血等级 → 出血基数 → (1+独立/250) → 直接HP减伤（跳过防御）

出血伤害不受防御减伤影响，但受独立攻击加成和异常状态伤害加成影响。这在70版本中形成了"双通道输出模型"：常规技能负责正面输出，出血负责绕过防御体系提供稳定伤害。

**案例1：Lv21出血原始基数3,800/秒** — 含被动常驻加成后实际4,940/秒，最高+55%加成。这个数值不受被攻击目标的防御值影响——无论对于防御2000的白图怪物还是防御15000的BOSS，出血伤害一致。这使出血成为70版本高难副本中对抗高防御怪物的稀缺手段。

**案例2：E2对出血等级的额外加成** — 十字斩出血等级+2（Lv19→Lv21），崩山击出血等级+2（Lv21→Lv23）。出血等级每提升1级，出血基数按固定步长增长。E2套装对出血的加成独立于对技能的伤害加成，形成"技能伤害+出血伤害"的双轨强化。

## 七、固伤 vs 百分比二元论：公式架構不对称理论的实证

固伤和百分比并非同一框架下的两种"数值偏好"，而是底层公式结构的分叉——它们的乘区依赖完全不同，导致同一套装备在两种职业身上产生天壤之别的收益。这不仅仅是数值差异，而是公式架构层面的不对称。

**公式架构不对称理论（三层不对称结构）：**

```
                    ┌─────────────────────────────────────┐
                    │  公式层（底层公式结构）               │
                    │  百分比：物理攻击 × 技能倍率 × (1+力量/250) │
                    │  固伤：技能基数 × (1+独立/250)        │
                    │  → 力量在固伤中结构性为零             │
                    └─────────────────────────────────────┘
                                ↓
                    ┌─────────────────────────────────────┐
                    │  成长层（装备/技能增益）              │
                    │  百分比：力量×物理攻击×暴击（三乘区）   │
                    │  固伤：独立×暴击（两乘区）            │
                    │  → 百分比职业收益乘区更多，叠加效应更大 │
                    └─────────────────────────────────────┘
                                ↓
                    ┌─────────────────────────────────────┐
                    │  Buff层（主动增益技能）               │
                    │  百分比：破极兵刃+30%物理攻击（乘算）   │
                    │  固伤：暴走+40%力量（对固伤零收益）     │
                    │  → 固伤职业的Buff在底层被公式层滤除    │
                    └─────────────────────────────────────┘
```

**独立加成稀缺定律：** 全70版本仅2件SS武器提供独立加成%（屠戮之刃+5%、魂·巨剑+7%），粉装首饰/史诗首饰提供独立加成%的为零。策划通过乘区隔离定律，有意控制独立加成乘区的规模，避免固伤职业数值膨胀失控。

**案例1：CC套对两种职业的收益差异（4.7倍）** — CC套6件套属性（力量+310、物理攻击+110、独立攻击+120、暴击+3%）对剑魂（百分比流）综合收益+43.95%，对狂战士（固伤流）综合收益仅+9.27%，相差约4.7倍。差异根源不在数值本身，而在于力量+310对固伤技能结构性零收益——力量是百分比职业的乘数，却是固伤职业的装饰。

**案例2：暴走技能陷阱** — 暴走+40%力量看似对狂战士极为有利，但狂战士核心输出技能（十字斩、血气之刃、怒气爆发）均为固伤，力量对固伤的收益为0。暴走的实际收益仅限于百分比技能部分（崩山裂地斩、嗜魂封魔斩），使暴走成为70版本最典型的"不对称Buff"——它的增益效果因底层公式分叉而被结构性削弱。

## 跨维度合成分析

### 公式架构不对称理论（核心元理论）

**定义：** DNF 70版本的伤害面值观上统一基于主属性/独立攻击/防御减伤/暴击的四乘区模型，但在公式底层存在结构性分叉——力量对固伤的零收益并非数值缺陷，而是公式层级的本质分工缺失。这种不对称在装备选型、流派定位、技能设计三个层面产生连锁效应。

### 三层结构模型（底座层→增益层→逃逸层）

**底座层**（属性地基）：力量/独立攻击作为全局基底，所有职业共享底层公式——但固伤公式实际上退化掉力量维度。「三层结构模型」在底座层即产生分叉。

**增益层**（装备/技能加成）：CC套力量+310在此层对两种职业产生4.7倍的收益差异——固伤流只能从790/280（独立+暴击）中共提约9.27%的收益，而百分比流可以从三个乘区（力量+物理攻击+暴击）中叠加到约43.95%。

**逃逸层**（特殊力学出血）：出血穿透防御的设计创造了"第三条通道"，为固伤职业提供了绕过防御减伤公式体系的额外伤害来源——这在一定程度上补偿了固伤在增益层的劣势。

### 乘区隔离律与独立加成稀缺定律

策划通过装备供给控制乘区规模：
- **力量乘区**：供给充裕（装备/技能/时装/宠物/称号全面覆盖）
- **物理攻击乘区**：供给充裕（武器/强化/破极兵刃/装备全面覆盖）
- **独立攻击乘区**：极稀缺（仅2件SS武器提供加成%）
- **暴击乘区**：中等供给（技能+装备可堆至约50-55%）

独立加成稀缺定律：乘区供给量与职业固伤依赖度呈反比——固伤职业需要的独立加成供给被有意控制在最低水平。

### 版本特征：70末期的乘区拓扑独特性

70版本末期处于DNF乘区拓扑的"早期简化阶段"——只有4个主要乘区（主属性/独立攻击/防御减伤/暴击），尚无黄字/白字/所有攻击力/技能攻击力/最终伤害等80/85版本引入的后续乘区。这一简化结构意味着：

1. **加成冲突更少**：70版本不存在黄字冲突（取最高）和白字乘算——CC套、异界套、宠物/称号/勋章的加成属于同一乘区内部简单相加。
2. **阈值机制更突出**：由于可用乘区少，装备设计依赖"阈值触发"（6件套才激活）来制造装备选型深度。
3. **流派分叉更极端**：同一套装备对两个职业产生4.7倍收益差异，这在此后版本因乘区增多而有所稀释。
