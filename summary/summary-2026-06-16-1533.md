## 宏观框架：70版本装备系统的「三轴×双通道×双三角」建筑结构

DNF 70版本末期的装备系统可以抽象为一座三轴支撑、双通道分流、双三角悖论相互制衡的**建筑结构**：

```
               ┌──────────────用户面板──────────────┐
               │         力量(728/600)               │
               │         ↙       ↘                   │
          ┌────┴───┐                         ┌───────┴────┐
          │物理攻击 │                         │ 独立攻击    │
          │(2000/2000)                       │ (1250)      │
          └────┬───┘                         └───────┬────┘
               │                                  │
               ▼                                  ▼
          ┌────────────┐                  ┌────────────────┐
          │ 百分比通道  │                  │    固伤通道    │
          │ (剑魂+41.1%)│◄═══════════════►│ (狂战士+9.3%) │
          │ 破极兵刃+30%│  4.4倍收益鸿沟  │ 独立/250系数   │
          └────────────┘                  └────────────────┘
               │                                  │
               └──────────── 三轴增伤 ────────────┘
          ┌──────────┐  ┌──────────┐  ┌──────────┐
          │  E2等级轴 │  │  E3倍率轴│  │ CC套基底  │
          │ (技能+级) │  │ (伤害+%) │  │(面板属性) │
          └──────────┘  └──────────┘  └──────────┘
```

**核心洞察**：70版本的装备加成存在一个**结构性矛盾**——策划试图用"同一套装备"服务于两种不同的伤害计算路径（百分比与固伤），但公式底层的分叉导致同一装备对两条路径产生4.4倍的收益差距。这个差距不是数值平衡的疏忽，而是**公式架构不对称**的必然结果。

## 1. 🔺 增伤理论 —— 三轴分层 × 乘区门控 × 乘区投资组合理论

**理论抽象**：DNF 70版本的增伤拓扑遵循"三轴分层——乘区门控——投资组合"三层决策模型。第一层（三轴分层）：E2等级轴（技能等级加成）、E3倍率轴（伤害百分比加成）、CC套基底轴（面板属性加成）各不相交，同时装备时收益全量叠加无稀释。第二层（乘区门控）：策划通过控制"哪类装备允许进入哪个乘区"来实现平衡调控——独立加成%被严格门控（仅2件SS武器），暴击伤害%被门控（无直接暴伤加成装备），技能攻击力乘算尚未引入。第三层（投资组合）：玩家在有限的装备槽位（10个防具+首饰+武器）中进行"乘区配置"，每个装备槽位只能容纳一种乘区类型，形成7-8维度的多乘区投资组合优化问题。

**案例佐证**：
- **独立加成%门控**：全70版本仅屠戮之刃(+5%)和魂·巨剑(+7%)两件SS武器提供独立加成%，粉装首饰/史诗首饰均为0件。这不仅是稀缺，而是"乘区门控"——策划明确拒绝让独立加成%进入首饰乘区，有意将固伤职业的伤害上限锁定在一个安全的范围。
- **三轴叠乘验证（狂战士毕业方案）**：E2血之狂暴6件（技能+2级/力量+3%→总+5%）× E3血染江山（各技能+15%）× CC套2件（面板加成型）→ 各轴加成独立可全量叠加。这是70版本"异类乘区相加→相乘"的经典设计——增加了装备搭配的策略深度，绕开单一属性堆叠的稀释问题。

## 2. 🔵 冷却压缩理论 —— 70版本的"无CDR"是一种主动设计选择

**理论抽象**：冷却压缩在70版本末期的≈0存在不是设计盲区，而是**策划明确的权力分割**——输出频率的决定权归技能系统所有，装备系统的边界限定在"单次伤害大小"。这种设计优于直观理解：如果冷却缩减可装备化，那么高频技能触发流派将获得不受控制的边际增长（"攻击次数×每次伤害"中左侧的自由放大），导致平衡失控。70版本的"技能周期原生性"意味着玩家必须接受"30秒爆发+30秒等待"的节奏，迫使策略思考转向"如何在30秒窗口内最大化输出"，而非"如何缩短等待时间"。

**案例佐证**：
- CC套、E2/E3套、粉装/史诗、宝石、宠物/称号/勋章——全阶段研究的**所有装备中0件提供百分比冷却减免**。冷却压缩的唯一来源是技能等级提升间接影响冷却（如暴走+2级→可忽略的冷却变化）。这一数据的一致性本身就是最强的证据。
- 暴走技能（冷却60s、持续30s）创造了DNF 70版本最经典的"压力窗口期"设计模板。玩家必须在30秒内完成爆发技能序列（嗜魂之手→十字斩→崩山击→血气之刃，总约18.1万伤害），剩余的30秒转变为"风采走位+积攒资源"阶段。这一节奏控制是70版本战斗体验的核心组成部分。

## 3. 🟤 属性地基理论 —— 力量的"分叉矛盾"：对百分比是地基，对固伤是0

**理论抽象**：力量/智力处于伤害公式的最底层位置——`(1+力量/250)`——这意味着它在计算顺序上先于其他所有乘区，但也意味着它对固伤技能完全零贡献（固伤公式中力量被完全跳过）。力量因此呈现出**分叉矛盾**的身份：对于百分比职业，它是"地基"——决定了后续所有物理攻击收益的放大倍率；对于固伤职业，它直接沦为"不存在"。这一设计绝非偶然——它使"力量"成为区分百分比与固伤职业的本质边界。70%的装备同时提供力量和独立/物理攻击，力量的存在实际上是在强制过滤收益流向：力量→百分比职业受益，独立→固伤职业受益，物理攻击→百分比职业受益。CC套一次性地提供了全部三种基底属性，但收益流向完全被职业类型锁定。

**案例佐证**：
- **CC套力量收益的倍率差异**：狂战士（固伤）力量+310→收益0%；狂战士（百分比技能）力量+310→+31.7%；剑魂（全百分比）力量+310→+36.5%。同样的+310力量，对不同输出的相对收益从0%到36.5%，展示了"属性地基"的**职业分叉属性**。
- **毕业面板的力量配置**：狂战士毕业力量728（暴走后1019.2）中，固伤部分完全无视力量。这意味着力量的边际收益在狂战士身上只对限定的百分比技能生效——进一步强化了"混伤职业"的资源分配张力。
- **暴走×力量的乘序放大**：暴走「+40%力量」对728基数的增幅为1019.2-728=291.2，再作用于CC套310力量→暴走后总力量1329.2。这里隐含了一个增益计算规则：**百分比技能BUFF在装备加成之前先放大了力量基数**，使得"技能乘区×装备乘区"的混合效应在力量这一地基层面就已经开始。这一"先技能乘率、后装备加算"的计算顺序优先级本质上是一个隐式协同机制——确保百分比技能增益装备不会与技能BUFF形成直接加算稀释。

## 4. ⚔️ 攻击力道理论 —— 三通道分工：物理攻击/独立攻击/属性强化

**理论抽象**：DNF 70版本的攻击力系统分化成三条互不相交的通道——物理攻击（百分比专属）、独立攻击（固伤专属）、属性强化（通用但需属性攻击触发）。物理攻击的下游挂钩力量系数和技能百分比倍率，独立攻击的下游挂钩固伤基数和独立/250系数，属性强化则位于公式尾部作为独立乘区（除以220）。这三条通道之间**不存在比例换算关系**——你不能用100物理攻击去换算成多少独立攻击，它们服务于两套完全不同的伤害生成机制。这意味着装备设计时，一个提供"物理攻击+X"的装备天然只能被百分比职业充分利用，同理"独立攻击+X"天然是固伤装备。CC套同时提供物理攻击+110和独立攻击+120，看似兼顾两派，实则两者的收益在不同职业侧被完全隔开。

**案例佐证**：
- **CC套的物理攻击收益**：剑魂（2600→2710）得+2.2%；狂战士固伤流（物理攻击无关）得0%。110物理攻击的价值在剑魂身上约等于2.2%的独立乘区收益，但对固伤职业分文不值。
- **CC套的独立攻击收益**：狂战士（1250→1370）+8.0%固伤；剑魂（无固伤技能）独立攻击收益=0。120独立的去向被职业锁定。
- **独立加成%的稀缺**：屠戮之刃+5%独立和魂·巨剑+7%独立——仅2件。独立加成%的稀缺与独立攻击面板（通过武器锻造可提升）的可得性形成对比，揭示了一个设计模式：**独立攻击面板易得，独立加成%稀缺**。这意味着策划允许固伤职业获得平缓的线性成长（每点独立×1/250），但切断了对数级爆发性增长的可能性。

## 5. 🎯 暴击概率论 —— 暴击率作为"均匀分配"的通用收益乘区

**理论抽象**：暴击在70版本中扮演着"平等税"的角色——同一暴击率对所有输出类型的伤害期望增幅是统一的，因此暴击是70版本中少数不歧视固伤/百分比的属性之一。暴击期望模型遵循 `1 + 暴击率 × 0.5`（基础暴击伤害1.5倍），这意味着暴击收益随暴击率增加而线性上升，但**边际收益恒定**（每1%暴击率贡献0.5%伤害期望）。值得注意的是70版本中不存在暴击伤害加成%（暴伤加成词条）的装备——这使得暴击收益的上限被锁定在一个已知的线性范围内，进一步印证了策划对"可控增长"的偏好。

**案例佐证**：
- **CC套+3%暴击的对称收益**：狂战士固伤流55%→58%（+1.2%期望）；狂战士百分比55%→58%（+1.18%）；剑魂50%→53%（+1.2%）。1.2%的暴击期望提升在各个流派之间几乎一致，暴击率是唯一在双通道之间收益对称的属性。
- **各职业基础暴击格局**：狂战士通过血气分流（被动+20%暴击）实现55%高暴击面板，剑魂基础暴击较低（50%）。暴击率的差异决定了收益饱和点不同——55%暴击意味着每增加1点暴击率的边际收益为0.5%×85%=0.425%（扣除不暴击部分），而50%暴击的边际收益相同。暴击率的设计达成了"公平"——双方收益曲线一致，仅有效率门槛不同。

## 6. 🩸 特殊力学 —— 出血穿透防御的设计：一条"不需验证的伤害通道"

**理论抽象**：狂战士出血机制是70版本中罕见的——或许也是唯一的——"穿透防御"伤害通道。其公式 `每秒出血伤害 = 出血固定值 × (1 + 独立攻击力/1000)` 中完全跳过了防御减伤乘区 `300/(300+防御)`。这一设计等同于创造了一条**不需要通过"验证"的伤害路径**——在防御乘区中，伤害需经过减伤率过滤（高防BOSS仅剩40%左右），而出血的穿透性质意味着全部伤害100%作用于目标。这使得狂战士在面对高防御BOSS时（与低防白图对比）具有显著的伤害结构优势：物理技能部分被严重削弱，但出血部分维持不变。70版本中出血穿透的存在本质上是策划对固伤职业"高防场景补偿"的一种设计手段——弥补固伤职业在百分比技能部分（受防御影响）输出能力的先天不足。

**案例佐证**：
- **出血穿透防御的验证**：十字斩Lv18基础出血3,800/秒，E2+2级后达Lv21级约10,030基数×(+55%加成)≈15,547/秒——全部穿透怪物防御。在安徒恩类高防BOSS（减伤约57%）场景下，这相当于一个被放大的"无畏通道"：物理部分被削弱至43%生效，出血部分100%生效。
- **狂战士出血vs剑魂无出血**：两个职业接受CC套同样加成后，狂战士还有一个隐性的出血穿透通道可挖掘，而剑魂100%输出都要过防御这一关。这也是固伤职业+出血机制的结构性价值被低估之处——虽然CC套固伤仅+9.3%的表面数字，但出血穿透在实战中的协同放大未被纳入标准计算（受限于数据不充分）。

## 7. 🌀 固伤 vs 百分比二元论 —— 4.4倍收益鸿沟的公式根源

**理论抽象**：CC套对剑魂（+41.1%）与狂战士固伤（+9.3%）之间**4.4倍收益鸿沟**的根源在于**公式级分叉**——并非数值设计上的偏差。根源分析：

| 根因 | 百分比公式路径 | 固伤公式路径 | 鸿沟来源 |
|------|---------------|-------------|---------|
| 力量+310 | `(1+910/250)/(1+600/250)-1=+36.5%` | **跳过** | 36.5% → 0% |
| 物理攻击+110 | `(1+2710/2500)/(1+2600/2500)-1=+2.2%` | **跳过** | 2.2% → 0% |
| 独立+120 | **跳过** | `(1+1370/250)/(1+1250/250)-1=+8.0%` | 0% → 8.0% |
| 暴击+3% | `+1.2%` | `+1.2%` | 对称 |

力量的+36.5%和物理攻击的+2.2%合计约38.7%收益全部流向百分比侧，固伤侧仅获得独立的+8.0%。**CC套增量收益中约89%流向百分比通道，仅11%流向固伤通道**。这一结构不均等不是"数值失衡"，而是**公式架构不对称律**的必然结果——当一个装备同时提供力量和物理攻击时，它天然地偏袒百分比侧。

**公式架构不对称理论**的核心公式表达：
```
Δ%_percent ≈ (Δ力量 + Δ物理攻击) → 多乘区域
Δ%_fixed  ≈ Δ独立 × (1/250) → 单一乘区
Δ力量 × 0 = 0  → 固伤交互失能
```

只要力量参数不进入固伤公式，任何包含力量的装备对固伤的贡献都是零——这不是收益大小的问题，而是**收益有无的二元分叉**。

**经济规律洞察**：CC套对固伤的"冷淡"映射出一种更深层的设计哲学——70版本中固伤职业的伤害天花板被有意压制在低于百分比职业的水平。固伤职业的补偿不在于装备加成（装备端被系统性地忽略），而在于另外两个通道：（1）出血穿透防御提供了"高阶场景补偿"；（2）武器锻造独立攻击提供了"稳定线性成长"。这是一种**通道补偿均衡**：固伤在装备=被冷落，在机制=有后门。
