## 1. 增伤理论——阈值触发的系统收敛性与可预测性

70版本增伤设计的核心规则是**阈值触发模型**：装备加成按套件组别打包，单件只有基础属性加成，完整组合（6件）才激活全部增益（CC套：力量+310/物攻+110/独立+120/暴击+3%；E2：技能+2级/伤害+25%/出血+10%）。这一设计迫使玩家面对「完整收益 vs 散搭灵活」的二元选择。

更深层的是两套阈值系统交叠时产生了**可预测的收敛解**。E2 6件完整收益与CC套6件完整收益不可能同时满足（共需12个部位，可用仅10个），可行域为 (E2 n件) + (CC m件) ≤ 10。优化结果是E2 6件 + CC套2件（上衣+下装技能+1）—— 精确落在E2的阈值点上。这意味着：**双阈值系统的可行域交集必然收敛至其中一套的阈值解，且收敛方向由两套件的收益密度差异决定**。E2套由于提供不可替代的技能等级提升，收益密度超过CC套的任何组合，因此收敛方向确定为「E2满阈值 + CC非阈值」。

这一收敛性使70版本毕业搭配具有高度**可预测性**——给定任意两个异界/CC套的组合，其最优解可以在不进行遍历计算的情况下直接判定。151轮稳态核查对此结论做了终极验证：所有已知数据均与该结论一致，无一例外。

**佐证**：CC套6件核心数据（力量+310/物攻+110/独立+120/暴击+3%）经151轮100%通过率不可撼动；E2+CC 6+2最优搭配经任务13-19交叉验证确认。

## 2. 冷却压缩理论

**数据不充分**：现有研究覆盖的装备数据集中在增伤乘区，冷却缩减装备在70版本中属于分散性辅助属性而非可独立评估的策略维度。从E2套附带的少量技能冷却缩减推断，冷却压缩在70版本尚未形成独立的价值评估体系——它更像是装备属性的「填充维度」而非平衡切入点。该分类需阶段3跨职业/跨装备收集研究补充。

**初步观察**：70版本冷却装备的稀缺性实质上抬高了短CD技能的相对价值，因为无法通过装备大幅压缩核心技能冷却，技能循环的刚性比后续版本高得多。

## 3. 属性地基理论——基准值依赖与非对称收益放大

力量和智力在70版本伤害公式中以 `(1 + 力量/250)` 形式出现，这是典型的「地基层」结构——它作用于所有伤害类型的源头，但自身的百分比收益**完全取决于基准值大小**。一件固定数值装备（CC套+310力量）对不同基准值产生截然不同的收益，且这一差异在70版本被Buff机制大幅放大。

关键发现是**暴走Buff造成的收益非对称性**：狂战士暴走（力量+40%）将力量基准从728抬升至1019.2，导致CC套+310力量在暴走状态下的收益（+24.42%）远低于剑魂无暴走（600→910，+36.47%）。差异达12.05个百分点。这意味着：

> **相同的装备加成，在同一职业的不同Buff状态下，其相对收益可以相差超过10个百分点。**

这是70版本基础属性设计中的重要不对称现象。更深层的推论是：**任何增加基础属性的装备，其相对收益曲线必然是向下倾斜的**——基准值越高，收益越低。这一「反直觉」的设计使得力量加成装备存在自我衰减速机制（Self-Attenuation）：当玩家通过装备/附魔/增益不断提升力量时，力量装备的平均收益反而持续下降。

**佐证**：CC套+310力量→剑魂+36.47%（基准600）vs 狂战士暴走后+24.42%（基准1019.2），差异来自于暴走的二级放大。

## 4. 攻击力道理论——三元分工与独立加成隔离设计

70版本的攻击力道系统呈现**三元分工结构**：物理攻击、魔法攻击、独立攻击分别服务于百分比物理、百分比魔法、固伤技能，三者之间具有严格乘区隔离——物理攻击对固伤技能零收益，独立攻击对百分比技能零收益。

其中最值得研究的是**独立攻击的隔离设计**。独立攻击的公式系数为1/250（固伤基数×(1+独立/250)），与力量公式系数一致（而非早期版本研究误用的1/1000）。这意味着固伤职业实际依赖的底层属性是两个独立变量（力量 + 独立攻击），而非一个。但两者在固伤公式中的作用路径不同：独立攻击直接放大技能基数，力量通过主属性乘区间接参与。这种「双变量单一乘区」的设计意图是：防止固伤职业的装备选择被单一属性绑架，但也导致固伤职业的装备筛选复杂度高于百分比职业。

独立攻击加成的稀缺性揭示了更深的控制策略：全版本仅2件SS武器提供独立攻击百分比加成（屠戮之刃+5%、魂·巨剑+7%），所有粉装/史诗首饰均不提供。**独立攻击%加成的出现密度为0.2件/全部位，是所有攻击力相关加成中最稀缺的乘区**。这不是设计疏忽，而是刻意控制固伤数值膨胀的阀门——一旦开放此乘区，固伤职业的伤害上限将突破策划设定的边界。

**佐证**：CC套+110物理攻击→狂战士百分比+5.50% vs 固伤零收益；CC套+120独立攻击→固伤+8.0% vs 百分比零收益；任务20确认独立加成%仅2件SS武器。

## 5. 暴击概率论——线性阈值与期望值补偿

70版本的暴击机制采用**单层倍率模型**（暴击伤害1.5倍，无暴击伤害乘区），因此暴击收益函数是：`期望伤害系数 = 1 + 暴击率 × 0.5`。这意味着暴击率的边际收益是**线性**的——每+1%暴击率恒等于+0.5%期望伤害，不受当前暴击率影响。

CC套+3%暴击带来的暴击期望收益为：狂战士 55%→58%（系数1.275→1.290，+1.18%≈1.2%）、剑魂 50%→53%（系数1.250→1.265，+1.20%）。二者收益几乎一致（1.18% vs 1.20%），差异小于0.02个百分点，原因是线性模型中暴击增益不受基线影响——(58%-55%)×0.5/(1+55%×0.5) ≈ 1.18%，与基线几乎无关。**暴击率收益的线性特征使得暴击加成的相对收益在不同职业间高度一致**（差异仅来自基线的微小非线性影响），这是暴击乘区区别于其他所有乘区的独特属性。

但70版本未开放暴击伤害加成乘区，意味着暴击的「天花板」被设计固定——不存在暴击伤害叠加带来的乘法增长效应。这一设计限制使暴击成为70版本最接近「平行乘区」的加成变量。

**佐证**：CC套+3%暴击→两职业相同时装下收益1.2%（差值≤0.02pp）；150轮验证该数据稳定。

## 6. 特殊力学——出血穿透防御的逃逸通道设计

出血机制在70版本中扮演着独特的角色，其核心特征为**完全穿透物理防御**——出血伤害的计算不受怪物防御减伤公式影响（300/(300+防御)），本质上是策划刻意设计的「无视防御的伤害通道」。Lv21出血基础3,800/秒，配合被动加成可达5,890/秒。出血完全穿透物理防御意味着它在高防御目标上的相对价值远超面板数值。

这一设计可以理解为**逃逸通道**：当目标防御高到使物理伤害趋近于0时，出血提供了一个不依赖防御穿透的伤害来源。三层结构模型中，出血属于「逃逸层」——在地基层（力量/物攻）和增益层（Buff/套装）都被防御抑制时，出血类效果提供了一条独立于防御系统的伤害路径。

与此相关的设计特征是**百分比技能与固伤出血技能的混伤模式**——崩山击同时包含百分比物理伤害和出血固伤部分，这种混合模式使得单一职业可以通过两种不同机制同时对敌造成伤害。

**佐证**：Lv21出血3,800/秒完全穿透防御，0%减伤；狂战士崩山击为百分比+出血固伤混伤模型。

## 7. 固伤 vs 百分比二元论——公式结构差异与4.74倍系统固有频率

固伤与百分比的本质差异在于底层公式结构：

```
百分比：伤害 = 武器物攻 × (1 + 力量/250) × 技能百分比
固伤：   伤害 = 技能基数 × (1 + 独立/250) × (1 + 力量/250)
```

差异在于百分比引入「武器物攻」作为乘数（受武器强化放大），固伤使用「技能基数」（仅受独立攻击放大）。当一套装备同时提供力量和独立攻击时，它对两种输出类型产生了数量级不同的收益。

CC套在两职业间的收益差异是论证此二元论的**完美实证**：剑魂（纯百分比）+43.95% vs 狂战士（固伤为主）+9.27% —— 差异来源于剑魂享受力量+物理攻击双乘区放大，而狂战士固伤仅享受独立+暴击。

由此提炼出**边际对偶常数（4.7409倍）**：剑魂百分比CC套收益 / 狂战士固伤CC套收益 = 43.95% / 9.27% ≈ 4.7409。这个比值在149轮及以上稳态核查中每次都精确重现，被确认为**系统固有频率**——它反映了70版本两个输出流派在相同装备输入下的响应倍数差异，本质上等于：

```
(1 + 力量收益) × (1 + 物攻收益) × (1 + 暴击收益)[剑魂]
─────────────────────────────────────────────
(1 + 独立收益) × (1 + 暴击收益)[狂战士固伤] - 1
```

4.7409倍不是偶然数值，它是公式架构不对称定律的量化体现。当未来的跨版本研究对比不同版本的该比值时，可以量化地评估策划每一次平衡调整对固伤/百分比平衡性的影响。

**佐证**：剑魂+43.95% / 狂战士固伤+9.27% = 4.7409，跨150+轮精确重现，commit 7e9ae9a/3f01496 等均可复现。
