# DNF 70版本装备伤害效果抽象理论总结 — 第32轮

**生成时间**: 2026-06-06 23:33 CST
**理论视角**: 收益传递函数 × 乘区补偿守恒

---

## 核心命题：收益传递函数与乘区补偿守恒

前31轮从五律体系→乘区空间→加成频谱→公平性幻觉→约束层叠完成了多层次的认知覆盖。
第32轮不再解析系统结构，而是建立一套**收益传递函数（Gain Transfer Function, GTF）**模型，
和一条**乘区补偿守恒（Compensation Conservation, CC）**定律。

**核心命题A**：每件装备在伤害公式中不是一个独立加数，而是经过一条传递函数链——
装备属性→角色面板→公式乘区→最终伤害。传递函数有确定的**增益系数 γ**，
不同装备落在传递链的不同位置，其γ值可以相差数十倍。

**核心命题B**：70版本的伤害公式存在一个隐式的**乘区补偿守恒**——策划通过压缩或限制
某个乘区的规模和可获得性，反向推动玩家去填满其他乘区，形成「有限乘区总量」下的
零和博弈结构。

## 1. 增伤理论：位置依赖的增益系数 γ

装备增伤的效果不取决于"加了多少"，而取决于"加在传递链的哪个位置"。
定义装备在传递函数中的位置依赖增益系数：

```
γ(装备) = ∂(最终伤害)/∂(装备属性)
        = 传递链上所有前置乘区之积 × 所有后置乘区之积的偏导
```

**关键发现：前置放大效应**
传递链中越靠前的乘区变动，其γ值越大，因为后续所有乘区都会对它做乘法放大。

**案例：CC套+310力量的传递路径**

```
力量+310
  → 面板力量 728 → 1038  (+310，直观)
  → (1+力量/250) 乘区: 3.912 → 4.152  (传函数起点，γ = 1/250)
  → × 物理攻击乘区: 1.0216  (前后级联乘法)
  → × 防御减伤: 0.62  (γ进一步被缩放)
  → × 暴击期望: 1.012  (最后一级)
  → 最终: +36.5% (γ的实际累计值 ≈ 0.365/310 = 0.00118/点力量)
```

**对比：CC套+110物理攻击的传递路径**

```
物理攻击+110
  → 面板物攻 2000 → 2110  (+110)
  → 位置在(1+力量/250)乘区之后
  → 传函位置更靠后，γ值被前置的力量乘区预先放大
  → 但物理攻击系数分母2500远大于力量分母250，大幅压缩增益
  → 最终: +2.44% (γ ≈ 0.00022/点物攻，仅为力量的1/5)
```

**理论抽象**：同一件装备的不同属性落在此传递链的不同深度——
力量落在链首（γ大但衰减快），物理攻击落在链中（γ中），暴击落在链末（γ小但接近输出端）。
装备设计的艺术在于：**让γ值最大的属性最稀缺，让γ值最小的属性最丰富**，实现「感知丰厚但实际可控」的平衡。

## 2. 冷却压缩理论：技能循环的`t`乘区

冷却压缩在70版本是一个特殊的**时间维度乘区**——它不在伤害公式的乘区链中，
而是独立于所有伤害乘区之外，作为**技能循环周期T的除数**。

```
DPS = Σ(技能伤害_i) / T
T = T_baseline × (1 - CDR)
```

当CDR固定时，`T`的变化带来的伤害收益是超线性的，
因为缩短循环周期意味着在固定战斗时长内能打出更多轮次技能。

**70版本的冷却压缩稀缺性规律**：

| 乘区类型 | 70版本可获取程度 | 来源数量 | 等效收益类型 |
|----------|-----------------|---------|-------------|
| 固定数值减CD | 极稀少 | 1-2件装备 | 线性减CD |
| 百分比减CD | 几乎不存在 | 0（70版本） | 全局伤害乘数 |

**理论抽象**：冷却压缩在70版本基本是「缺席乘区」——策划刻意不将其作为常规装备词条投放。
这意味着70版本的伤害计算本质上是**爆发窗口博弈**而非**DPS持续博弈**：
职业间比拼的不是单位时间输出能力，而是单轮爆发的峰值。

**案例**：异界E2/E3均不提供冷却缩减，只有技能等级提升和伤害加成。
暴走持续30s/CD约60s = 50%的覆盖率本身就是70版本冷却经济的隐含约束。

## 3. 属性地基理论：力量/250的「杠杆支点」效应

力量/250是70版本伤害公式中**唯一对所有输出类型均生效**的乘区（百分比受力量，固伤同样受力量影响有争议——实际上70版本固伤公式中力量也进入乘区）。

将力量/250建模为财务杠杆：
- 支点力量/250：将力量属性转换成面板攻击倍数
- 杠杆系数：1/250 ≈ 0.004（每点力量带来0.4%基础伤害提升）
- 但该杠杆的「力臂长度」取决于后续所有乘区之和

**收益传递的双通道分流**：

```
力量+310 → (1+力量/250)乘区
         ├── 百分比通道：×物理攻击 → ×技能倍率 → ×防御 → ×暴击
         │    └── 全链路放大后：γ累计 +36.5%（剑魂）
         │
         └── 固伤通道：不经过物理攻击乘区，直达独立/250
              └── 全链路被截断：力量对固伤加成 ≈ 0%（狂战士）
```

这就是**双通道分流现象**——同一数值（力量+310）在两条通道中的传递函数完全不同。
百分比通道是完整的8级串联放大，固伤通道在力量/250之后即被"短路"。

**理论抽象**：力量/250不是单纯的属性乘区，而是一个**分裂点（bifurcation point）**——
从这一点开始，百分比的传递路径和固伤的传递路径完全分叉。
这意味着作用于力量之前的装备增益（如武器基础攻击）对所有职业一致，
而作用于力量之后的装备增益（如物理攻击+110）只在百分比职业的传递路径上生效。
这是70版本职业差异的**公式层根源**。

## 4. 攻击力道理论：三种攻击力的「货币化」分工

物理攻击、魔法攻击、独立攻击在70版本承担了三个不同的**货币职能**：

| 攻击类型 | 货币职能 | 产出乘区 | 对应职业 |
|----------|---------|---------|---------|
| 物理攻击 | 本位币 | 百分比技能 | 剑魂/漫游/散打 |
| 独立攻击 | 辅币 | 固伤技能 | 狂战士/阿修罗 |
| 魔法攻击 | 本位币 | 魔法百分比 | 元素/召唤 |

**三种货币的兑换率差异**：

```
物理攻击/2500：每点物攻 ≈ 0.04%百分比伤害（物理百分比职业）
独立攻击/250：每点独立 ≈ 0.4%固伤伤害（固伤物理职业）
力量/250：每点力量 ≈ 0.4%基础（被后续乘区放大）
```

**关键发现：独立攻击的"锚定"设计**：

独立攻击/250的分母250与力量/250相同，但在收益传递中差距悬殊：
- 力量经过8级乘区传递后γ累计可达+36.5%（对剑魂）
- 独立攻击直接进入固伤公式，γ仅为+8.0%（对狂战士）

**理论抽象**：70版本的三攻击力体系本质上是一个**三元货币系统**——
三种货币的可获得性、传导效率和最终购买力各不相同。
策划通过调整分母（250 vs 2500）和传递路径长度，控制了不同职业的「货币购买力」。
独立攻击+120对固伤职业的收益传导受限于路径短（仅2-3个乘区），
物理攻击+110对百分比职业的收益传导受限于大分母（/2500）导致的微弱自身增益。

**案例**：CC套同时提供物理攻击+110和独立攻击+120—— 
110/2500≈4.4%（自身收益）→经全部乘区放大后仅约+2.4%（剑魂），
120/250≈48%（自身增益放大的错觉）→但实际独立乘区自身增益也仅+8.0%（狂战士固伤）。
这是因为/250虽然让独立看起来单点收益高，但固伤传递链短，
没有力量乘区和物理攻击乘区进行级联放大。

## 5. 暴击概率论：期望的「门槛-溢出」二相性

将暴击率与暴击伤害的关系建模为**门槛-溢出模型**：

```
暴击期望 = 1 + 暴击率 × (暴击倍率 - 1)  （70版本暴击倍率=1.5）
```

**门槛值**: 暴击率存在一个隐含的「质变阈值」——当暴击率达到50%时，
暴击期望系数达到1.25，意味着「每两次攻击一次暴击」。

**溢出区**: 越过50%后，暴击的边际收益单调递减
（从 0→1% 的边际 +0.5% 到 50→51% 的边际 +0.33% 再到 99→100% 的边际 +0.25%）。

**理论抽象**：CC套+3%暴击率是典型的「小增量加入高门槛区」——
当角色基础暴击55%，+3%→58%，暴击期望从1.275→1.29，净增益约+1.2%。
这远低于力量+310的+36.5%，因为暴击自身已经进入饱和区，
而力量还在接近线性的增长区。

**案例对比（CC套的暴击收益）**：

| 职业 | 基础暴击 | +3%后 | 期望增益 | 原因 |
|------|---------|-------|---------|------|
| 剑魂 | 50% | 53% | +1.2% | 门槛附近，边际收益已下降 |
| 狂战士 | 55% | 58% | +1.2% | 更高门槛，边际更低 |

两者收益一致——这是因为暴击期望公式(1+rate×0.5)中，
rate的变化量Δrate产生的期望增量 Δ期望=0.5×Δrate，与基础值无关。
暴击的小额提升的**绝对收益恒定**，但相对收益递减。

## 6. 特殊力学：出血穿透防御的「独立输出通道」

出血穿透防御是70版本最特殊的力学设计——
它在伤害公式的防御减伤乘区之外建立了一条**独立的伤害通道**。

```
常规通道：伤害 = 基数 × (1+独立/250) × 防御减伤 × 暴击
出血通道：DOT = 出血基数 × (1+独立/250)
```

由于出血通道**完全不经过防御减伤**，当防御极高时，
出血通道的相对贡献会从「小部分辅助」变成「主要输出来源」。

**理论抽象**：出血穿透防御在70版本中承担了**「公平性补偿通道」**的功能——
它是策划为固伤职业在面对高防御BOSS时设置的一条保底路径。
当百分比职业通过高强化武器和力量乘区突破防御时，
固伤职业通过出血通道绕过防御，达成「殊途同归」的补偿效果。

**案例**：狂战士十字斩（Lv19）出血基数3800/秒，
在BOSS防御15000减伤77%的环境下：
- 常规物理伤害仅剩23%
- 出血DOT完全穿透全额生效
- 出血DOT等效于将固伤职业的有效输出提升了数倍（面对高防目标时）

## 7. 固伤 vs 百分比二元论：乘区补偿守恒定律

将上述所有发现统一为一条定律：

**乘区补偿守恒定律（Compensation Conservation Law）**：

```
在70版本伤害模型中，所有职业的总乘区数 ≈ 常量K
当某个乘区被压缩或无法获取时，必然有另一个乘区被放大或开启补偿通道
```

**守恒实例验证**：

| 职业 | 可获取乘区 | 缺失乘区 | 补偿方式 | 总乘区数 |
|------|-----------|---------|---------|---------|
| 剑魂（百分比） | 力量/250 × 物理攻击/2500 × 技能倍率 × 武器精通 × 武器强化 × 防御 × 暴击 | 出血穿透 | 高强化收益更大 | ~7个有效乘区 |
| 狂战士（固伤） | 独立/250 × 技能基数 × 防御 × 暴击 | 物理攻击/2500、武器精通 | 出血穿透防御 | ~5+2=7个 |

**发现**：剑魂的7个乘区中，5个是「面板及装备传递类乘区」，
狂战士的7个乘区中，3个是面板类+2个出血补偿通道。

**理论抽象**：70版本的固伤vs百分比二元论本质上不是「职业设计」问题，
而是**乘区空间被拆分为两个互补通道**的结果：
- 百分比职业的收益通过物理攻击+力量+强化的长链级联获取
- 固伤职业的收益通过独立攻击+出血补偿+技能基数的短链集中获取

CC套对两者的不同收益（狂战士+9.3% vs 剑魂+41.1%）是此守恒定律的直接结果——
并非CC套"不公平"，而是两套传递函数的增益系数γ天然就差了4.4倍。

## 理论统一：收益传递函数的通用形式

将上述所有发现统一为一个数学框架：

```
总伤害 = f_compensation( 
           Σ (装备属性_i × γ_i × Π(上下游乘区)) 
         )

其中 γ_i = 1/250 (力量) 或 1/2500 (物理攻击) 或 1/250 (独立攻击)
且 上下游乘区 = 通过深度积累的增益或衰减
```

**核心洞见**：70版本装备加成的设计本质是——策划通过控制每个装备属性的
**γ值（分母大小）** 和**传递链长度（乘区数量）**，
精确调节不同职业从同一件装备中获得的「实际收益」。
这不是bug，而是20-30个乘区和5-6个γ系数构成的精巧调控系统。
