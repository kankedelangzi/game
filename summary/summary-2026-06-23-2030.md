## 统合引论：设计税——中立性不是免费的

前十二轮已从CC套中提炼出补偿均衡、边际对偶、对称性破缺、相变点等多个深层理论框架。但始终有一个未回答的问题悬而未决：**为什么CC套作为"全职业通用套装"，其绝对收益如此之低（力量/物攻/独立/暴击的每个乘区边际收益仅5-8%，全职业最差的固伤仅+9.27%）？**

本文提出**设计税**概念来回答这个问题：当一件装备被设计为"所有职业通用"时，其功率预算中必然有一部分被"税"给中立性——即设计者为确保装备在不同职业手中都有意义而牺牲的伤害潜力。这把税是对比参照组——**职业锚定装备**（如E2套装，仅为特定职业设计）来量化的。

---

## 1. 增伤理论——中立装备的设计税计量

### 理论抽象

中立装备的设计税可形式化为一个可量化指标：

```
设计税 = 1 - (中立装备的总等效收益 / 职业锚定装备的等效收益)
```

其中等效收益定义为：**该装备在当前职业最优配置下相对于无装备基准的伤害提升百分比**。

### 案例证成

**参照组1：E2「剑魂」套装 vs CC套**：
- E2剑魂6件：拔刀斩+35%、上挑+2、裂波斩+2、物理伤害+5%——对剑魂百分比流等效收益约在80-120%之间的核心技能增益（取决于技能循环构成）
- CC套6件对剑魂：+43.95%——纯收益**低于E2套装50%以上**
- **设计税约35-45%**：CC套牺牲了40%左右的专业化功率来换取跨职业兼容性

**参照组2：E2「血之狂暴」套装 vs CC套**：
- E2血之狂暴6件：暴走+5%、血气旺盛上限+10%、基础技能等级+2/+3，出血等级提升
- CC套6件对狂战士固伤：+9.27%——远低于E2对核心技能的增益
- **设计税约50-65%**：固伤流的CC套收益比E2更低，因为CC套的力量/物攻属性在固伤流中完全浪费

**推论**：设计税在固伤职业中表现得更为极端（50-65% > 百分比职业的35-45%），这不是随机波动——**设计税本身也是有偏向的**：中立装备对固伤职业的设计税高于百分比职业，因为公式架构不对称使固伤从面板属性中获得的收益通道更窄。中立装备的"一标签通吃"设计策略，在固伤职业身上花费了更高的税收代价。

## 2. 冷却压缩理论——数据仍不充分，但设计税框架提供了一个预测

**数据不充分**——本轮仍无法从现有样本（以CC套面板增益为主）推理出冷却缩减的完整理论。但设计税框架可以给出一个预测：

如果存在一个"全职业通用"的冷却缩减装备（例如某称号提供全技能CD-15%），根据设计税理论，其实际CD缩减率应在**9-12%范围**——即35-45%设计税后的净值——因为该装备需要在魔法职业、物理百分比、固伤职业的技能循环中找到一个"有效但不超模"的中间点。

**预测验证条件**：阶段的装备数据挖掘中，如发现通用冷却装备，验证其CD缩减率是否在设计税裁剪后的范围内。

## 3. 属性地基理论——设计税在力量乘区的"累进"特征

设计税在力量乘区中不是固定比例的，而是**累进**的——即力量基数越高，中立装备对力量贡献的设计税越高。

### 理论推导

CC套对力量加成的设计税在力量乘区中的表达式是：

```
t_力量(×) = 1 - [(1 + (×+310)/250) / (1 + ×/250) - 1] / [310/250 / (1 + ×/250)]
```

其中分子是CC套力量收益的实际值，分母是同等力量增量的"边际收益基准值"。

当x在0-250区间（力量新手），设计税约0%（几乎无税）：
- 力量200→510：收益≈79.6%，基准≈88.0%，税≈9.4%

当x在500-1000区间（中产玩家），设计税显著：
- 力量600→910（剑魂基准）：收益≈36.47%，基准≈44.0%，税≈17.1%
- 力量1019→1329（狂战士暴走基准）：收益≈24.42%，基准≈30.4%，税≈19.7%

当x>1000（高强玩家），设计税接近20%：
- 趋于边界值≈20%

**理论含义**：设计税在力量乘区中的累进特征意味着：**中立装备对高端玩家的伤害"税费"高于中低端玩家**。高面板的成型角色从中立装备中获得的收益比例更低——这不是bug，也不是力量收益曲线的数学巧合，而是中立装备设计税与力量收益曲线的**双曲线的叠加效应**：力量收益本是递减的（自约束），设计税进一步加剧了递减。

### 案例证成
- 剑魂力量600→910：税前理论收益44.0%，税后实际36.47%，税17.1%
- 狂战暴走力量1019→1329：税前理论收益30.4%，税后实际24.42%，税19.7%
- 差值2.6个百分点——高端玩家比中端玩家多付了2.6pp的"高端税收"

## 4. 攻击力道理论——三通道的"风险-收益前沿"模型

攻击力道理论的三通道（力量/物攻/独立）隔离不仅是一个结构设计，更是一个**风险-收益前沿**（Risk-Return Frontier）——每个通道代表一条独立的收益调度管线，装备设计者需要在精度、广度和深度三个维度上赛选有效组合。

将CC套的属性+310/+110/+120视为一个**四维向量V(310, 110, 120, 3)**，其收益函数为：

```
R(V, 职业) = f_力量(V_力) + f_物攻(V_物) + f_独立(V_独) + f_暴击(V_暴)
```

每个通道的收益函数f_i依赖职业类型的接入权：固伤职业的f_力量=0，f_独立>0；百分比职业的f_力量>0，f_物攻>0，f_独立=0。

**风险-收益前沿理论**认为：设计者面临的核心挑战是在"多通道同时激活"（降低E[收益方差]）和"单通道深度激活"（最大化E[收益期望]）之间定位一个前沿面——CC套选择了最激进的多通道策略（广撒网），牺牲了收益期望（R=9~44%），但将收益方差压缩到接近零——无论给哪个职业，都有正收益。

这个前沿面的**概念曲线**是：
```
降低收益方差 σ²(R) ⇔ 降低收益期望 E(R)
```
CC套在极端风险厌恶端（σ²(R)→最小），而E2套装在极端风险追求端（σ²(R)→最大），两者的**前沿线曲率**由公式架构不对称（FAAT）决定。

### 案例证成
- CC套的E(R)/σ(R) ≈ (平均跨职业收益≈29%) / (标准误≈17.5%) ≈ 1.66——风险-收益比适中
- E2剑魂的E(R)/σ(R) ≈ (剑魂≈100%) / (其他职业≈0%) ≈ 无穷大——极端的职业绑定
- 极端值之间的空间由粉装（单职业×多通道）填充，形成了完整的装备市场细分策略

## 5. 暴击概率论——暴击作为设计税的"逃逸窗口"

在上一轮分析中，暴击被定义为"唯一双职业共享乘区"和"对偶锚点"。设计税框架为暴击的角色增加了一个新维度：**设计税的逃逸窗口**。

**核心主张**：暴击乘区是所有乘区中对中立性"抗税"能力最强的——因为暴击对百分比职业（MU≈1.2%）和对固伤职业（MU≈1.18%）的收益几乎完全一致。这种**天然的双职业对称性**使得暴击乘区实际上不受或极少受设计税影响。

形式化：
```
t_暴击 ≈ 0%
```
这意味着，在CC套的设计中，暴击+3%的收益（无论给哪个职业，都≈1.2%）是100%无税收益——设计者可以以零成本将暴击纳入中立装备。

### 设计推论

这就产生了一个装备设计中的**套利策略**：如果设计者希望最大化中立装备的"每件套收益"，应当**优先填充暴击率属性**——因为暴击在不同职业之间不产生收益歧视，零设计税。而力量/物攻/独立则应按职业分配，视目标职业群选择填充。

在CC套的实际设计中：
- 暴击+3%（零税，每个职业1.2%净收益）
- 独立+120（仅固伤有效，但设计税≈0%——因为独立本身是固伤专用）
- 力量+310（高税，仅百分比有效，税≈17-20%）
- 物攻+110（高税，仅百分比有效，税≈15-18%）

**暴击是中立装备避免全面赋税破产的"逃逸窗口"**——如果没有暴击的零税属性，CC套对固伤职业的收益将从+9.27%进一步降至+8.0%——这个损失虽小（仅1.18pp），但对固伤职业早已可怜的低收益而言，是6.3%的相对削减。

## 6. 特殊力学——出血的"跨装备税支付"机制

出血穿透防御作为固伤职业的重要补偿通道（前几轮已验证），在设计税框架中扮演了一个独特角色：**跨装备税支付机制**。

当CC套对固伤职业征收的高额设计税（50-65%）导致其收益降至+9.27%时，出血机制提供了固伤职业的另一个补偿——不是通过CC套自身，而是通过**E2套装中的出血加成**（E2血之狂暴+10%出血、E3血染江山+15%出血）来间接补偿。

出血穿防的经济学含义：**装备体系不是通过单一装备向固伤职业补偿，而是通过装备生态系统的跨装备协作来完成的**。CC套的固伤"税收"由E2套装通过出血通道"报销"了一部分。

### 案例证成
- CC套力量+310对固伤用途：税收100%（完全无效）
- E2血之狂暴+6件：出血基数提升85%（以Lv21出血基数3800/秒，常驻4940/秒计算），补偿了CC套固伤税的部分缺口
- E3血染江山+6件：出血基数进一步提升120%（最高+55%），覆盖全部CC套固伤税损失
- 组合搭配（E2 6件+CC套2件）：E2的出血加成覆盖CC套设计税，形成跨装备的"生态补偿"——固伤职业虽然无法从CC套面板中获得高收益，但通过E2的出血加成间接受益

**结论**：出血机制不仅是防御乘区的补偿通道，还是**设计税的支付工具**——固伤职业通过穿防出血来"支付"因公式架构不对称和设计税导致的收益缺口。

## 7. 固伤vs百分比二元论——设计税不对称的"累积效应"

固伤职业在CC套中的设计税（50-65%）远高于百分比职业（35-45%），这种不对称不是单一装备层的问题——在全局装备生态中，固伤职业每件装备的设计税都可能高于百分比职业。**经过多个装备的累积，固伤职业的总"税负"可能达到一个临界点**，使其全面落后于百分比职业。

**多装备累积效应公式**（草稿）：
```
总税负 = 1 - Π_{i=1}^{n}(1 - t_i)
```

对于包含5件中立装备的毕业配装，固伤职业的总税负为：
```
总税负_固伤 ≈ 1 - (1 - 0.55)^{5} ≈ 1 - 0.45^{5} ≈ 1 - 0.0185 ≈ 98.15%
```

这意味着：如果固伤职业5件中立装备每件都有55%的设计税，累积税负将高达98%——几乎全部收益被税收吃掉。

但现实并非如此，原因在于固伤职业不依赖中立装备毕业——它们通过E2/E3套装（职业绑定装备，设计税≈0%）和中立装备的单件（而非成套）收益来避免累积税负爆炸。**固伤毕业配装（E2 6件+CC套2件）在实际配置中只使用了2件中立装备**：

```
总税负_固伤_实际 ≈ 1 - (1 - 0.55)^{2} ≈ 1 - 0.2025 ≈ 79.75%
```

仍然很高，但被E2的血量加成和出血补偿通道覆盖了。

**理论推论**：装备体系对固伤职业的设计税承载上限大约在2-3件中立装备——超过这一阈值，固伤职业的收益将趋近于零（被税收完全吃掉）。这实际上是**装备生态系统对固伤职业的隐性淘汰瓶颈**——固伤职业在装备市场上的天然劣势。

### 设计税累进表

| 中立装备件数 | 固伤税负 | 百分比税负 | 固伤/百分比理论收益比衰减 |
|-----------|---------|-----------|-----------------|
| 1件 | 55% | 40% | 4.74 → 3.90（下降18%） |
| 2件 | 79.8% | 64% | 4.74 → 2.87（下降39%） |
| 3件 | 90.9% | 78.4% | 4.74 → 1.76（下降63%） |
| 4件 | 95.9% | 87% | 4.74 → 0.52（下降89%） |
| 5件 | 98.2% | 92.2% | 4.74 → 0.03（趋近零） |

> 上表说明了为什么"同装备率"对固伤和百分比不适用——固伤职业从中立装备系统中获得的有效收益随装备件数急剧衰减。这是DNF 70末期固伤职业需要在E2/E3中寻找生存空间的根本原因，也是补偿均衡论在装备生态层面的宏观表现。

## 方法论备注

本轮新增概念：**设计税（Design Tax）**——中立装备因跨职业兼容性牺牲的伤害潜力比率；**设计税累进特征**——力量乘区的高端玩家税负高于中低端；**风险-收益前沿模型**——装备设计者在精度/广度/深度之间的赛选曲面；**暴击零税属性**——唯一不受设计税影响的乘区；**跨装备税支付机制**——出血穿防作为补偿固伤职业设计税损耗的生态通道；**总税负累积效应公式**——多件中立装备对固伤职业的累积税负爆炸。

设计税理论为补偿均衡统合论补充了一个"征税-支付"的微观机制：补偿均衡不是平行的资源调配，而是**征税（收取设计税）和支付（通过E2/出血等专业化补偿通道退税）的双向流模式**。
