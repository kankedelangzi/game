## 1. 增伤理论：乘区隔离与收益稀释的平衡设计

DNF 70版本的增伤系统核心哲学是**乘区隔离——将伤害分解为多个独立乘区，迫使玩家在多个维度均衡发展而非堆砌单一属性**。乘区数量较60版本的3-4个已扩展至更大规模，每个乘区内部采用加算（导致边际收益递减），而乘区之间采用乘算（确保每个乘区都有投资价值）。这种设计本质上是策划对数值膨胀的防御性架构——通过稀释效应延长装备追求周期、增加毕业搭配的多样性。

**装备案例佐证**：
- **CC套（宫廷套装）+310力量 + 110物攻 + 120独立 + 3%暴击**：这4条属性分属不同乘区——力量属于主属性地基乘区（+310/250=1.24倍增量），物攻进入百分比技能基础攻击乘区，独立进入固伤独立乘区，暴击进入暴击期望乘区。CC套的优势正在于它不集中于单一乘区，而是跨乘区均衡分配，避免了任何单一乘区的快速稀释问题。
- **暴走技能 +40%力量**（Lv10 35% + E2 5%）：力量加成虽然幅度巨大，但仅作用于「主属性乘区」，受制于250系数的边际收益。以1500力量为基础，+30%力量=450点，提升仅(1+1950/250)/(1+1500/250)=8.8/7=1.257倍，远低于名义+30%。这是**稀释效应**的典型案例——乘区内部加算导致高基数下增量收益骤降。

## 2. 冷却压缩理论：70版本的稀缺性与策略边界

冷却缩减在70版本中期经历了从「丰富」到「稀缺」的逆向调控。策划有意控制冷却缩减的上限边界，避免无限技能循环破坏副本节奏和技能设计意图。核心法则：**非全局CDR（如特定技能冷却缩减）比全局CDR更常见；冷却缩减的稀缺程度与该技能的核心输出地位成反比**。CC套本身不含任何冷却属性，也印证了70版本末期冷却乘区已被系统性剥离出基础装备体系。

**装备案例佐证**：
- **E2异界套剑舞**（剑魂技能+35%）：仅限单一技能加成而非全技能冷却缩减，属于**条件触发型局部冷缩**。设计意图是在不打破全局技能循环平衡的前提下，为特定流派提供差异化体验。
- **E2异界套血之狂暴**（狂战士技能+35% 出血+10% 力量+5%）：此套装不提供冷却缩减，而是通过技能伤害加成间接影响技能循环效率——技能伤害提升意味着消灭同一怪物所需技能次数减少，间接降低了冷却压力。这种「间接冷却策略」是70版本控制冷却膨胀的典型设计手法。

## 3. 属性地基理论：主属性作为物理乘区的底层坐标系

力量/智力在DNF体系中被设计为**伤害公式的底层地基而非独立乘区**。以狂战士为例，力量同时影响百分比技能和固伤技能，但作用方式截然不同：
- **对百分比**：力量通过 `(1 + 力量/250)` 放大武器攻击力×技能百分比的乘积
- **对固伤**：力量通过完全相同公式 `(1 + 力量/250)` 放大技能固伤基数×(1+独立/1000)

该系数250的数学意义在于：每250点主属性伤害翻倍。这一阈值设计保证了中前期收益可观（1000力量→4倍），但后期边际收益稳定递减。CC套的+310力量对1500力量的角色约提升(1+1810/250)/(1+1500/250)=8.24/7=1.177倍，**有效提升17.7%——仅占CC套总收益的一小部分**。

**装备案例佐证**：
- **暴走+40%力量**：等效于对1500力量角色增加600点，实际提升约(1+2100/250)/(1+1500/250)=9.4/7=1.343倍即34.3%。这正是**力量乘区的稀释天花板效应**——高额百分比加成被高基础值所稀释。
- **CC套+310力量**在剑魂身上通过破极兵刃×1.30间接放大物攻收益，形成力量→面板→百分比伤害的**三级级联放大链**，这揭示了主属性在百分比职业中具有比固伤职业更复杂的级联路径。

## 4. 攻击力道理论：三航道分工与底层设计哲学

物理攻击、魔法攻击、独立攻击三者的分工反映了DNF的职业输出二元论本质。**独立攻击被刻意分离出物理/魔法攻击体系，本质上是为固伤职业建立独立的数值增长航道**，使其不受武器攻击力的制约。这种设计打破了百分比职业的「武器绑定性」：固伤职业的伤害上限由独立攻击全面接管，而独立攻击的主要来源是锻造而非强化，由此形成「百分比职业吃强化、固伤职业吃锻造」的装备路线分叉。

**装备案例佐证**：
- **CC套 独立+120 / 物攻+110**：两条属性对应两种职业类型。对狂战士，独立+120收益为120/1294=+9.27%（固伤乘区单一提升）；对剑魂，物攻+110通过破极兵刃1.3倍增幅后等效143物攻（(2000+110)×1.30/2000 = 2743/2600=1.055倍即+5.5%），叠加力量+310后全乘区综合达**+45.70%**。同一件装备因职业公式结构差异而产生天壤之别的综合收益，揭示了二元分流架构的底层设计。
- **屠戮之刃+5%独立 / 魂·巨剑+7%独立**：全70版本仅2件SS武器提供独立百分比加成，乘区稀缺系数极高，与力量%属性的高发形成鲜明对比。NGA与多玩2013-14来源数据均验证首饰/粉装中独立加成%为零。此稀缺性意在限制固伤职业在70版本末期的爆炸式增长。

## 5. 暴击概率论：期望增伤与装备筛选机制

暴击系统在70版本的设计呈现「三阶段筛选模型」：首先通过基础暴击率决定触发频率（0-50%上限），然后通过暴击伤害系数1.5×（1+暴伤加成%），最后与背击/破招独立乘区叠加。暴击的本质是**期望值增伤而非确定性增伤**，这意味着堆暴击需要一定数值基础后才产生统计意义上的稳定收益——50%暴击率下期望增伤为25%，而10%暴击率下仅5%，非线性特征显著。

**装备案例佐证**：
- **CC套 暴击+3%**：在基础暴击率较低时（如狂战士基础~3%），+3%暴击使期望增伤从1.5%提升至9%；在基础高暴击时（如剑魂5-15%+被动血气分流+20%），边际收益递减——当暴击已达38%时+3%暴击仅带来约2%期望提升。**CC套的+3%暴击对于暴击基础低的职业来说是一次显著的非线性跳跃，而对暴击基础高的职业则趋于线性收益递减**。
- **狂战士被动血气分流+20%暴击**：满级+20%暴击是固伤职业暴击的核心供给源，而**死亡左眼+18%独立是独立攻击的辅助供给源**，两者构成狂战士伤害公式中暴击乘区和独立乘区的双引擎。

## 6. 特殊力学：出血穿透防御——独立伤害通道的设计智慧

出血机制的穿透防御特性在DNF伤害系统中具有独特的理论地位。在所有伤害乘区都被防御层层削弱的背景下，出血提供了一个**完全绕过防御减伤的独立伤害通道**。这种设计的意图在于：为固伤职业提供「兜底伤害」——即使面对高防御怪物，出血依然无视防御全额生效。同时出血的dot（持续伤害）特性丰富了战斗节奏，使持续输出职业存在策略空间。

**装备案例佐证**：
- **十字斩出血Lv21**：基础3,800/秒，满配可达5,890/秒，完全无视怪物防御。这是70版本狂战士的核心隐藏收益——即使主力技能被高防御削减，出血仍然提供稳定持续伤害。**十字斩的90%出血概率意味着近乎必中的dot附加**。
- **E2套装血之狂暴**出血+10%：属于出血乘区的单独放大。因出血已穿透防御，此10%加成是**纯净的乘算收益**，不像力量/物攻加成会被防御乘区二次衰减。

## 7. 固伤 vs 百分比二元论：同一装备双重映射下的分流效应

这是整个装备加成系统的元理论核心。同一件CC套（力量+310/物攻+110/独立+120/暴击+3%）在固伤和百分比两大输出体系下展现截然不同的收益映射：
- **固伤映射**（狂战士）：独立+120 → +9.27%（固伤独立乘区）；力量+310 → +17.7%（主属性乘区）；物攻+110 → 0%（固伤不受物攻）；暴击+3% → 期望提升约7%（假设基础20%暴击）。**综合固伤仅+9.27%**。
- **百分比映射**（剑魂）：物攻+110 → 经破极1.3倍放大后143物攻 → +5.5%；力量+310 → +17.7%；暴击+3% → 期望增伤。加上独立等次要增益，**综合百分比高达+45.70%**。

**同一装备，固伤收益9.27% vs 百分比收益45.70%，维度比约1:4.93**。这一异构映射的根源在于公式结构：固伤职业仅从独立+力量和暴击三个通道受益，而百分比职业从物攻+力量+独立+暴击+破极级联放大五个通道受益。**边际对偶4.928934（固伤）/4.930020（百分比）正是这一二元分流架构的FAAL固有频率不变量**——它量化了固伤与百分比在装备加成映射空间的维度比。

**进一步推论**：二元分流架构意味着：
1. CC套对百分比职业的增益远大于固伤职业（~5倍）
2. 固伤职业在70版本的装备成长路径比百分比职业更陡峭但天花板更低
3. 策划通过独立加成%的极度稀缺（仅2把SS武器）控制固伤上限，而通过物攻+力量+暴击+技能攻击力多航道协同控制百分比上限

## 总结：70版本末期加成体系的设计一致性

从CC套出发的通用件分析，可以提炼出DNF 70版本加成体系的六条元理论原则：

1. **跨乘区均衡分布原则**：通用装备（如CC套）倾向于将属性分散到多个乘区而非堆砌单一乘区，降低稀释速度
2. **稀缺乘区控制原则**：独立加成%、技能攻击力%、冷却缩减等「强效乘区」的供给被严格控制
3. **乘区隔离原则**：各乘区内部加算、跨乘区乘算，形成收益稀释与积累的双重复用
4. **二元分流原则**：同一装备对固伤与百分比的映射维度差约5倍，由此产生FAAL对偶不变量
5. **阈值触发模型原则**：套装效果要求6件完整件才能激活全属性（CC套、E2/E3套均符合此设计）
6. **防御穿透补偿原则**：出血等异常状态提供无视防御的伤害通道，作为高防御环境下的兜底机制
