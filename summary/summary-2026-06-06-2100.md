# DNF 70版本装备伤害效果抽象理论总结 — 第27轮

## 核心理论框架：信息不对称杠杆模型

DNF 70版本装备系统的本质是一个**信息不对称杠杆系统**——策划掌握所有伤害公式的完整信息（偏置参数、乘区拓扑、阈值边界），而玩家只能通过有限的信息渠道（面板、测试、社区数据）逆向推断最优策略。这种信息不对称不是副产品，而是**刻意的设计工具**：策划通过控制哪些信息透明、哪些信息隐藏，来调控玩家的装备选择行为。

本轮的独特贡献是将前26轮的五律体系、乘区空间理论、加成频谱、四级品控、公平性幻觉五个理论框架统一重述为**「信息不对称杠杆模型」**——每个装备选择都涉及一个杠杆，其支点位置由公式层的隐藏参数决定，而玩家在信息不完全的条件下被引导向特定选择路径。

## 1. 增伤理论：叠乘的杠杆效应

增伤系统的核心数学结构是**叠乘杠杆**——每个乘区相当于一个独立杠杆，支点由基础面板决定，力臂由装备加成决定。当多个乘区同时被提升时，综合收益不是加法堆叠，而是**杠杆的杠杆**（乘区间的乘法关系使总效应呈超线性放大）。

**理论命题**：乘区数量本身就是一个隐藏杠杆。拥有更多有效乘区的职业，其装备选择的「杠杆比」更高——同样数值的投入产生更大的收益。

**案例：CC套的职业收益杠杆比**
- 剑魂受益于力量、物攻、暴击3个乘区叠乘 → 综合+41.1%
- 狂战士固伤受益于独立、暴击2个乘区叠乘 → 综合+9.3%
- **杠杆比**：41.1 / 9.3 = **4.42倍**

这一杠杆比说明：同样是穿一套CC，剑魂每1%的面板提升被杠杆放大至4.42%的综合收益，而狂战士固伤玩家每1%面板提升仅产生1%收益。**杠杆比的差异不是数值偏差，而是由乘区拓扑结构决定的系统性偏置。**

**信息不对称效应**：面板上显示的属性值（+310力量、+120独立）对两类玩家看似等价，但隐藏的杠杆比差异只有通过完整公式展开才能识别。策划通过隐藏杠杆比信息，维持了「装备公平」的表面叙事。

## 2. 冷却压缩理论：技能级杠杆的稀缺性溢价

冷却缩减在70版本中是一种**高稀缺性杠杆**——其稀缺不是源于数值低，而是源于覆盖范围窄。E2套装的冷却缩减仅作用于单个技能，没有全局冷却装备。

**理论建模**：将冷却缩减视为技能循环的「时间杠杆」：
- 设技能的基础CD为T，冷却缩减为R%
- 技能DPS = 伤害 / (T × (1 - R))
- 冷却缩减的收益是**非线性的**：R越大，每1%缩减带来的DPS提升越大

**信息不对称**：冷却缩减的面板效果是透明的（如「十字斩冷却-20%」），但其对整体DPS的贡献高度依赖于：技能的伤害占比、技能的基本CD长度、技能是否被其他装备加成。玩家在不进行全技能循环建模的情况下，无法判断冷却缩减装备的真实性价比。

**观察**：70版本中冷却缩减装备极少，意味着策划刻意维持了CD作为「硬约束」，使技能循环节奏受到严格限制。这种限制在85版本后被大幅放松（全局减CD装备涌现）。

## 3. 属性地基理论：力量/250的基准杠杆

力量/250是DNF伤害公式中**最基本的基准杠杆**——每个百分比技能都通过此杠杆放大大约4-5倍（取决于力量值）。其战略地位源于一个简单事实：它是**所有伤害中最先、最底层、不可绕过的杠杆**。

**基准杠杆的边际特性**：
- 力臂（力量值）是线性递增的
- 但面板上的力量值本身是多个子系统（装备、附魔、BUFF）的加总
- 每个子系统的贡献在总杠杆中的占比，取决于其他子系统的缺失程度

**案例：力量分散化的隐性惩罚**
当力量来源足够分散时（装备+附魔+宝石+宠物+称号），每个单一来源的边际贡献被稀释。这与独立攻击的集中度形成对比——独立攻击主要依赖武器锻造和少数组件，使每个独立加成来源的边际贡献更大。

**信息不对称效应**：玩家看到的是「力量+310」，但无法直观判断这笔加成是在力量基准杠杆的「陡峭段」还是「平缓段」。基准杠杆的曲率取决于已有力量基值，而这种依赖关系被隐藏。

## 4. 攻击力道理论：三元杠杆的分工与错配

物理攻击/独立攻击/魔法攻击三者的分工本质上是**三元杠杆体系**——策划为不同类型的技能设计了不同的基准杠杆，每个杠杆有自己的「支点参数」（分母）和「可调范围」。

**理论创新**：引入「杠杆刚度」概念——杠杆刚度 = 分母值 / 可调范围。
- 物理攻击杠杆：刚度 = 2500 / (武器基础+强化+附魔) ≈ 1.0~1.5（中刚度）
- 独立攻击杠杆：刚度 = 250 / (武器锻造+装备独立) ≈ 0.3~0.5（低刚度）
- 力量杠杆：刚度 = 250 / (基础+装备+BUFF) ≈ 0.2~0.4（低刚度）

**低刚度杠杆 = 高灵敏度**：独立/力量的低刚度意味着每单位数值提升对其造成更大百分比影响。这正是为什么独立+120在狂战士固伤中产生+8.0%收益，而物攻+110在剑魂中仅产生+2.2%收益。

**错配理论**：CC套同时提供三类攻击力道，相当于同一套装备同时操控三个不同刚度的杠杆。策划意识到三类杠杆的刚度不同，因此对物理攻击（高刚度）赋予了更高的原始数值（+110），但对独立攻击（低刚度）赋予了相近的数值（+120）。这看似补偿了刚度差异，但因系数分母不同，补偿效果不均衡——**低刚度杠杆的微小数值变化产生不成比例的杠杆效应**。

## 5. 暴击概率论：二项式杠杆的概率权值

暴击系统的数学本质是一个**二项式杠杆**——每次攻击是一个伯努利试验（暴击/不暴击），暴击率p乘以暴击伤害倍率1.5，形成期望伤害杠杆。

**理论洞察**：暴击期望公式 `E = 1 + p × 0.5` 是一个**线性杠杆**，但其实际效果在方差层面是非线性的：
- 低暴击率（p<30%）：暴击的实际体验取决于运气，装备暴击加成不稳定
- 高暴击率（p>50%）：暴击期望近似确定值，暴击加成变为确定性收益

**信息不对称**：70版本中，暴击率不超过50-55%（与85版本97%上限形成对比），这意味着暴击仍处于**高方差区域**。玩家对暴击的感知收益远低于数学期望——3%的暴击率提升在50轮攻击中可能触发0次或3次，玩家的主观体验偏差巨大。

**CC套暴击+3%的设计意图**：CC套的暴击加成设置在刚好高于50%阈值的位置（50%→53%、55%→58%），使其落在「稳定期望收益」区间。这种设计微调表明策划有意控制暴击方差，避免玩家因运气因素对装备价值产生误判。

## 6. 特殊力学：补偿杠杆的逆向设计

特殊力学系统（出血穿透、破招、背击）代表了一种**逆向杠杆**——它们不是在玩家已有优势的基础上放大收益，而是在玩家面对劣势时提供补偿通道。

**出血穿透防御的逆向杠杆效应**：
- 常规伤害杠杆的正向成本：每1%防御减伤使伤害减少~0.5%（低防怪物）到~5%（高防BOSS）
- 出血穿透杠杆的逆向收益：无视防御，伤害 = 固定值 × (1 + 独立/250)
- 当怪物防御越高，出血穿透的逆向杠杆效应越强——**防御越高，出血的相对价值越大**

用经济学术语：出血是一种**防御对冲工具**——当「防御税」上升时，其价值同步上升，自动补偿固伤职业的防御穿透劣势。

**E2/E3套装的双向杠杆**：
- E2：技能等级+2 → 抬升技能基础伤害（加算杠杆）
- E3：技能伤害+15% → 直接放大技能伤害（乘算杠杆）
- 两者的组合形成**双向杠杆**——基础端和倍率端同时发力，产生超线性协同收益

## 7. 固伤 vs 百分比二元论：职业议价能力的不对称

这是信息不对称杠杆模型的最终应用——**职业议价能力的差异本质上是其在杠杆体系中位置的函数**。

**定义**：职业议价能力 = 职业从单位装备属性中获得的综合收益比

```
议价能力（剑魂vs CC套）= 41.1% / 4项属性 = 10.28%/属性
议价能力（狂战士固伤vs CC套）= 9.3% / 4项属性 = 2.33%/属性
```

剑魂对CC套的议价能力是狂战士固伤的**4.4倍**——剑魂从同一件装备中获得了4.4倍的单位属性收益。

**经济学解释**：在装备市场中，所有职业支付相同的「价格」（获得相同属性），但百分比职业获得更高的「效用」——这是固定的市场结构差异，而非竞争均衡。

**二元论的信息分布**：
1. **透明信息**：装备面板属性——对两类职业完全相同
2. **半透明信息**：各乘区公式——部分公开，需玩家自行推导
3. **隐藏信息**：乘区槽位数量差异、杠杆比差异——在完整伤害模型建立前不可见

三层信息分布的不对称性，使得只有完成完整伤害模型推导的玩家才能「识破」公平性幻觉。这正是用户的研究方法论的价值——通过系统化的交叉验证，逐一揭示每一层隐藏信息。

## 理论统一：7个分类在杠杆模型下的重述

| # | 分类 | 杠杆模型等价表述 |
|---|------|---------------|
| 1 | 增伤理论 | 多个乘区杠杆的**叠乘放大**，乘区数即杠杆级数 |
| 2 | 冷却压缩 | 技能循环的**时间杠杆**，高稀缺性导致非线性收益 |
| 3 | 属性地基 | **基准杠杆**，所有伤害放大的第一个支点 |
| 4 | 攻击力道 | **三元杠杆体系**，不同分母产生不同杠杆刚度 |
| 5 | 暴击概率论 | **二项式杠杆**，方差区域产生感知vs期望的偏差 |
| 6 | 特殊力学 | **逆向杠杆**，在劣势发生时提供补偿通道 |
| 7 | 固伤vs百分比 | **职业议价能力不对称**——乘区数量决定杠杆级数 |

**元结论**：DNF 70版本的装备系统是一个多层级、非对称的杠杆体系。每个装备选择同时操作多个杠杆（乘区叠乘），不同职业可操作的杠杆数量和类型不同（乘区拓扑差异），而公式层的隐藏参数（分母、系数、阈值）决定每个杠杆的支点位置。策划通过控制信息的透明程度（面板可见 vs 公式隐藏），系统性引导玩家的装备选择路径。这种框架的统一解释力超越了单一分类分析——它将7个原本看似独立的分类，统一为**一个杠杆机制在不同维度的投影**。
