## 一、增伤理论

**理论抽象：** 70版本装备增伤系统呈现"稀缺性梯度控制"的典型特征——策划通过控制不同增伤乘区的装备供给量来调控职业天花板。力量/物理攻击/独立攻击/暴击等"面板属性"充足供给（CC套、宝石、附魔均可获取），而"独立加成%"、"技能等级"、"异常伤害%"等"非线性乘区"则被严格限量供给。这种"基础乘区充足+高阶乘区限量"的分层供给策略，确保了大部分玩家能获得基础强度提升，同时为追求极致的玩家留下深度边际——但被限量的乘区恰好决定了上限。

**论据：**
- **充裕层**（面板属性）：CC套力量+310/物攻+110/独立+120/暴击+3%，宝石系统四类属性石可自由镶嵌，多件供给叠加无冲突。
- **限量层**（非线性乘区）：独立加成%仅2件SS武器（屠戮之刃+5%、魂·巨剑+7%），所有粉装0件提供。E2/E3技能等级加成限特定部位（每件+1~2级，集满6件约+6~12级技能）。出血加成长仅屠戮之刃1件提供（+20%）。
- **稀有度-强度关联**：限定供给量越小的乘区，单件收益密度越高——独立加成%单件+5~7%（稀缺），面板独立单件+18~26（充裕）。这种正关联设计迫使玩家在"确保基础供给覆盖率"和"争取限量高位乘区"之间做装备配置抉择。

## 二、冷却压缩理论

**理论抽象：** 冷却缩减在70版本显著处于"被抑制的乘区"状态——装备层面几乎不提供冷却缩减词条，冷却控制完全由职业技能自身承担（暴走/破极兵刃的CD体系）。这一设计暗示DNF 70版本的策划策略是"冷却=职业特征，而非装备能力"——将冷却控制锚定职业身份，而非装备可购买属性。这种设计的后果是：冷却管理能力在玩家之间差异极小（仅来自技能等级差异），装备搭配不会因CD产生流派分化。

**论据：**
- CC套6件套0冷却相关词条，加成全部集中在面板属性。
- E2/E3异界套无冷却缩减词条，仅提供技能等级+1~2和伤害%加成。如果技能等级附带冷却缩短，也属职业被动机制间接产物而非装备主动设计。
- 宝石系统（属性石/强化石）0冷却相关属性。
- 宠物/称号/勋章系统同样0冷却属性。
- **该分类数据充分表明**：冷却压缩并非70版本装备设计维度。与85版本后的异界套冷却缩减词条形成鲜明对照，70版本的冷却维度停留在"职业被动"的原始阶段。

## 三、属性地基理论

**理论抽象：** 力量/智力在70版本伤害体系中的角色可定义为"选择性地基"——它们的乘区地位因职业输出类型而异：在百分比职业的公式中，力量是独立成区的全局倍率（`× (1 + 力量/250)`）；在固伤职业的公式中，力量完全不参与伤害计算。这种不对称不仅是一个数值事实，更塑造了整套装备评价体系：所有提供力量的装备（如CC套+310力量）天然对百分比职业产生高收益，对固伤职业则出现"装备外观属性≠实际收益"的衰减。这导致固伤职业的装备价值需要基于"有效属性"（独立/暴击/属强）而非"面板属性"进行重估。

**论据：**
- 百分比公式中力量以`(1 + 力量/250)`形式全局作用于伤害：力量600→910，收益+36.5%。该乘区与物理攻击乘区（`(1+物攻/2500)`）独立相乘，产生力量×物理攻击的双重放大效应。
- 固伤公式中力量完全缺席：狂战士728→1038（+310力量）对十字斩/怒气爆发等核心固伤技能零收益，固伤仅由`技能基数 × (1+独立/250)`决定。
- 边际收益递减已通过v54-v97多轮交叉验证确认：等量+310力量，基线600→+36.5%，基线728→+31.7%。收益差4.8pp，符合`1/(x+250)`的严格递减函数预期。
- 宝石优先级体系验证此理论：百分比职业**力量石>属强石>暴击石**，固伤职业**属强石>独立石>暴击石**（力量石零收益，归入无效选择集）。

## 四、攻击力道理论

**理论抽象：** DNF 70版本建立了"双通道攻击力道体系"——物理/魔法攻击服务于百分比技能，独立攻击服务于固伤技能，两套系统采用完全不同的系数分母（2500 vs 250），形成了"密度不对称"的底层设计。独立攻击的系数分母250（250点翻倍）使得它成为"高密度属性"——少量独立攻击即可带来可观提升；物理攻击的系数分母2500（2500点翻倍）使物理攻击成为"低密度属性"——需要大量堆叠才有显著效果。然而这种不对称并未导致固伤职业天然优势，因为独立攻击供给总量（CC套独立+120/宝石+30~60）远小于物理攻击供给总量（武器基础2000+CC套+110+强化+36%+破极+30%）。密度和总量的反比例关系，形成了两个渠道的收益均衡。

**论据：**
- CC套独立+120对狂战士1250→1370，收益+8.0%（系数分母250，每点独立提升0.4%）。
- CC套物理攻击+110对剑魂2600→2710，收益+2.2%（系数分母2500，每点物理攻击提升0.04%）。
- 收益密度比：独立攻击每点收益是物理攻击的8.0%/110 ÷ 2.2%/110 ≈ 0.073%/0.020% ≈ **3.6倍**。但独立总供给（宝石+时装+武器基础约1250）远低于物理攻击（武器2000+破极600×系数+强化），两种力量以不同渠道实现均衡。
- 独立加成%（屠戮之刃+5%）是独立攻击的"增强版"——直接作用于固伤技能最终伤害，与面板独立的`1+独立/250`体系叠加而非冲突。这种"面板独立+独立加成%"的双层结构，是70版本唯一为固伤职业提供类似百分比职业"攻击×强化"双维受益路径的设计。

## 五、暴击概率论

**理论抽象：** 70版本的暴击系统是"线性期望模型"的典型应用——暴击伤害固定1.5倍，暴击率通过`1 + 暴击率 × 0.5`转化为期望增益。每1%暴击率提供+0.5%期望伤害增幅（绝对值），在暴击率0%~100%区间内恒定。但由于相对收益计算公式中分母随暴击率递增，导致实际相对收益呈递减形态。70版本装备设计正好利用了这一递减性质——将暴击加成安置在50%~55%的"中等饱和区间"，使暴击词条对最终收益贡献处于"显著但不爆炸"的合理范围（每3%暴击≈+1.2%），起到了"稳健补充"而非"高风险高回报"的设计效果。

**论据：**
- 70版本暴击伤害固定为1.5倍（无额外暴伤装备），验证倍数来源：[DNF Wiki-暴击机制]。`暴击期望 = 1 - p + p × 1.5 = 1 + 0.5p`（p=暴击率）。
- CC套+3%暴击在50%~55%区间：期望从1.25→1.265（收益+1.2%），与55%→58%的1.275→1.29（收益+1.18%）基本一致。
- 注意：暴击从0%→3%时期望收益1.5%（`(1+0.5×3%)/(1+0.5×0%) = 1.015/1.0 = +1.5%`），而从50%→53%仅+1.2%，验证了相对收益递减。
- 宝石系统暴击石：四级暴击石+4%/个，镶嵌建议中暴击>50%后排位下降，与此理论一致——暴击越高，继续追加暴击的边际收益越低。

## 六、特殊力学

**理论抽象：** 70版本中的"出血穿透防御"机制代表了一种"打破封闭公式"的刻意设计——DNF的核心防御公式形成了一个闭合的伤害抑制系统（`防御减伤 = 防御/(防御+等级×200)`），出血伤害完全绕过此系统，开辟了一条独立于物理/魔法壁垒的伤害路径。这种"破格"设计在数值控制上反而比常规系统更严格：出血的绝对值（基础3,800/秒）远低于技能直伤，需要多段叠加和持续维持才能产生有效输出。换言之，穿透防御的"特权"被较低的基数所对冲，形成了"高穿透度×低基数"的平衡。特殊力学在70版本更像是"策略通道"而非"捷径"——它提供差异化路径而非绝对优势。

**论据：**
- 出血伤害公式：`每秒伤害 = 出血固定值 × (1 + 独立/1000)`。完全穿透防御，不受`防御/(防御+等级×200)`影响。
- 狂战士十字斩Lv21出血：基础3,800/秒，最高+55%加成后5,890/秒。对比直伤技能（如怒气爆发基数11,210，出血仅为其~34%）。
- 屠戮之刃的"出血伤害+20%"是70版本唯一能放大出血收益的装备词条。出血流综合收益约+25%（独立+5% × 出血+20%双乘区效果），与魂·巨剑（+7%纯独立加成）对比，出血路径有先天穿透优势但受限于出血装备供给（仅1件）。
- 其他特殊类型数据不足。破招（估计系数×1.25）、背击（×1.3）在70版本存在但未作为装备核心词条，说明特殊力学在70版本停留在"操作技巧补偿"阶段。

## 七、固伤 vs 百分比二元论

**理论抽象：** 70版本固伤与百分比的二元结构，本质上是**"攻击力依赖度"在技能设计层面的极致分化**。百分比技能将全部倍率锚定在面板攻击力上（`技能百分比 × 面板攻击力`），使攻击力成长性线性放大于技能伤害；固伤技能则将伤害锚定在技能基数上（`技能固定基数 × (1+独立/250)`），使技能等级成为核心成长通道。这一设计导致了装备评价体系的根本性分岔：同一件装备在两个职业的语境下是"不同的装备"——CC套在剑魂语境中是"高收益装备"（+41.1%），在狂战士语境中是"低收益装备"（+9.3%）。这种二元结构迫使装备评测必须标注职业语境，任何脱离职业类型的"泛化装备排名"在70版本都是无意义的。

**论据：**
- CC套收益率对照（已验证稳定的v97最终值）：
  | 职业 | 输出类型 | CC套综合收益 | 有效加成属性数 | 无效属性 |
  |------|----------|-------------|--------------|---------|
  | 剑魂 | 百分比 | **+41.1%** | 全部3个 | 无 |
  | 狂战士 | 固伤 | **+9.3%** | 2个 | 力量+物攻 |
  | 差异比 | — | **4.4倍** | — | — |
- 验证交叉验算：剑魂综合 = 1.3647 × 1.0216 × 1.012 - 1 = 41.09% ≈ 41.1%。狂战士综合 = 1.08 × 1.012 - 1 = 9.30% ≈ 9.3%。62项交叉验证+19项Python精确验算100%通过。
- 武器强化系统进一步放大：百分比职业+12→+36%、+13→+42%物攻收益，固伤职业零受益。
- E2/E3最优搭配均为E2 6件+CC套2件，但力学相同、原因不同——百分比为技能等级倍率提升，固伤为技能等级基数提升。

## 八、综合理论模型："双通道过滤漏斗"

综合七维度分析，可将70版本装备加成系统抽象为"双通道过滤漏斗"模型：

```
                ┌──────────────────────┐
                │  装备属性输入层        │
                │  力量+310 / 物攻+110  │
                │  独立+120 / 暴击+3%   │
                └───────┬──────────────┘
                        │
              ┌─────────┴─────────┐
              ▼                   ▼
    ┌────────────────┐  ┌────────────────┐
    │ 百分比通道过滤   │  │  固伤通道过滤   │
    │ 力量→有效        │  │ 力量→无效      │
    │ 物攻→有效        │  │ 物攻→无效      │
    │ 独立→无效        │  │ 独立→有效      │
    │ 暴击→有效        │  │ 暴击→有效      │
    │ 有效通道: 3/4    │  │ 有效通道: 2/4  │
    └────────┬───────┘  └────────┬───────┘
             ▼                   ▼
    ┌────────────────┐  ┌────────────────┐
    │ 输出:+41.1%    │  │ 输出:+9.3%    │
    │ ①力量×1.365   │  │ ①独立×1.080  │
    │ ②物攻×1.022   │  │ ②暴击×1.012 │
    │ ③暴击×1.012   │  │               │
    └────────────────┘  └────────────────┘
```

**模型预测能力：**
1. 任何提供力量/物攻为主的套装 → 百分比职业收益将系统性地是固伤职业的**3~5倍**
2. 任何提供独立/异常为主的套装 → 固伤职业收益将反超（如屠戮之刃：固伤+23%~25%，百分比仅+5%）
3. E2 6件+CC套2件是两职业共性最优解，但推理路径相反

**模型局限：** 冷却压缩和属性强化未纳入该模型量化，因为数据源中冷却不是装备词条，属强计算缺乏70版本具体怪物属抗数据。
