## 一、核心命题：乘区空间 = 经济资源空间

### 1.1 定义：伤害公式中的"经济体"

将DNF 70版本伤害公式视为一个经济体，其中每个乘区具有三大经济学属性：

| 属性 | 符号 | 含义 | 伤害公式中的对应 |
|------|------|------|-----------------|
| **产能** | P | 该乘区封顶时能提供的最大伤害倍数 | 如力量翻倍因子(1+S/250)、独立翻倍因子(1+I/250) |
| **供给弹性** | ε | 装备对该乘区进行扩展的难易程度 | 稀缺=低弹性，充裕=高弹性 |
| **穿透性** | τ | 该乘区收益不受其他乘区影响的程度 | 独立乘区（技能攻击力）> 加算乘区（所有攻击力） |

**核心命题**：70版本策划通过控制每个乘区空间的三维向量⟨P, ε, τ⟩，精确调控了不同职业/流派的成长曲线形状。

### 1.2 各乘区的经济学画像

```
乘区名称       |   产能 P    | 供给弹性 ε  | 穿透性 τ
────────────────┼─────────────┼─────────────┼────────────
力量/智力乘区   |  高 (7.0×)  |  高 (丰富)   |  中 (受武器耦合)
物理攻击乘区   |  高 (7.0×)  |  高 (丰富)   |  低 (受技能系数影响)
独立攻击乘区   |  中 (4.0×)  |  低 (稀缺)   |  中 (独立绑定固伤)
暴击乘区       |  低 (1.5×)  |  中 (可控)   |  高 (线性乘区)
防御穿透       |  中 (~2×)   |  极低        |  极高 (无视防御)
属性强化       |  中 (~2×)   |  中          |  中 (受怪物抗性)
冷却缩减       |  无系统性   |  极低/不存在  |  — (70版本无此市场)
```

**关键发现**：乘区的P值与ε值呈**反相关**——产能越高的乘区供给弹性越高，产能越低的乘区供给越稀缺。这构成了70版本装备经济的**基本约束定律**：

> **乘区反相关律**：一个乘区的理论最大增长空间越大，策划对其供给弹性控制越紧；反之，一个乘区的理论增长空间有限，策划越倾向于提供丰富的供给源。

实证：
- 力量乘区P=7.0×（最大），ε=高——来源丰富（CC套、暴走、宝石、宠物、称号、E2套装）
- 独立攻击乘区P=4.0×（中等），ε=低——来源极少（仅武器锻造+CC套+2件SS武器）
- 暴击乘区P=1.5×（最小），ε=中——来源适中（技能被动+装备+宝石+CC套）
- 冷却缩减在70版本中P值被设为0（等同于「该乘区市场被关闭」）

## 二、阈值机制的经济学解释：市场准入壁垒

### 2.1 六件阈值作为准入壁垒

CC套的「6件阈值」机制在经济学中对应**市场准入壁垒**——只有凑齐6件的玩家才能获得套装收益。这一设计创造了：

1. **信息壁垒**：新玩家需要消耗时间/精力去了解「为什么要凑齐6件」
2. **时间壁垒**：凑齐6件需要时间（单件获取慢），将玩家按游戏时长分层
3. **决策压力**：凑齐6件意味着「放弃其他套装的6件位」，是一种**沉没成本承诺**

### 2.2 阈值触发与消费者剩余转化

装备CC套6件前的玩家处于「消费者剩余为正」状态——单件属性（力量~260、物理攻击~90）已经提供了可用加成。6件阈值逼迫玩家在「保持散搭灵活性」和「付出解锁成本」之间做出选择，实际上是将部分散搭期的轻松感**转化为**6件套的集中收益。

**阈值效应量化**：
```
CC套单件6件总和 = 力量260 + 物理攻击92 + 独立攻击100 + 暴击2.5%
CC套6件套装总和 = 力量310 + 物理攻击110 + 独立攻击120 + 暴击3.0%

阈值激活增量   = 力量+50(16.1%) + 物理攻击+18(16.4%) + 独立攻击+20(16.7%) + 暴击+0.5%(16.7%)
```

所有属性的阈值激活增量在16-17%之间，高度一致——这并非巧合，而是策划有意设计的**统一阈值折扣率**：集齐6件比散搭多获得约16.5%的额外属性，作为一种「套装忠诚度奖励」。

### 2.3 不同阈值的对比

| 系统 | 阈值类型 | 准入条件 | 收益特征 |
|------|---------|---------|---------|
| CC套 | 单品6件全激活 | 投入6个部位 | 统一增量~16.5% |
| E2套 | 存在3/6件两档阈值 | 投入3或6件 | 核心技能加成（等级+1/+2） |
| E3套 | 存在3/6件两档阈值 | 投入3或6件 | 伤害加成（非技能等级） |

E2/E3的双阈值设计（3件与6件）比CC套的单阈值设计更灵活，给玩家「渐进投入」的选择。而CC套没有3件/6件两档，**要么全有要么全无**——因此CC套更适合毕业切换而非过渡使用。

## 三、稀缺控制的经济学分析：独立攻击乘区的供给约束

### 3.1 独立攻击的供需模型

70版本独立攻击乘区呈现**极低供给弹性**（ε接近于0）：

```
独立攻击供给源（70版本）：
  1. 武器锻造：+独立攻击固定值（70粉武器锻造+约300）
  2. CC套6件：+独立攻击120（仅对固伤有效）
  3. SS武器%加成：屠戮之刃+5%、魂·巨剑+7%（仅2件）
  4. E2套装腿：独立攻击+3%（杀之守护）
```

独立攻击的「弹性」低的根本原因：它不是独立供给品，而是附着在其他装备上的**副产品属性**。没有一件装备的**核心设计目标**是「提供独立攻击力」（除了SS武器上的%加成），独立攻击始终以附加属性形式出现。

### 3.2 供需失衡的设计意图

策划通过控制独立攻击的供给，精确地设定了固伤职业的「封顶收益」：

```
固伤封顶模型：
  面板独立上限 ≈ 1250（基础） + 300（锻造） + 120（CC套） = 1670
  对应的固伤翻倍因子 = (1 + 1670/250) = 7.68×
  
  对比百分比翻倍因子：
  物理攻击上限 ≈ 2000（基础）+ 200（CC/附魔）= 2200
  力量翻倍因子 = (1 + 910/250) = 4.64×
  物攻翻倍因子 = (1 + 2200/2500) = 1.88×
  百分比综合翻倍 = 4.64 × 1.88 × 暴走×破极 ≈ 13.6×
```

固伤职业的伤害上限被压缩在约7.68×，而百分比职业可达13.6×——差距约1.8倍。这一差距并非BUG而是**有意设计的职业天花板差异**：固伤职业的入门门槛低（不需要高强武器），但上限也低；百分比职业相反。

### 3.3 屠戮之刃的「突破」角色

屠戮之刃作为55级SS太刀，提供「独立攻击+5%」+「出血伤害+20%」，在经济系统中扮演**稀缺资源黑市突破**的角色——它打破了独立攻击供给弹性的极限，在供给曲线上创造了一个小的向上偏移。但这件武器的存在并不改变整体供给曲线形状，因为它本身也极难获取（SS掉落率极低）。

## 四、固伤vs百分比二元论：两种经济增长模式的对比

### 4.1 增长函数差异

将固伤与百分比视为两种**不同的经济增长模式**：

**固伤增长函数**（收敛型）：
```
f_fixed(x) = 技能基数 × (1 + 独立/250) × 技能等级乘数 × 暴击期望
           = 技能基数 × [常数 × (1 + x/250)] × x² × x
增长率特性：随着独立攻击投入增加，边际收益递减 → 收敛型增长
```

**百分比增长函数**（发散型）：
```
f_percent(x, y, z) = 物理攻击 × 技能倍率 × (1 + 力量/250)
                    = x × y × (1 + z/250)
增长率特性：三个变量各自独立增长 → 发散型增长（多变量叠加）
```

### 4.2 增长曲线形式

```
伤害值
│
│                     ┌── 百分比（发散型）
│                   ╱
│                 ╱
│               ╱
│             ╱
│           ╱
│         ╱  ─────────── 固伤（收敛型，渐近极限）
│       ╱   ╱
│     ╱   ╱
│   ╱   ╱
│ ╱   ╱
└───────────────────── 装备投入 →
```

固伤曲线在低投入阶段高于百分比（因为固伤不受武器品质影响），但很快被百分比曲线超越。两条曲线的**相交点**就是策划设计的「职业性价比转折点」——在此之前固伤占优，在此之后百分比领先。

### 4.3 相交点的决定因素

相交点由以下参数决定：
1. **武器攻击力**：越高则百分比曲线越陡，相交点左移
2. **力量**：越高则百分比曲线越陡，相交点左移
3. **独立攻击**：越高则固伤曲线整体上移，相交点右移
4. **技能等级**：E2/E3加成使固伤曲线上移，但E2/E3本身也影响百分比（通过辅助技能等级）

70版本的相交点位置：
- 在粉装+10/+11级别武器和异界E2非完整配置下≈55%/45%
- 在+12粉武和E2 6件下≈40%/60%（百分比开始超出）
- 在+13粉武和E2+CC混搭下≈35%/65%（百分比明显占优）

## 五、暴走+40%力量的经济效应：杠杆放大原理

### 5.1 杠杆原理

暴走提供+40%力量（Lv10基础+35% + E2血之狂暴+5%），在经济学中属于**财务杠杆**——投入基础力量后，暴走再将其放大1.4倍。但这一杠杆仅适用于百分比乘区（固伤公式中力量不参与计算），因此：

> 暴走是**百分比流专用的杠杆工具**，对固伤流是"死杠杆"——即使力量再高，也不在固伤公式中产生收益。

### 5.2 杠杆倍率的固定乘数结构

暴走的力量加成与装备力量加成是**加算后**再整体放大，而非各自作为独立乘区：

```
面板力量 = (基础力量 + 装备力量) × (1 + 暴走比例)
         = (600 + 310) × 1.40
         = 910 × 1.40
         = 1,274
力量倍率 = (1 + 1,274/250) = 6.096
原始倍率 = (1 + 600/250)   = 3.40
净收益   = 6.096 / 3.40 - 1 = 79.3%
```

其中CC套贡献：`(1 + 910/250) / (1 + 600/250) - 1 = 36.5%`
暴走的杠杆效应就是将CC套的36.5%收益从「基础力量提升」转化为「暴走状态下的面板力量提升」，而暴走本身是30秒持续/45秒冷却的**时间窗口资源**——这意味着该收益仅在66.7%的战斗时间内生效。

### 5.3 时间资源定价

将暴走的冷却时间视为「时间资源成本」，则CC套的36.5%力量收益仅在暴走开启的30秒内被杠杆放大，在其他15秒内仅以基础力量形式存在：

```
加权收益 = 36.5% × (30/45) + 31.7% × (15/45)
         = 36.5% × 0.667 + 31.7% × 0.333
         = 24.3% + 10.6%
         = 34.9%
```

相比理论最高36.5%的标签数值，实际加权收益34.9%低于表面值约4.4%。但相比无暴走下的CC套收益31.7%，仍高出10.1%（34.9%/31.7%-1）——暴走杠杆的效率增益约为10%。

## 六、五律的经济学重新表述

将五律体系转换为经济学语言，得到「乘区空间经济学」的统一表述：

| 原五律 | 经济学表述 | 在装备系统中的含义 |
|--------|-----------|-------------------|
| 阈值隔离律 | 准入壁垒效应 | 套装阈值创造市场分层，玩家按投入程度获得不同收益率 |
| 乘区分界律 | 市场分割效应 | 各乘区形成独立子市场，力量市场与独立市场互不渗透 |
| 稀缺控制律 | 供给约束效应 | 独立攻击、冷却缩减的供给被刻意压低以控制固伤上限 |
| 职业分流律 | 比较优势效应 | 固伤与百分比各具比较优势，形成职业分工的均衡 |
| 补偿通道律 | 正外部性补贴 | 出血穿透防御是固伤职业的"外部补贴"，纠正市场失衡 |

**统一表述**：

> DNF 70版本装备系统的底层逻辑可以完全用经济学语言描述——它是一个由**多乘区市场**组成的**受限增长系统**。每个乘区是一个子市场，拥有独立的供给弹性、产能上限和穿透性参数。策划通过调节这些参数，使固伤与百分比两种"经济模式"各自在比较优势区间内发展，并通过补偿通道纠正极端失衡。五律是这一经济系统的**市场规则总和**。

## 七、结论：70版本装备系统的经济学本质

本轮回溯将20轮理论构建的五律体系和形式化约束模型置于**经济学框架**下重新审视，得出以下本质理解：

1. **乘区即市场**：每个乘法因子构成一个独立子市场，各市场有不同的市场结构（垄断、寡头、完全竞争）
2. **阈值即壁垒**：套装阈值构成市场准入壁垒，将玩家分层并按层分配收益率
3. **稀缺即调节**：独立攻击的稀缺性是策划最精细的调控杠杆
4. **杠杆即夹层**：暴走、破极兵刃等BUFF技能作为财务杠杆，放大基础投入的产出
5. **补偿即福利**：出血穿透防御是社会福利政策，为固伤职业提供市场失灵时的托底

**五律体系在经济学框架下仍然完备**——没有任何一条定律被否定或替代。经济学表述只是提供了更深层的、统一的原因解释：这些设计规律之所以存在，是因为装备系统本质上是一个经济系统，玩家行为是经济决策，策划设计是经济调控。
