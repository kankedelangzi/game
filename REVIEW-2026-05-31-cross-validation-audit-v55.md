```python
ind_before, ind_after = 1250, 1370
crit_before, crit_after = 0.55, 0.58

ind_bonus = (1 + ind_after/250) / (1 + ind_before/250) - 1  # 8.00%
crit_coeff_before = (1 - crit_before) + crit_before * 1.5   # 1.275
crit_coeff_after = (1 - crit_after) + crit_after * 1.5      # 1.290
crit_bonus = crit_coeff_after / crit_coeff_before - 1       # 1.18%
fixed_total = (1 + ind_bonus) * (1 + crit_bonus) - 1        # 9.27%
```

- 独立攻击收益：**8.00%** ✅ 匹配

## 三、问题项分析

### ❌ 问题1：狂战士百分比综合收益计算不一致（🔴 严重）

| 项目 | HTML报告 | 精确验算 | 差异 |
|------|----------|----------|------|
| 力量收益 | +31.7% | +31.70% | ✅ 一致 |
| 物理攻击收益 | +2.44% | +2.44% | ✅ 一致 |
| 暴击收益 | +1.2% | +1.18% | 🟡 四舍五入 |
| **百分比综合收益** | **+36.67%** | **+36.50%** | **0.17pp** |

**根本原因分析**：
HTML报告声称的百分比综合收益公式为：
```
1.317 × 1.0244 × 1.012 - 1 = 36.67%
```

但实际计算：
```
1.317 × 1.0244 × 1.012 - 1 = 1.3653 - 1 = 36.53%
```

即使使用HTML报告的中间值，综合收益也应该是**36.53%**，而非声称的**36.67%**。

**精确验算**（使用未四舍五入的中间值）：
```
31.70% × 2.44% × 1.18% → (1.3170 × 1.0244 × 1.0118) - 1 = 36.50%
```

**结论**：HTML报告的百分比综合收益存在**计算错误**，声称值36.67%与实际计算值36.50%存在0.17pp差异。

**影响评估**：
- 差异0.17pp属于**中等精度问题**
- 对实际配装决策影响较小（36.5% vs 36.7%）
- 但作为权威数据报告，公式与结果必须一致

**严重程度**: ❌ **中等** — 公式与结果不一致
