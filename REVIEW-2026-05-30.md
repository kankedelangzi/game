## 🔧 修正建议

### 1. E2套装部位修正（🟡中等）
**问题**: BERSERKER.md和berserker-e2-bonus.html中E2套装部位标注为"左槽"，实际70版本E2血之狂暴套装第6件为**右槽**（辅助装备）。

**修正**:
```
原：上衣、下装、头肩、腰带、鞋、左槽
正：上衣、下装、头肩、腰带、鞋、右槽
```

### 2. 出血等级表述优化（🟡中等）
**问题**: "出血等级 = 十字斩Lv18 + 出血精通(+2) + 血气爆发(+1) = Lv21" 表述有歧义——出血精通和血气爆发是**加成**而非等级来源。

**修正**:
```
出血等级 = 十字斩Lv18 + 出血精通(+2) + 血气爆发(+1) = Lv21
出血伤害加成 = 出血精通(+10%) + 血气爆发(+20%) = +30%

说明：出血等级由技能等级决定，出血精通和血气爆发提供的是伤害加成而非等级提升。
```

### 3. 力量计算表述优化（🟡中等）
**问题**: "暴走后982.8 → E2后1032，额外+49.2点" 应明确说明这是乘法叠加结果。

**修正**:
```
暴走加成：728 × 1.35 = 982.8
E2加成：982.8 × 1.05 = 1032
E2额外收益：1032 - 982.8 = +49.2点（在暴走基础上的乘法增量）
```

### 4. 数据来源标注补充（🟡中等）
**问题**: 部分数值（如武器属性）仅标注"DNF Wiki"，建议补充具体URL或NGA帖子标题。

**修正**: 在HTML文件中，将来源标注从通用描述改为具体引用：
```
来源：DNF Wiki 70版本粉装数据库（https://dnf.wiki/...）
来源：NGA DNF专区「70版本异界套装数据汇总」（精品帖，2013-12-15）
```
