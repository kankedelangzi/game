### 验算4：穿透防御表格

```python
def physical_damage(base, defense):
    return base * 300 / (300 + defense)

targets = [
    ("普通小怪", 500),
    ("精英怪", 2000),
    ("普通BOSS", 5000),
    ("高防BOSS", 10000),
]

for name, def_val in targets:
    dmg = physical_damage(10000, def_val)
    print(f"{name}(防御{def_val}): {dmg:.0f}")
```

**验算结果**:
- 普通小怪(500): 3,750 ✅
- 精英怪(2000): 1,304 ✅
- 普通BOSS(5000): 566 ✅
- 高防BOSS(10000): 291 ✅

bleed_total = 24700
