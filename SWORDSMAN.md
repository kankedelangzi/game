## 六、伤害计算公式

```
百分比最终伤害 = 物理攻击 × 技能倍率 × (1 + 力量/250) × 防御减伤 × 暴击期望

破极兵刃加成 = 基础物理攻击 × (1 + 30%)

防御减伤 = 300 / (300 + 目标防御)
暴击期望系数 = (1 - 暴击率) + 暴击率 × 1.5
```

## 九、伤害特点总结

1. **百分比技能为主**：裂波斩、拔刀斩、上挑为核心，依赖物理攻击力
2. **破极兵刃核心BUFF**：破极兵刃+30%物理攻击，是剑魂伤害计算的核心乘区
3. **武器强化收益极高**：+13提升42%物理攻击，是剑魂最重要的投资
4. **巨剑精通主流**：物理攻击力最高，伤害最高
5. **异界E2毕业装**：「剑魂」6件是剑魂核心毕业装（裂波斩+2，上挑+2，拔刀斩+2）
