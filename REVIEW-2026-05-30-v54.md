### 建议3：修正力量宝石收益计算（🟡中等，v54新增发现）

**当前问题**：
```
四级力量宝石 ×4 | +128 力量 | — | — | 约 +12.8%（728力量面板）
```

**独立验算**：
```
基准面板：728力量
新增力量：+128 → 856力量
收益 = (1 + 856/250) / (1 + 728/250) - 1
     = (1 + 3.424) / (1 + 2.912) - 1
     = 4.424 / 3.912 - 1
     = 1.131 - 1
     = +13.1%
```

**注意**：主Agent使用的是"面板力量728→暴走后1019"的暴走中状态，但宝石收益计算应在暴走前基准计算。若按暴走前728计算：
- 暴走前：728 → 856，收益 = (1+856/250)/(1+728/250)-1 = 4.424/3.912-1 = +13.1%
- 若主Agent使用不同基准（如基础力量350+装备378=728），则需明确标注

**建议修正**：明确标注计算基准（暴走前/暴走后），修正数值为"约+13.1%"（暴走前基准）或重新计算。

### 建议4：修正混合配置收益计算（🟡中等，v54新增发现）

**当前问题**：
```
混合配置（2力+2属强）| +64 力量 | +80 属强 | — | 约 +22.9%（728力量/原属强40）
```

**独立验算**：
```
力量收益：(1 + 792/250) / (1 + 728/250) - 1 = 4.168/3.912 - 1 = +6.54%
属强收益：(220 + 300) / (220 + 40) - 1 = 520/260 - 1 = +100%

综合收益（乘法叠加）：
(1 + 6.54%) × (1 + 100%) - 1 = 1.0654 × 2.0 - 1 = +133.1%
```

**⚠️ 严重偏差**：主Agent标注+22.9%，但独立验算结果为+133.1%。差异巨大，需重新核实。

**可能原因**：
1. 主Agent可能使用了不同的属强基准（如原属强220而非40）
2. 公式理解有误：属强收益公式应为(220+新增属强)/(220+原属强)，而非(220+原属强+新增)/(220+原属强)

**建议修正**：
- 明确标注原属强基准值
- 重新计算并修正数值
- 如原属强为40，新增属强80，则：(220+120)/(220+40)-1 = 340/260-1 = +30.8%
- 综合收益：(1+6.54%)×(1+30.8%)-1 = 1.0654×1.308-1 = +39.4%
