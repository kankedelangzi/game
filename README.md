# 游戏伤害原理研究 🎮

长线研究项目，系统分析各类2D游戏的底层伤害计算机制。

## 研究范围

### 🎯 重点：DNF（地下城与勇士）60-85版本
- 伤害计算公式（物理/魔法攻击力 → 实际伤害）
- 属性强化 / 属性抗性机制
- 暴击伤害计算
- 技能百分比 vs 固伤机制
- 防御力减伤公式
- 异常状态伤害（感电、出血等）
- 队友伤害加成（协同攻击）
- 装备属性叠加规则

### 👊 拳皇（KOF）系列
- 连招伤害衰减机制
- 破防值计算
- 能量槽（Power Gauge）对伤害的影响
- 帧数数据与伤害的关系

### 🧟 向僵尸开炮（小游戏）
- 塔防伤害计算
- 暴击 / 穿透机制
- 怪物护甲与减伤
