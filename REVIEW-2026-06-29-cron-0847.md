### 问题详细说明

**问题1 & 2：剑魂物理攻击收益计算基准不一致**

HTML文件使用**破极兵刃后**物理攻击2600作为计算基准：
```
物理攻击收益 = 2710/2600 - 1 = 4.23%
综合收益 = 1.3647 × 1.0423 × 1.012 - 1 = 43.95%
```

SWORDSMAN.md（v793修正，2026-06-28 19:19）使用**破极兵刃前**基础物理攻击2000作为计算基准：
```
物理攻击收益 = 2110/2000 - 1 = 5.50%
综合收益 = 1.3647 × 1.055 × 1.012 - 1 = 45.70%
```

**v793修正逻辑**：CC套的+110物理攻击是固定数值加成，应基于角色基础面板（破极兵刃前）计算收益，而非基于临时Buff后的面板。破极兵刃是临时Buff（30秒），CC套是常驻装备，两者不应混淆计算基准。

**问题3：HTML内部边际对偶不一致**

HTML中边际对偶显示为4.930020，但根据HTML中展示的剑魂综合收益43.95%和狂战士固伤收益9.27%：
```
43.95 / 9.27 = 4.740937 ≠ 4.930020
```

而4.930020对应的是v793修正后的值：
```
45.70 / 9.27 = 4.930020 ✅
```

这说明HTML的边际对偶值已更新到v793版本，但剑魂的详细数值仍停留在v868版本（43.95%），存在内部不一致。
