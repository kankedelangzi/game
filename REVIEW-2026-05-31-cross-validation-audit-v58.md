### 1.2 🔴 严重错误确认：物理伤害计算公式错误（持续未修正）

**HTML报告错误公式**（第4节"穿透防御验证"表格 + 第6.1节"高防BOSS战"）：
```
物理伤害 = 基础伤害 × 减伤比例     ← ❌ 错误
```

**正确公式**（DNF 70版本标准）：
```
减伤比例 = 300 / (300 + defense)
物理伤害 = 基础伤害 × (1 - 减伤比例)   ← ✅ 正确
```
