### 验算2：武器对比HTML钝器 vs 巨剑差异

| 技能 | 钝器独立 | 巨剑独立 | 钝器系数 | 巨剑系数 | 差异 | 表格差异 |
|------|----------|----------|----------|----------|------|----------|
| 十字斩 | 1200 | 1180 | 1+1200/250=5.8 | 1+1180/250=5.72 | 5.8/5.72≈1.014 | +11.1% ❌ |

**⚠️ 发现问题**：表格标注钝器 vs 巨剑差异为+11.1%，但根据武器独立攻击差异（1200 vs 1180），实际差异应为：
- 钝器系数 = 1 + 1200/250 = 5.8
- 巨剑系数 = 1 + 1180/250 = 5.72
- 差异 = 5.8 / 5.72 ≈ 1.014 = **+1.4%**

但表格显示的是+11.1%，这意味着表格值可能使用了不同的独立攻击基准（如1250 vs 1180）：
- 1250/250 = 5.0 → 系数 = 6.0
- 1180/250 = 4.72 → 系数 = 5.72
- 差异 = 6.0 / 5.72 ≈ 1.049 = **+4.9%**

**仍然不匹配+11.1%**。需要进一步核查。

**重新验算**：如果钝器独立=1250，巨剑独立=1180：
- 钝器系数 = 1 + 1250/250 = 6.0
- 巨剑系数 = 1 + 1180/250 = 5.72
- 差异 = 6.0 / 5.72 - 1 = +4.9%

如果钝器独立=1200，巨剑独立=1180：
- 钝器系数 = 1 + 1200/250 = 5.8
- 巨剑系数 = 1 + 1180/250 = 5.72
- 差异 = 5.8 / 5.72 - 1 = +1.4%

**+11.1%的来源**：如果钝器独立=1250，巨剑独立=1125（而非1180）：
- 1250/250 = 5.0 → 系数 = 6.0
- 1125/250 = 4.5 → 系数 = 5.5
- 差异 = 6.0 / 5.5 - 1 = +9.1%

**仍然不匹配**。+11.1%可能是从其他基准推导的，需要标注来源或修正。
