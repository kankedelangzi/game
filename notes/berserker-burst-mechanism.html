2.2 E2套装阈值机制

E2血之狂暴件数力量加成触发条件
1~5件+0%❌ 不生效
6件+5%✅ 激活

🔑 E2套装的+5%力量加成是阈值型——只有集齐6件才激活，1~5件完全不生效。这是典型的"凑件"设计，迫使玩家追求完整套装。

2.3 总加成公式

暴走总加成 = 技能等级加成 + E2套装加成
= 35%（Lv10） + 5%（E2 6件）
= 40%

4️⃣ 暴走对不同伤害类型的影响

4.1 百分比技能（巨剑流核心）

百分比技能的伤害公式中包含物理攻击力，而物理攻击力与力量成正比：

物理攻击 = 武器攻击 + (力量 × 武器攻击系数 / 250)

暴走前：728 力量 → 物理攻击 ≈ 2000
暴走后：1019 力量 → 物理攻击 ≈ 2342

物理攻击增幅 = (2342 - 2000) / 2000 ≈ +17.1%

但考虑到武器基础攻击部分的稀释效应，实际百分比技能收益约为 +29.7%（精确计算见下方）：

百分比技能收益 = (1 + 暴走后力量/250) / (1 + 暴走前力量/250) - 1
= (1 + 1019/250) / (1 + 728/250) - 1
= 5.076 / 3.912 - 1
= +29.7%

📌 这是力量面板的边际收益递减效应——力量越高，每点力量带来的百分比技能增幅越小。
