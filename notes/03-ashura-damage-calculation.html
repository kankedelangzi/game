四、伤害计算实例

实例1：邪光斩（4段固伤）- 光强流暴击期望

【技能参数（70版本，E2+3后Lv26）】
- 技能固伤基数：5200 × 1.18 = 6136（4段，每段1534）
- 每段独立伤害：1534
- 技能范围：超大（全图横向）
- 段数：4段
- 属性：光属性（受光强加成）

【角色面板】
- 智力：1050
- 独立攻击力：1280
- 光属性强化：72
- 魔法暴击率：52%
- 防御减伤：25%（假设目标为普通怪，防御约1000）

【逐步计算（单段）】
1. 基础固伤 = 1534

2. 独立攻击力加成：
   独立系数 = 1 + 独立/1000 = 1 + 1280/1000 = 2.28
   独立后伤害 = 1534 × 2.28 = 3,498

3. 智力加成：
   智力系数 = 1 + 智力/250 = 1 + 1050/250 = 5.2
   智力后伤害 = 3,498 × 5.2 = 18,190

4. 光属性强化加成：
   属强系数 = 1 + 光强/220 = 1 + 72/220 = 1.327
   属强后伤害 = 18,190 × 1.327 = 24,138

5. 防御减伤：
   减伤系数 = 300 / (300 + 防御) = 300 / (300 + 1000) = 0.231
   减伤后伤害 = 24,138 × 0.231 = 5,576

   ⚠️ 注：70版本怪物防御普遍较高，BOSS防御可达3000+
   对BOSS（防御3000）：减伤系数 = 300/3300 = 0.091
   BOSS伤害 = 24,138 × 0.091 = 2,196

6. 暴击期望：
   暴击倍率 = 150%（70版本无额外暴伤装备）
   暴击期望系数 = (1 - 暴击率) × 1.0 + 暴击率 × 1.5
               = (1 - 0.52) × 1.0 + 0.52 × 1.5
               = 0.48 + 0.78 = 1.26
   暴击期望伤害 = 5,576 × 1.26 = 7,026

💡 邪光斩单段暴击期望伤害：7,026
💡 邪光斩4段总伤害（暴击期望）：7,026 × 4 = 28,104
💡 邪光斩4段总伤害（对BOSS暴击期望）：2,196 × 4 × 1.26 = 11,063

【结论】
邪光斩对普通怪单发约2.8万，对高防BOSS约1.1万。
4段击飞效果+超大范围，是70版本阿修罗清图核心技能。

实例2：杀意波动（主动）- 短CD固伤爆发

【技能参数（70版本，E2+2后Lv20）】
- 技能固伤基数：1200 × 1.18 = 1416
- CD：约8秒（短CD高频技能）
- 范围：中等（身前扇形）
- 属性：光属性

【逐步计算】
1. 基础固伤 = 1416

2. 独立加成 = 1416 × (1 + 1280/1000) = 1416 × 2.28 = 3,228

3. 智力加成 = 3,228 × (1 + 1050/250) = 3,228 × 5.2 = 16,786

4. 光强加成 = 16,786 × (1 + 72/220) = 16,786 × 1.327 = 22,275

5. 防御减伤（普通怪）= 22,275 × 0.231 = 5,146

6. 暴击期望 = 5,146 × 1.26 = 6,484

💡 杀意波动单发暴击期望：6,484
💡 CD仅8秒，可频繁使用补充伤害

【杀意波动被动收益】
- 被动每级+2%魔法攻击力，满级Lv18 = +36%魔法攻击
- 对百分比技能（地裂、冰刃、爆炎）有显著加成
- 被动收益：2800 × 36% = +1008魔法攻击力

实例3：不动明王阵（持续固伤）- E3套装核心

【技能参数（70版本，E3+1后Lv19）】
- 技能固伤基数：7200 × 1.18 = 8496
- 持续时间：约10秒
- 每秒伤害：8496 / 10 = 850/秒
- 范围：中等（阵内持续伤害）
- E3套装加成：伤害+20%

【逐步计算（含E3加成）】
1. 基础固伤（含E3）= 8496 × 1.20 = 10,195

2. 独立加成 = 10,195 × 2.28 = 23,245

3. 智力加成 = 23,245 × 5.2 = 120,874

4. 光强加成 = 120,874 × 1.327 = 160,400

5. 防御减伤（普通怪）= 160,400 × 0.231 = 37,052

6. 暴击期望 = 37,052 × 1.26 = 46,686

💡 不动明王阵10秒总伤害（暴击期望）：46,686
💡 每秒约4,669，持续输出稳定

【战术价值】
- 不动明王阵是70版本阿修罗控场+输出核心
- 阵内敌人无法移动，配合邪光斩可实现"定身+秒杀"
- E3套装是阿修罗后期毕业装之一

实例4：觉醒技能「杀意·波动剑」- 单体爆发

【技能参数（70版本，Lv1）】
- 技能固伤基数：18000 × 1.18 = 21,240
- CD：约60秒（觉醒技能CD长）
- 范围：中等（单体/小范围）
- 无双波被动：固伤技能伤害+1%×觉醒等级

【逐步计算】
1. 基础固伤 = 21,240

2. 独立加成 = 21,240 × 2.28 = 48,427

3. 智力加成 = 48,427 × 5.2 = 251,820

4. 光强加成 = 251,820 × 1.327 = 334,165

5. 防御减伤（普通怪）= 334,165 × 0.231 = 77,192

6. 暴击期望 = 77,192 × 1.26 = 97,262

💡 杀意·波动剑暴击期望伤害：97,262
💡 觉醒技能是阿修罗单体爆发天花板

【觉醒被动「无双波」】
- Lv1：固伤技能伤害+1%
- 满级Lv10：固伤技能伤害+10%
- 对邪光斩、杀意波动、不动明王阵均有加成
- 实际收益：97,262 × 1.10 = 106,988（满级觉醒被动）

实例5：感电机制计算（波动眼+冰刃·波动剑）

【感电机制（70版本）】
- 波动眼被动：感电触发率+15%（Lv10=100%触发率）
- 感电状态：敌人受到物理/魔法伤害时，追加感电伤害
- 感电伤害：基础感电值 × (1 + 雷神之息等级加成)
- 70版本基础感电值：约500-800/段

【冰刃·波动剑感电触发】
- 冰刃·波动剑：魔法百分比技能，倍率280%
- 攻击时触发感电，感电持续3秒，每0.5秒触发一次

【感电伤害计算】
1. 冰刃·波动剑本身伤害（百分比）：
   基础 = 2800 × 280% = 7,840
   智力加成 = 7,840 × 5.2 = 40,768
   冰强加成 = 40,768 × (1 + 68/220) = 40,768 × 1.309 = 53,365
   防御减伤 = 53,365 × 0.231 = 12,327
   暴击期望 = 12,327 × 1.26 = 15,532

2. 感电追加伤害（3秒内，每0.5秒一次，共6次）：
   每次感电 = 600（基础值）× (1 + 雷神之息Lv5=50%) = 900
   每次感电独立计算乘区：
   - 智力加成 = 900 × 5.2 = 4,680
   - 冰强加成 = 4,680 × 1.309 = 6,126
   - 防御减伤 = 6,126 × 0.231 = 1,415
   - 暴击期望 = 1,415 × 1.26 = 1,783
   
   6次感电总伤 = 1,783 × 6 = 10,698

💡 冰刃·波动剑 + 感电总伤害（暴击期望）：15,532 + 10,698 = 26,230
💡 感电贡献约40%总伤害，是阿修罗重要伤害来源

六、阿修罗70版本伤害特点总结

  1. 固伤为主，百分比为辅：邪光斩、杀意波动、不动明王阵为核心输出，地裂/冰刃/爆炎为补刀技能
  2. 属强收益极高：每22点属强=+10%伤害，光强/冰强是阿修罗两大流派
  3. 感电机制独特：波动眼触发感电，追加持续伤害，是阿修罗区别于其他固伤职业的特色
  4. 控制能力强：不动明王阵定身+邪光斩击飞，70版本副本控制需求高
  5. 异界套装依赖：E2「波动之眼」是邪光斩流核心，E3「不动明王」是阵鬼流核心
  6. 爆发+持续双修：觉醒技能爆发高，不动明王阵持续输出稳定

💡 伤害计算公式速查

固伤最终伤害 = 基础固伤 × (1 + 独立/1000) × (1 + 智力/250) × (1 + 属强/220) × 防御减伤 × 暴击期望

百分比最终伤害 = 基础倍率 × 魔法攻击 × (1 + 智力/250) × (1 + 属强/220) × 防御减伤 × 暴击期望

防御减伤 = 300 / (300 + 目标防御)
暴击期望系数 = (1 - 暴击率) + 暴击率 × 1.5
