🗡️ RPG / 战斗类（最接近 DNF 研究）

⭐ DevilutionX ★★★★★ 推荐指数

项目：github.com/diasurgical/DevilutionX

语言： C++ ｜ ★ Stars： 9.5k

原作：暗黑破坏神1 开源重制版

可研究机制：

武器伤害公式护甲减伤魔法伤害 抗性系统掉落系统属性加成

伤害公式（已知）：

// 暗黑1 武器伤害计算（简化）
finalDamage = baseMin + random(baseMax - baseMin) - armor;
if (finalDamage < 1) finalDamage = 1;

// 魔法伤害
magicDamage = spellBase + (spellLevel * spellPerLevel) - magicResist;

// 抗性减免
resistReduction = damage * (resistPercent / 100);

研究价值：装备系统/技能伤害/属性加成的教科书级实现，最接近 DNF 的参考对象。

状态：⏳ 待研究（主线完成后启动）

🏰 塔防 / RTS 类（继承僵尸开炮研究）

⭐ OpenRA ★★★★★ 推荐指数

项目：github.com/OpenRA/OpenRA

语言： C# ｜ ★ Stars： 15.3k

原作：红色警戒/泰伯利亚黎明 开源重制

可研究机制：

伤害类型（Pierce/Normal/Siege/Magic） 护甲克制表武器射程 建筑护甲单位碰撞 
// OpenRA 伤害计算（简化）
damage = weapon.Damage;
damage * = 1.0 + attacker.Modifiers.Damage;
// 护甲克制
vsArmor = weapon.Warheads[0].VersusArmor[target.ArmorType];
damage * = vsArmor;
// 最小伤害保护
if (damage < weapon.MinDamage) damage = weapon.MinDamage;

研究价值：塔防/RTS 伤害类型克制系统最完善的参考，向僵尸开炮可借鉴。

状态：⏳ 待研究（主线完成后启动）
