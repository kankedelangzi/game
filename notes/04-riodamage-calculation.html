三、70版本柔道核心机制详解

🎯 抓取机制原理

柔道的抓取技能在70版本具有无视防御的特性：

  • 无视物理防御：抓取技能伤害不受目标物理防御减伤影响
  • 无视部分无敌：部分抓取技能可穿透无敌状态（如死亡之握）
  • 强制抓取：对可抓取目标100%命中（抓取强化被动加成）
  • 抓取状态：被抓取目标无法行动、无法使用技能、无法闪避

📐 抓取技能伤害公式

抓取技能最终伤害 = 基础固伤 × (1 + 独立/1000) × (1 + 力量/250) × 技能加成 × 空斩打加成 × 摔技强化加成

关键区别：
- 抓取技能不计算防御减伤（无视防御）
- 固伤技能正常计算独立攻击力
- 百分比技能正常计算物理攻击力

⚡ 摔技连招系统

经典连招：背摔 → 折颈术 → 抛投 → 螺旋彗星落

1. 背摔（起手抓取）：4,956固伤 + 抓取状态
2. 折颈术（控制）：6,608固伤 + 眩晕3秒
3. 抛投（核心输出）：10,030固伤 + 200%百分比追加
4. 螺旋彗星落（爆发）：14,160固伤（多段）+ 300%百分比落地

连招总伤害 ≈ 各技能伤害之和（抓取状态下无缝衔接）

五、伤害计算实例

实例1：抛投（抓取固伤核心）- 完整计算

【技能参数（70版本，E2+2后Lv20）】
- 技能固伤基数：8500 × 1.18 = 10,030
- 第二段百分比：200%（抛投落地追加）
- 抓取特性：无视防御
- CD：约12秒

【角色面板（法杖流）】
- 力量：950
- 独立攻击力：1150
- 物理暴击率：55%
- 空斩打被动：+50%抓取技能伤害
- 摔技强化被动：+30%摔技伤害（Lv6）

【逐步计算】
1. 基础固伤 = 10,030

2. 独立攻击力加成：
   独立系数 = 1 + 独立/1000 = 1 + 1150/1000 = 2.15
   独立后伤害 = 10,030 × 2.15 = 21,565

3. 力量加成：
   力量系数 = 1 + 力量/250 = 1 + 950/250 = 4.8
   力量后伤害 = 21,565 × 4.8 = 103,512

4. 空斩打加成（抓取技能+50%）：
   空斩打后伤害 = 103,512 × 1.50 = 155,268

5. 摔技强化加成（+30%）：
   摔技后伤害 = 155,268 × 1.30 = 201,848

   ⚠️ 抓取技能无视防御，无需计算防御减伤！

6. 暴击期望：
   暴击倍率 = 150%（70版本无额外暴伤装备）
   暴击期望系数 = (1 - 0.55) × 1.0 + 0.55 × 1.5 = 1.325
   暴击期望伤害 = 201,848 × 1.325 = 267,449

7. 第二段百分比追加（200%）：
   基础百分比 = 1800 × 200% = 3,600
   力量加成 = 3,600 × 4.8 = 17,280
   暴击期望 = 17,280 × 1.325 = 22,896

💡 抛投第一发暴击期望：267,449
💡 抛投第二段暴击期望：22,896
💡 抛投总伤害（暴击期望）：267,449 + 22,896 = 290,345

【结论】
抛投是柔道70版本核心输出技能，单发约29万暴击期望。
抓取无视防御，对高防BOSS收益极高。

实例2：螺旋彗星落（多段固伤爆发）

【技能参数（70版本，Lv20）】
- 技能固伤基数：12000 × 1.18 = 14,160
- 段数：约6段（旋转摔打）
- 每段固伤：约2,360
- 落地百分比：300%
- CD：约15秒

【逐步计算（固伤部分）】
1. 基础固伤 = 14,160

2. 独立加成 = 14,160 × 2.15 = 30,444

3. 力量加成 = 30,444 × 4.8 = 146,131

4. 空斩打加成 = 146,131 × 1.50 = 219,197

5. 摔技强化加成 = 219,197 × 1.30 = 284,956

6. 暴击期望 = 284,956 × 1.325 = 377,567

💡 螺旋彗星落固伤部分暴击期望：377,567

【落地百分比追加】
1. 基础百分比 = 1800 × 300% = 5,400
2. 力量加成 = 5,400 × 4.8 = 25,920
3. 暴击期望 = 25,920 × 1.325 = 34,344

💡 螺旋彗星落总伤害（暴击期望）：377,567 + 34,344 = 411,911

【结论】
螺旋彗星落是柔道单体爆发天花板，约41万暴击期望。
多段击打适合触发武器特效，大范围适合群怪。

七、柔道70版本伤害特点总结

  1. 抓取无视防御：柔道最大优势，对高防BOSS收益远超其他职业
  2. 固伤为主，百分比为辅：抛投、彗星落、死亡之握为核心，雷霆膝击/碎骨为补刀
  3. 连招体系成熟：背摔→折颈→抛投→彗星落是70版本柔道标准连招
  4. 控制能力强：折颈眩晕、死亡之握全屏抓取，副本控制需求高
  5. 异界套装依赖：E2「疾风追击」是抛投流核心，E3「死亡之握」是控制流核心
  6. 空斩打被动关键：满级+50%抓取技能伤害，是柔道伤害计算的核心乘区

💡 伤害计算公式速查

抓取固伤最终伤害 = 基础固伤 × (1 + 独立/1000) × (1 + 力量/250) × 空斩打加成 × 摔技强化加成

注意：抓取技能无视防御减伤！

百分比最终伤害 = 基础倍率 × 物理攻击 × (1 + 力量/250) × 防御减伤 × 暴击期望

防御减伤 = 300 / (300 + 目标防御)
暴击期望系数 = (1 - 暴击率) + 暴击率 × 1.5
