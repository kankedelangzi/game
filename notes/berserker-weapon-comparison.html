二、流派机制差异

🔴 钝器流（固伤为主）

核心机制独立攻击 + 固定伤害
技能构成固伤技能占比 ~70%
暴走收益+8~12%（面板影响小）
强化收益低（固伤不受物理攻击加成）
高防BOSS优势（穿透防御）
清图效率一般（范围技能较少）

🔵 巨剑流（百分比为主）

核心机制物理攻击 + 武器强化
技能构成百分比技能占比 ~60%
暴走收益+18~25%（面板收益高）
强化收益高（+12/+13显著）
高防BOSS衰减（受防御减伤）
清图效率优秀（范围技能多）

三、伤害公式解析

固伤技能伤害：
fixed = base × (1 + independent / 250) × skill_rate

百分比技能伤害：
percent = base × (1 + str / 250) × (1 + weapon_atk / 1000) × (1 + weapon_str / 250)

关键发现

  • 固伤技能 不受物理攻击影响，只依赖独立攻击和基础数值
  • 百分比技能 受物理攻击/强化等级影响，巨剑流优势明显
  • 70版本 无黄白字叠加，装备选择更直接
  • 暴走加成 对固伤零收益，仅影响百分比技能

八、版本对比与未来展望

版本伤害机制演变：
60版本：+25% 暴走 → 70版本：+40% 暴走 → 80版本：+45% 暴走 → 85版本：+50% 暴走

关键机制变化：
80版本引入黄白字叠加 → 百分比技能收益大幅提升 → 巨剑流优势扩大
85版本出血机制削弱 → 穿透防御被移除 → 钝器流优势缩小

70版本末期 是钝器流的黄金时期，出血穿透防御机制为固伤流派提供了独特优势。进入80版本后，机制变化导致流派格局重新洗牌。
