🔴 DNF：狂战士（红眼）70版本伤害实例

【狂战士 · 崩山裂地斩（血爆）单发伤害计算（70版本）】

基础信息：
  武器：70粉巨剑（+1200物攻）
  防具：远古套装（提升率+35%）
  首饰：悲鸣首饰（+10%暴伤）

最终伤害 = 基础伤害
    × (1 + 力量/250)              # 主属性乘区：力量1800 → 1+1800/250 = 8.2
    × (1 + Σ所有攻击力%)           # 加算乘区：远古套35% = 1.35
    × (1 - 防御减伤)              # 防御乘区：远古图BOSS防御8000 → 减伤=8000/(8000+70×200)=36.4%
    × (1 + 属性强化/220)          # 属强乘区：火强66 → 1+66/220 = 1.3
    × 暴击倍率(1.5×(1+暴伤%))    # 暴击乘区：暴伤+40% → 1.5×1.4 = 2.1

代入计算：
  基础伤害（血爆14级）= 35000（无视防御）+ 80000（受防御影响）
  防御后基础 = 35000 + 80000×0.636 = 85880
  力量乘区：85880 × 8.2 = 704216
  所有攻击力：704216 × 1.35 = 950692
  属强乘区：950692 × 1.3 = 1235900
  暴击乘区：1235900 × 2.1 = 2595390

✅ 血爆单发（暴击）≈ 259万伤害
   非暴击：259万 ÷ 2.1 × 1.5 = 185万
   期望（45%暴击率）：185×0.55 + 259×0.45 = 102+116 = 218万

🔍 关键：力量乘区贡献最大（8.2倍），暴击乘区次之（2.1倍）

⚔️ DNF：剑魂（白手）70版本感电流伤害实例

【剑魂 · 猛龙断空斩 + 感电伤害计算（70版本）】

核心机制：感电状态 = 每次攻击附加固定感电伤害（不随武器强化变化）

基础信息：
  武器：70粉光剑（+1350物攻，感电触发）
  防具：寂静套装（高暴击）
  首饰：悲鸣套（+10%暴伤）

感电伤害公式（70版本）：
  感电单次 = 1000 + 2.5×物攻，每0.5秒触发一次，持续6秒 = 12次

代入计算（70粉光剑触发感电）：
  单次感电 = 1000 + 2.5×1350 = 1000 + 3375 = 4375
  12次感电总伤害 = 4375 × 12 = 52500（未算乘区）

  加上乘区（力量1600，暴击率50%，暴伤+40%）：
  52500 × (1+1600/250)=7.4 × 1.35(所有攻)× 1.3(属强)× 2.1(暴击)
  = 52500 × 7.4 × 1.35 × 1.3 × 2.1
  ≈ 52500 × 25.4 ≈ 1,333,500

✅ 感电12次总伤害 ≈ 120万（暴击时）
   + 猛龙断空斩本身 ≈ 500万
   = 总计约 620万（单次猛龙全命中）

🔍 关键：感电是固定伤害，不受武器强化影响，平民玩家友好

👊 KOF XIII：八神庵 5B→6A→葵花×3→八稚女 连招实例

【八神庵 · 标准连招伤害计算（无能量/无HD/无Max）】

连招序列：5B（轻拳）→ 6A（重拳）→ 葵花×3（↓↘→+A×3）→ 八稚女（↓↘→↘↓↘→+A）

伤害衰减表（85%衰减率）：
  第1击：5B   基础=40  → 衰减1.00 → 实际=40
  第2击：6A   基础=70  → 衰减0.85 → 实际=59.5
  第3击：葵花1 基础=60  → 衰减0.72 → 实际=43.2
  第4击：葵花2 基础=60  → 衰减0.61 → 实际=36.6
  第5击：葵花3 基础=60  → 衰减0.52 → 实际=31.2
  第6击：八稚女 基础=150 → 衰减0.44 → 实际=66.0

连招总伤害 = 40+59.5+43.2+36.6+31.2+66.0 = 276.5（27.65%血量）
随机波动±10% → 实际范围：248.85 ~ 304.15（24.9% ~ 30.4%）

【带1颗能量 + Max模式】
  能量加成 = 1 + 1×0.05 = 1.05，Max模式 = 1.3
  总伤害 = 276.5 × 1.05 × 1.3 = 377.5（37.75%血量）

【带2颗能量 + HD模式 + Max模式】
  能量=1.10，HD=1.1，Max=1.3
  总伤害 = 276.5 × 1.10 × 1.1 × 1.3 = 435.3（43.53%血量）

🔍 关键：6连击衰减后只剩44%效果，防止一击必杀

🔥 KOF XIII：草薙京 5B→6A→荒咬→毒咬→大蛇薙 连招实例

【草薙京 · 同系招式衰减惩罚】

连招序列：5B→6A→荒咬（↓↘→+A）→毒咬（荒咬中→↘↓↙←+A）→大蛇薙（↓↙←↙↓↘→+A）

特殊规则：同系招式重复出现，后续伤害强制衰减至10%
  荒咬出现后，毒咬（同系）衰减系数强制 = 0.10

伤害计算：
  第1击：5B = 35 × 1.00 = 35
  第2击：6A = 65 × 0.85 = 55.25
  第3击：荒咬 = 80 × 0.72 = 57.6
  第4击：毒咬 = 90 × 0.10（强制衰减!）= 9.0
  第5击：大蛇薙 = 160 × 0.52 = 83.2

总伤害 = 35 + 55.25 + 57.6 + 9.0 + 83.2 = 240.05（24.0%血量）

🔍 关键：草薙京连招伤害低于八神，大蛇薙单体高，适合确反一击

🧟 向僵尸开炮：机枪塔 vs 装甲僵尸（穿透机制实例）

【机枪塔 · 攻击装甲僵尸伤害计算（含穿透）】

基础信息：
  机枪塔等级：20级，基础攻击力：450
  攻击速度：8发/秒，物理穿透：30
  暴击率：25%，暴击伤害：250%

装甲僵尸属性：护甲=80，血量=5000，移动速度=慢

伤害计算（单发）：
  有效护甲 = max(80-30, 0) = 50
  护甲减伤 = 50/(50+150) = 0.25（保留75%伤害）
  单发伤害 = 450 × 0.75 = 337.5
  暴击时：337.5 × 2.5 = 843.75
  随机波动±5%：320.6~354.4（普通）/ 801.6~886.0（暴击）

期望单发 = 337.5×0.75 + 843.75×0.25 = 253.1 + 210.9 = 464.0
DPS = 464.0 × 8 = 3712
击杀用时：5000 ÷ 3712 ≈ 1.35秒

【对比：无穿透时】
  有效护甲=80，减伤=80/230=34.8%，单发=450×0.652=293.4
  期望单发=293.4×0.75+733.5×0.25=220+183.4=403.4
  DPS=403.4×8=3227，击杀用时≈1.55秒

✅ 穿透30点 → DPS提升15.0%，击杀时间缩短13.0%
🔍 关键：穿透对高护甲单位效果显著，对低护甲收益递减

三、完整伤害计算推演（逐步分解）

3.1 DNF 狂战士 远古图 30秒输出循环推演（70版本）

【狂战士 · 30秒输出循环（远古图）】

技能循环：暴走→血之狂暴→崩山击（起手）→怒气爆发→血爆→大崩→普攻填充

各技能伤害（暴击时，含所有乘区）：
  崩山击：约 100万
  怒气爆发：约 200万（多段，共8hit）
  血爆（核心）：约 336万（14级，全中）
  大崩：约 400万（16级，全中）
  普攻（30秒内约20hit）：约 50万（每hit约2.5万）

30秒总伤害 = 100+200+336+400+50 = 1086万
理论DPS = 1086万 ÷ 30秒 ≈ 36.2万/秒

【考虑实战因素】
  - 技能命中率：约85%（部分技能未全中或被打断）
  - 暴击率：45%（实际暴击次数减少）
  - 怪物移动（需要预判）：命中率再-10%
  实战修正：1086万 × 0.85 × 0.9 ≈ 831万
  实战DPS = 831万 ÷ 30秒 ≈ 27.7万/秒

【团队增益（远古图常见）】
  圣骑士荣誉祝福 → 831万 × 1.35 = 1122万
  实战DPS（含队友增益）= 1122万 ÷ 30秒 ≈ 37.4万/秒

✅ 远古图输出标准：狂战士需达到 30万+ DPS 才算合格

3.2 KOF XIII 能量管理与连招选择（博弈层面）

【八神庵 · 能量管理决策树】

  结果：击杀对手，但你只剩1颗能量
  评价：极致爆发，但资源耗尽，被反打风险高 ⚠️

【AI决策模拟】
  if 对手血量 < 40% and 我方能量 >= 1:
      选择 选项B（Max模式击杀）
  elif 对手血量 < 30% and 我方能量 >= 2:
      选择 选项C（HD击杀，更稳妥）
  else:
      选择 选项A（保留资源，等待机会）

🔍 关键：KOF伤害系统不只是数字，更是资源管理博弈

3.3 向僵尸开炮 第50波BOSS战推演

【第50波 · 巨型僵尸BOSS伤害计算】

BOSS属性：血量=500000（50万），护甲=150（重型），移动速度=极慢
特殊：每10秒召唤小僵尸×10

防御塔配置（满配）：
  机枪塔×3（20级，穿透30）
  电磁塔×2（15级）
  激光塔×1（25级）
  火箭塔×2（18级）

各塔对BOSS的DPS：
  机枪塔（单座）：
    有效护甲=120，减伤=120/270=44.4%

    DPS=344.1×8=2753，三座总计=8259 DPS

  电磁塔（单座）：
    有效护甲=120，减伤=44.4%
    单发=300×0.556=166.8，期望=216.9，DPS=433.8
    两座总计=867 DPS

  激光塔（单座）：
    有效护甲=120，减伤=44.4%
    基础=150/秒×0.556=83.4/秒，期望=102.2/秒

  火箭塔（单座，范围对单体减半）：
    单发=800×0.556×0.5=222.4，期望=305.8，DPS=305.8
    两座总计=611.8 DPS

总DPS = 8259+867+102.2+611.8 = 9840 DPS
击杀时间 = 500000 ÷ 9840 ≈ 50.8秒

【考虑小僵尸干扰】
  每10秒召唤10只小僵尸（每只500血）
  需分1座机枪塔处理小僵尸 → 主BOSS DPS减少2753
  修正后DPS = 9840-2753 = 7087
  修正击杀时间 = 500000 ÷ 7087 ≈ 70.5秒

✅ 结论：第50波需要合理分配火力，否则会被小僵尸拖垮
🔍 关键：塔防不只是堆伤害，更需要科学的火力分配
