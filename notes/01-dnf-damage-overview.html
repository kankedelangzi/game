一、伤害系统核心架构

DNF的伤害计算是一个多层次的复合系统，主要包含以下几个层面：

最终伤害 = 基础伤害 × 独立增伤乘区 × 暴击乘区 × 属性乘区 × 防御减伤 × 其他修正

1.1 伤害类型分类

伤害类型典型来源代表职业
物理百分比伤害大部分物理职业主动技能剑魂、狂战士（部分技能）
魔法百分比伤害大部分魔法职业主动技能元素师、贝亚娜斗神
物理固伤特定物理技能狂战士（崩山击、怒气爆发）
魔法固伤特定魔法技能阿修罗（邪光斩、波动爆发）
混伤部分特殊技能大枪（量子爆弹）、决战者

二、60版本 vs 70版本核心差异

2.1 版本演变关键节点

版本核心机制变化伤害乘区数量
60版本（早期）基础伤害体系，装备属性简单3-4个
70版本异界套装出现，技能等级提升4-5个
70版本末期异界E2/E3、远古套、CC套成型5-6个

2.2 增伤乘区对比

【60版本典型增伤】
最终伤害 = 技能基础伤害 
          × (1 + 力量/250)              # 主属性乘区
          × 武器强化系数                # 强化乘区
          × 物理/魔法暴击伤害(1.5)      # 暴击乘区

【70版本典型增伤】
最终伤害 = 技能基础伤害 
          × (1 + 力量/250)                    # 主属性乘区
          × (1 + 所有攻击力加成)               # 所有攻击力乘区（加算）
          × (1 + 暴击伤害加成 - 基础暴伤)      # 暴击伤害乘区
          × (1 + 属性强化/220)                # 属性强化乘区（70版本已成型）
          × (1 - 怪物防御减伤)                # 防御减伤

三、核心属性说明

3.1 主属性（力量/智力）

物理/魔法攻击力 = 武器基础攻击力 × (1 + 力量/250)

独立攻击力 = 武器基础独立攻击力 × (1 + 力量/250)  # 固伤职业核心属性
设计意图：
  • 提供基础伤害增幅，鼓励堆主属性
  • 250作为分母，意味着每250点主属性翻倍伤害
  • 60版本主属性收益更高（乘区少），70版本相对稀释

3.2 物理/魔法暴击

暴击率 = 基础暴击率 + 装备暴击率 + 技能暴击率
上限：50%（60版本）→ 70版本可堆至80%

暴击伤害 = 普通伤害 × 1.5  # 基础暴击伤害150%

70版本暴伤加成：
实际暴击伤害 = 1.5 × (1 + 暴伤加成%)

3.3 属性强化系统（70版本已成型）

属性伤害加成 = 属性强化值 / 220

实际伤害 = 基础伤害 × (1 + 属性强化/220)

典型数值：
- 22点属强 → +10%伤害
- 44点属强 → +20%伤害
- 66点属强 → +30%伤害
- 88点属强 → +40%伤害

🗡️ 实例1：狂战士（红眼）- 物理固伤爆发流

角色配置（70版本满级）

属性数值说明
力量780含装备、BUFF（暴走+10%力）
独立攻击力1100武器+附魔（70版本水平）
物理暴击率45%含装备和技能
武器70紫巨剑基础独立约480
核心技能崩山击（怒气爆发）固伤为主

崩山击（满级）伤害计算

技能基础固伤：约 3600（70版本满级崩山击）
独立攻击力加成：1100 / 1000 = 1.1 倍
力量加成：(1 + 780/250) = 4.12 倍
暴击判定：45%概率触发 × 1.5倍暴击伤害
防御减伤：假设怪物防御中等，减伤约25%

计算步骤：
1. 基础伤害 = 3600 × (1 + 1100/1000) = 3600 × 2.1 = 7560
2. 力量加成 = 7560 × 4.12 = 31147
3. 防御减伤 = 31147 × (1 - 0.25) = 23360
4. 暴击期望 = 23360 × (1 × 0.55 + 1.5 × 0.45) = 23360 × 1.225 = 28616

💡 实际命中伤害：约 28,600（非暴击）→ 42,900（暴击）

怒气爆发（满级）伤害计算

技能基础固伤：约 6500（多段伤害，每段约1625，共4段）
独立攻击力加成：1100 / 1000 = 1.1 倍
力量加成：(1 + 780/250) = 4.12 倍
暴击判定：每段独立计算暴击

计算步骤（单段）：
1. 基础伤害 = 1625 × 2.1 = 3413
2. 力量加成 = 3413 × 4.12 = 14062
3. 防御减伤 = 14062 × 0.75 = 10547
4. 暴击期望 = 10547 × 1.225 = 12920

💡 总伤害（4段）：约 42,188（非暴击）→ 63,282（全暴击）

狂战士伤害特点分析

  • 优势 固伤技能占比高，独立攻击力收益大
  • 优势 暴走技能提供10%力量加成，爆发能力强
  • 优势 出血状态持续伤害，不受防御影响（穿透）
  • 劣势 百分比技能较少，武器强化收益相对较低
  • 关键点 70版本堆独立攻击力是核心，优先附魔独立
  • 关键点 血气分流可换装堆等级，提升暴走力量加成

  • 穿透机制 出血伤害无视防御，对高防御BOSS特别有效

十字斩（满级Lv20）伤害计算 + 出血触发

【面板数据】
角色力量：800（含暴走）
独立攻击力：1150
武器基础独立：520
强化+12系数：1.95
实际独立 = 520 × 1.95 + 副武器/装备 = 约1150

【十字斩技能数据（Lv20）】
技能固伤：约8500（70版本满级）
出血触发概率：90%
出血等级：Lv23（异界E2+3）
出血固定值：4200/秒，持续5秒

【步骤1：十字斩直接伤害】
基础固伤 = 8500
独立加成 = (1 + 1150/1000) = 2.15倍
力量加成 = (1 + 800/250) = 4.2倍
防御减伤（假设BOSS防御6000）= 6000/(6000+70×200) = 6000/20000 = 30%
实际减伤 = 1 - 0.30 = 70%

直接伤害（非暴击）= 8500 × 2.15 × 4.2 × 0.70 = 53,659
直接伤害（暴击150%）= 53,659 × 1.5 = 80,489

【步骤2：出血持续伤害（穿透防御）】
出血每秒伤害 = 4200 × (1 + 1150/1000) × (1 + 800/250)
              = 4200 × 2.15 × 4.2 = 37,926/秒
总出血伤害（5秒）= 37,926 × 5 = 189,630

💡 关键：出血伤害不受防御影响！对高防御BOSS同样有效

【步骤3：十字斩 + 出血总伤害（非暴击）】
总伤害 = 53,659（直接）+ 189,630（出血）= 243,289

【步骤4：十字斩 + 出血总伤害（暴击）】
总伤害 = 80,489（直接暴击）+ 189,630（出血）= 270,119

【步骤5：暴击期望（50%暴击率）】
期望伤害 = 243,289 × 0.5 + 270,119 × 0.5 = 256,704

💡 十字斩+出血总伤害：约 25.7万（暴击期望）
💡 出血占比：189,630 / 256,704 = 73.9%（出血是主要伤害来源）

血气分流 + 暴走连招爆发计算

【连招：血气分流 → 暴走 → 崩山击 → 十字斩 → 怒气爆发】

【前置BUFF：血气分流 + 暴走】
血气分流（Lv10，70版本）：
  - 增加暴走技能等级 +10
  - 暴走原Lv10 → 变为Lv20
  - 暴走Lv20效果：力量+35%，攻击速度+35%，移动速度+30%

  基础力量（无BUFF）：728
  力量加成 = 728 × 1.35 = 982
  暴走状态下力量 = 728 + 982 = 1710

【步骤1：崩山击（满级Lv23，异界E2+3）】
技能固伤：约3600 × 1.18（等级提升）= 4248
独立加成：2.15倍
力量加成（暴走状态）= (1 + 1710/250) = 7.84倍

直接伤害（非暴击）= 4248 × 2.15 × 7.84 × 0.70 = 50,393
直接伤害（暴击）= 50,393 × 1.5 = 75,590

【步骤2：十字斩（满级Lv23）+ 出血】
技能固伤：8500 × 1.18 = 10,030
独立加成：2.15倍
力量加成：7.84倍
防御减伤：70%

直接伤害（非暴击）= 10,030 × 2.15 × 7.84 × 0.70 = 118,634
直接伤害（暴击）= 118,634 × 1.5 = 177,951

出血伤害（不受防御影响）= 37,926 × 5 = 189,630

十字斩+出血总伤害（非暴击）= 118,634 + 189,630 = 308,264
十字斩+出血总伤害（暴击）= 177,951 + 189,630 = 367,581

【步骤3：怒气爆发（满级Lv23，4段）】
技能固伤/段：约1625 × 1.18 = 1918
独立加成：2.15倍
力量加成：7.84倍
防御减伤：70%

单段伤害（非暴击）= 1918 × 2.15 × 7.84 × 0.70 = 22,689
单段伤害（暴击）= 22,689 × 1.5 = 34,034

4段总伤害（全非暴击）= 22,689 × 4 = 90,756
4段总伤害（全暴击）= 34,034 × 4 = 136,136

【连招总伤害计算（暴击期望50%）】
崩山击期望 = 50,393 × 0.5 + 75,590 × 0.5 = 62,992
十字斩+出血期望 = 308,264 × 0.5 + 367,581 × 0.5 = 337,923

  单段期望 = 22,689 × 0.5 + 34,034 × 0.5 = 28,362
  4段期望 = 28,362 × 4 = 113,448

连招总期望伤害 = 62,992 + 337,923 + 113,448 = 514,363

八、防御力减伤公式

实际减伤比例 = 怪物防御力 / (怪物防御力 + 角色等级 × 200)

示例对比：
1. 70级角色，怪物防御1800（普通白图）
   减伤 = 1800 / (1800 + 70×200) = 1800 / 15800 = 11.4%
   实际受到伤害 = 88.6%

2. 70级角色，怪物防御5000（远古图BOSS）
   减伤 = 5000 / (5000 + 70×200) = 5000 / 19000 = 26.3%
   实际受到伤害 = 73.7%

3. 70级角色，怪物防御8000（异界E3 BOSS）
   减伤 = 8000 / (8000 + 70×200) = 8000 / 22000 = 36.4%
   实际受到伤害 = 63.6%

💡 70版本高防御副本需要更高伤害才能有效输出
