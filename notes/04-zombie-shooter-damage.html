一、游戏伤害系统核心架构

"向僵尸开炮"的伤害计算是一个多层次的复合系统，主要包含以下几个层面：

最终伤害 = 基础伤害 
          × (1 + 攻击力加成%) 
          × 暴击判定(1.5~2.0) 
          × 穿透修正(1 - 护甲减伤) 
          × 技能加成 
          × 连击加成 
          × 随机波动(±5%)

二、防御塔伤害详细计算

2.1 机枪塔（基础炮塔）

机枪塔DPS = 基础攻击力 × 攻击速度 × (1 + 等级加成%) × (1 + 科技加成%)

其中：
- 基础攻击力：50（1级）→ 200（10级）
- 攻击速度：5次/秒（1级）→ 10次/秒（10级）
- 等级加成：每级+10%（线性增长）
- 科技加成：0-50%（科技树投入）

实例计算：10级机枪塔（无暴击）

配置：
- 等级：10级
- 基础攻击力：200
- 攻击速度：10次/秒
- 等级加成：90%（9级提升）
- 科技加成：30%

计算步骤：
1. 单次伤害 = 200 × (1 + 0.90 + 0.30) = 200 × 2.2 = 440
2. DPS = 440 × 10 = 4,400 伤害/秒

💡 对比1级机枪塔：
   单次伤害 = 50 × 1.0 = 50
   DPS = 50 × 5 = 250
   提升倍数：4400 / 250 = 17.6倍

暴击判定（机枪塔）

暴击率：基础5% + 装备15% = 20%
暴击伤害：基础1.5倍 + 装备0.5倍 = 2.0倍

期望DPS计算：
基础DPS = 4,400
暴击期望 = 4,400 × (1 × 0.80 + 2.0 × 0.20) = 4,400 × 1.2 = 5,280

💡 暴击让DPS提升20%，堆暴击率到40%时：
   期望DPS = 4,400 × (1 × 0.60 + 2.0 × 0.40) = 4,400 × 1.4 = 6,160

2.2 激光炮（高伤单体）

激光炮伤害 = 基础攻击力 × (1 + 等级加成%) × 穿透系数 × 暴击判定

穿透系数：
- 无穿透：1.0（普通僵尸）
- 轻甲穿透：1.2（铁桶僵尸）
- 重甲穿透：1.5（装甲车僵尸）

实例计算：10级激光炮打不同僵尸

配置：
- 等级：10级
- 基础攻击力：800
- 等级加成：90%
- 暴击率：10%，暴击伤害1.8倍

基础单次伤害 = 800 × 1.9 = 1,520

对普通僵尸（无护甲）：
实际伤害 = 1,520 × 1.0 = 1,520
暴击期望 = 1,520 × (1 × 0.90 + 1.8 × 0.10) = 1,520 × 1.08 = 1,641.6

对铁桶僵尸（轻甲，减伤30%）：
实际伤害 = 1,520 × (1 - 0.30) = 1,064
暴击期望 = 1,064 × 1.08 = 1,149.1

对装甲车僵尸（重甲，减伤60%）：
实际伤害 = 1,520 × (1 - 0.60) = 608
暴击期望 = 608 × 1.08 = 656.6

💡 激光炮对重甲僵尸伤害骤降，需要升级穿透科技

2.3 冰冻炮（控制+伤害）

冰冻炮伤害 = 基础攻击力 × (1 + 等级加成%) × 减速系数

减速效果：
- 基础减速：30%（持续3秒）
- 升级后：50%（持续5秒）
- 配合其他炮塔：增伤20%（冻结状态）

实例计算：冰冻炮配合机枪塔

配置：
- 冰冻炮10级：基础100，攻速5次/秒，等级加成+90%
- 机枪塔10级：440单次（见上文），攻速10次/秒

冰冻炮单次伤害 = 100 × 1.9 = 190
冰冻炮DPS = 190 × 5 = 950

配合效果（僵尸被冻结）：
机枪塔增伤20% → 单次伤害 = 440 × 1.2 = 528
机枪塔DPS = 528 × 10 = 5,280

组合DPS = 950 + 5,280 = 6,230

💡 冰冻炮本身伤害不高，但控制+增伤让组合DPS提升19%
💡 关键是冻结状态让其他炮塔享受增伤buff

3.2 主角射击实例

场景：主角射击打BOSS僵尸（护甲50%减伤）

配置：
- 基础攻击力：300
- 攻击力加成：50%（装备）
- 暴击率：30%
- 暴击伤害：2.5倍
- 穿透等级：3级（30%护甲穿透）

计算步骤：
1. 基础伤害 = 300 × (1 + 0.50) = 450
2. 穿透后护甲减伤 = 50% × (1 - 0.30) = 35%
3. 穿透后伤害 = 450 × (1 - 0.35) = 292.5
4. 暴击期望 = 292.5 × (1 × 0.70 + 2.5 × 0.30) = 292.5 × 1.45 = 424.1

💡 无穿透时（50%减伤）：
   伤害 = 450 × 0.50 = 225
   暴击期望 = 225 × 1.45 = 326.3
   穿透提升：(424.1 - 326.3) / 326.3 = 30%提升

💡 暴击是关键！30%暴击率+2.5倍暴伤让期望提升45%

5.1 护甲类型与减伤

僵尸类型护甲减伤应对策略

铁桶僵尸30%穿透炮塔/主角射击
装甲车僵尸60%高穿透/技能

5.2 穿透机制

实际护甲减伤 = 怪物护甲减伤 × (1 - 穿透等级%)

穿透等级：
- 1级：10%护甲穿透
- 2级：20%护甲穿透
- 3级：30%护甲穿透
- 5级：50%护甲穿透（满级）

示例：装甲车僵尸（60%护甲），3级穿透
实际护甲减伤 = 60% × (1 - 0.30) = 42%
减伤效果从60%降低到42%，提升伤害：1 / 0.58 = 1.72倍

七、连击系统与伤害加成

7.1 连击机制

连击加成 = 1 + (连击数 × 0.05)  # 每次连击+5%伤害
上限：连击100 = 500%加成（6倍伤害）

连击数获取：
- 每次命中：+1连击
- 暴击：+2连击
- 击杀：+5连击

连击衰减：停止攻击3秒后，每秒-10连击

7.2 连击实例

场景：主角连续射击30秒（无暴击）

配置：
- 基础DPS：1,000（主角射击）
- 攻击间隔：0.5秒（2次/秒）
- 每次命中+1连击

30秒后连击数 = 30 × 2 = 60连击
连击加成 = 1 + (60 × 0.05) = 4.0倍

实际DPS = 1,000 × 4.0 = 4,000
总伤害 = 4,000 × 30 = 120,000

💡 对比无连击系统：总伤害 = 1,000 × 30 = 30,000
💡 连击系统让持续输出提升4倍！

九、完整伤害计算代码（Python模拟）

def calculate_tower_dps(base_atk, atk_speed, level, tech_bonus, crit_rate, crit_dmg):
    """
    炮塔DPS计算
    """
    # 等级加成（每级10%）
    level_bonus = level * 0.10
    
    # 单次伤害
    per_hit = base_atk * (1 + level_bonus + tech_bonus)
    
    # 暴击期望
    crit_expectation = 1 * (1 - crit_rate) + crit_dmg * crit_rate
    
    # DPS
    dps = per_hit * atk_speed * crit_expectation
    
    return dps

def calculate_player_shoot(base_atk, atk_bonus, crit_rate, crit_dmg, penetration, monster_armor):
    """
    主角射击伤害计算
    """
    # 基础伤害
    base_dmg = base_atk * (1 + atk_bonus)
    
    # 穿透后护甲减伤
    actual_armor = monster_armor * (1 - penetration)
    armor_reduction = 1 - actual_armor
    
    # 穿透后伤害
    dmg_after_armor = base_dmg * armor_reduction
    
    # 暴击期望
    crit_expectation = 1 * (1 - crit_rate) + crit_dmg * crit_rate
    
    final_dmg = dmg_after_armor * crit_expectation
    
    return final_dmg

# 使用示例
