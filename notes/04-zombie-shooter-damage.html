一、游戏伤害系统核心架构

"向僵尸开炮"的伤害计算是一个多层次的复合系统，主要包含以下几个层面：

最终伤害 = 基础伤害 
          × (1 + 攻击力加成%) 
          × 暴击判定(1.5~2.0) 
          × 穿透修正(1 - 护甲减伤) 
          × 技能加成 
          × 连击加成 
          × 随机波动(±5%)

二、防御塔伤害详细计算

2.1 机枪塔（基础炮塔）

机枪塔DPS = 基础攻击力 × 攻击速度 × (1 + 等级加成%) × (1 + 科技加成%)

其中：
- 基础攻击力：50（1级）→ 200（10级）
- 攻击速度：5次/秒（1级）→ 10次/秒（10级）
- 等级加成：每级+10%（线性增长）
- 科技加成：0-50%（科技树投入）

实例计算：10级机枪塔（无暴击）

配置：
- 等级：10级
- 基础攻击力：200
- 攻击速度：10次/秒
- 等级加成：90%（9级提升）
- 科技加成：30%

计算步骤：
1. 单次伤害 = 200 × (1 + 0.90 + 0.30) = 200 × 2.2 = 440
2. DPS = 440 × 10 = 4,400 伤害/秒

💡 对比1级机枪塔：
   单次伤害 = 50 × 1.0 = 50
   DPS = 50 × 5 = 250
   提升倍数：4400 / 250 = 17.6倍

暴击判定（机枪塔）

暴击率：基础5% + 装备15% = 20%
暴击伤害：基础1.5倍 + 装备0.5倍 = 2.0倍

期望DPS计算：
基础DPS = 4,400
暴击期望 = 4,400 × (1 × 0.80 + 2.0 × 0.20) = 4,400 × 1.2 = 5,280

💡 暴击让DPS提升20%，堆暴击率到40%时：
   期望DPS = 4,400 × (1 × 0.60 + 2.0 × 0.40) = 4,400 × 1.4 = 6,160

2.2 激光炮（高伤单体）

激光炮伤害 = 基础攻击力 × (1 + 等级加成%) × 穿透系数 × 暴击判定

穿透系数：
- 无穿透：1.0（普通僵尸）
- 轻甲穿透：1.2（铁桶僵尸）
- 重甲穿透：1.5（装甲车僵尸）

实例计算：10级激光炮打不同僵尸

配置：
- 等级：10级
- 基础攻击力：800
- 等级加成：90%
- 暴击率：10%，暴击伤害1.8倍

基础单次伤害 = 800 × 1.9 = 1,520

对普通僵尸（无护甲）：
实际伤害 = 1,520 × 1.0 = 1,520
暴击期望 = 1,520 × (1 × 0.90 + 1.8 × 0.10) = 1,520 × 1.08 = 1,641.6

对铁桶僵尸（轻甲，减伤30%）：
实际伤害 = 1,520 × (1 - 0.30) = 1,064
暴击期望 = 1,064 × 1.08 = 1,149.1

对装甲车僵尸（重甲，减伤60%）：
实际伤害 = 1,520 × (1 - 0.60) = 608
暴击期望 = 608 × 1.08 = 656.6

💡 激光炮对重甲僵尸伤害骤降，需要升级穿透科技

2.3 冰冻炮（控制+伤害）

冰冻炮伤害 = 基础攻击力 × (1 + 等级加成%) × 减速系数

减速效果：
- 基础减速：30%（持续3秒）
- 升级后：50%（持续5秒）
- 配合其他炮塔：增伤20%（冻结状态）

实例计算：冰冻炮配合机枪塔

配置：
- 冰冻炮10级：基础100，攻速5次/秒，等级加成+90%
- 机枪塔10级：440单次（见上文），攻速10次/秒

冰冻炮单次伤害 = 100 × 1.9 = 190
冰冻炮DPS = 190 × 5 = 950

配合效果（僵尸被冻结）：
机枪塔增伤20% → 单次伤害 = 440 × 1.2 = 528
机枪塔DPS = 528 × 10 = 5,280

组合DPS = 950 + 5,280 = 6,230

💡 冰冻炮本身伤害不高，但控制+增伤让组合DPS提升19%
💡 关键是冻结状态让其他炮塔享受增伤buff

七、连击系统与伤害加成

7.1 连击机制

连击加成 = 1 + (连击数 × 0.05)  # 每次连击+5%伤害
上限：连击100 = 500%加成（6倍伤害）

连击数获取：
- 每次命中：+1连击
- 暴击：+2连击
- 击杀：+5连击

连击衰减：停止攻击3秒后，每秒-10连击

7.2 连击实例

场景：主角连续射击30秒（无暴击）

配置：
- 基础DPS：1,000（主角射击）
- 攻击间隔：0.5秒（2次/秒）
- 每次命中+1连击

30秒后连击数 = 30 × 2 = 60连击
连击加成 = 1 + (60 × 0.05) = 4.0倍

实际DPS = 1,000 × 4.0 = 4,000
总伤害 = 4,000 × 30 = 120,000

💡 对比无连击系统：总伤害 = 1,000 × 30 = 30,000
💡 连击系统让持续输出提升4倍！

def calculate_tower_dps(base_atk, atk_speed, level, tech_bonus, crit_rate, crit_dmg):
    """
    炮塔DPS计算
    """
    # 等级加成（每级10%）
    level_bonus = level * 0.10
    
    # 单次伤害
    per_hit = base_atk * (1 + level_bonus + tech_bonus)
    
    # 暴击期望
    crit_expectation = 1 * (1 - crit_rate) + crit_dmg * crit_rate
    
    # DPS
    dps = per_hit * atk_speed * crit_expectation
    
    return dps

def calculate_player_shoot(base_atk, atk_bonus, crit_rate, crit_dmg, penetration, monster_armor):
    """
    主角射击伤害计算
    """
    # 基础伤害
    base_dmg = base_atk * (1 + atk_bonus)
    
    # 穿透后护甲减伤
    actual_armor = monster_armor * (1 - penetration)
    armor_reduction = 1 - actual_armor
    
    # 穿透后伤害
    dmg_after_armor = base_dmg * armor_reduction
    
    # 暴击期望
    crit_expectation = 1 * (1 - crit_rate) + crit_dmg * crit_rate
    
    final_dmg = dmg_after_armor * crit_expectation
    
    return final_dmg

十一、🔥 新增武器实例：火焰喷射器

11.1 火焰喷射器（持续伤害型）

武器特性

属性数值（10级）说明
基础伤害80/秒持续火焰伤害
燃烧伤害150/秒持续3秒的DOT
攻击范围3格扇形区域
攻击速度持续不需要装填
能量消耗10/秒需要能量管理

伤害计算公式

火焰喷射器总DPS = 直接伤害 + 燃烧DOT

直接伤害 = 基础伤害 × (1 + 等级加成%) × 持续时间
燃烧DOT = 燃烧伤害 × (1 + 等级加成%) × 燃烧持续时间 × 叠加层数

叠加机制：
- 每层燃烧独立计算
- 最多叠加3层
- 重新命中刷新持续时间

实例计算：10级火焰喷射器打僵尸群

场景：5只普通僵尸（HP 500）挤在一起

配置：
- 基础伤害：80/秒
- 燃烧伤害：150/秒
- 等级加成：90%（9级提升）
- 持续喷射：5秒

计算过程：
1. 直接伤害 = 80 × 1.9 × 5 = 760
2. 燃烧DOT（1层）= 150 × 1.9 × 3 = 855
3. 总伤害/只 = 760 + 855 = 1,615
4. 5只总伤害 = 1,615 × 5 = 8,075

💡 对比机枪塔（DPS 5,280）：
   机枪塔5秒伤害 = 5,280 × 5 = 26,400
   覆盖5只 = 26,400总伤害（但分散攻击）
   
💡 火焰喷射器优势：
   - 范围伤害，同时攻击多个目标
   - 持续DOT，即使僵尸离开范围也受伤
   - 对密集僵尸群效率极高
   
💡 劣势：
   - 单体DPS低于机枪塔
   - 需要能量管理
   - 对高血量BOSS效果一般

燃烧叠加实例

场景：持续喷射9秒，叠加3层燃烧

时间线：
- 0-3秒：1层燃烧（855伤害）
- 3-6秒：2层燃烧（855×2 = 1,710伤害）
- 6-9秒：3层燃烧（855×3 = 2,565伤害）

燃烧总伤害 = 855 + 1,710 + 2,565 = 5,130
总伤害 = 1,368 + 5,130 = 6,498

十二、⚡ 新增武器实例：雷电塔

12.1 雷电塔（连锁闪电型）

武器特性

属性数值（10级）说明
基础伤害300单次闪电伤害
连锁次数3次弹跳到附近目标
衰减系数0.7每次连锁伤害×0.7
攻击速度2次/秒间隔0.5秒
攻击范围4格首击范围

连锁伤害公式

雷电塔总伤害 = 首击伤害 + Σ(连锁伤害_i)

连锁伤害_i = 基础伤害 × 衰减系数^i × (1 + 等级加成%)

i = 0（首击）, 1（第一次连锁）, 2（第二次连锁）...

实例计算：10级雷电塔打4只僵尸

场景：4只普通僵尸（HP 500）排成一排，间距2格

配置：
- 基础伤害：300
- 等级加成：90%
- 连锁次数：3次
- 衰减系数：0.7

首击伤害 = 300 × 1.9 = 570
第1次连锁 = 570 × 0.7 = 399
第2次连锁 = 399 × 0.7 = 279.3
第3次连锁 = 279.3 × 0.7 = 195.5

总伤害 = 570 + 399 + 279.3 + 195.5 = 1,443.8

分配到4只僵尸：
- 首击目标：570伤害 → 击杀（HP 500），溢出70
- 第2只：399伤害 → 击杀，溢出-101（实际499）
- 第3只：279.3伤害 → 剩余HP 220.7

class Weapon:
    def __init__(self, name, base_dmg, atk_speed, range_type, special=""):
        self.name = name
        self.base_dmg = base_dmg
        self.atk_speed = atk_speed
        self.range_type = range_type  # "single", "aoe", "chain", "dot"
        self.special = special
        
    def calculate_dps(self, level=10, crit_rate=0.15, crit_dmg=2.0, 
                     monster_armor=0.0, penetration=0.0):
        """计算DPS（考虑等级、暴击、护甲、穿透）"""
        # 等级加成
        level_bonus = (level - 1) * 0.10
        per_hit = self.base_dmg * (1 + level_bonus)
        
        # 穿透后护甲
        actual_armor = monster_armor * (1 - penetration)
        armor_reduction = 1 - actual_armor
        
        # 实际伤害
        dmg_after_armor = per_hit * armor_reduction
        
        # 暴击期望
        crit_expectation = 1 * (1 - crit_rate) + crit_dmg * crit_rate
        
        # 单次期望伤害
        expected_dmg = dmg_after_armor * crit_expectation
        
        # DPS
        dps = expected_dmg * self.atk_speed
        
        return dps

# 创建武器实例
weapons = [
    Weapon("机枪塔", 200, 10, "single"),
    Weapon("激光炮", 800, 2, "single", "penetration"),
    Weapon("冰冻炮", 100, 5, "single", "slow"),
    Weapon("火焰喷射器", 80, 19, "dot"),  # 80/秒, 持续19秒总DPS
    Weapon("雷电塔", 300, 2, "chain"),
    Weapon("导弹塔", 600, 0.5, "aoe")
]

# 输出DPS对比
print("=== 武器DPS对比（10级，20%暴击，BOSS 50%护甲） ===")
for weapon in weapons:
    dps = weapon.calculate_dps(level=10, crit_rate=0.20, 
                                 monster_armor=0.50, penetration=0.3)
    print(f"{weapon.name:8s}: {dps:8.1f} DPS")

# 模拟战斗场景
def simulate_battle(weapons_list, monster_hp, duration=10):
    """模拟战斗，计算总伤害"""
    total_dps = sum([w.calculate_dps(level=10, crit_rate=0.20,
                                      monster_armor=0.50, penetration=0.3) 
                    for w in weapons_list])
    total_damage = total_dps * duration
    kills = int(total_damage / monster_hp)
    
    print(f"\n=== 组合DPS: {total_dps:.1f} ===")
    print(f"战斗{duration}秒总伤害: {total_damage:.0f}")
    print(f"击杀怪物（HP {monster_hp}）: {kills} 只")
    print(f"剩余HP: {total_damage - kills * monster_hp:.0f}")

# 场景1：BOSS战
print("\n" + "="*50)
print("场景1：BOSS战（HP 10,000，护甲50%）")
simulate_battle([weapons[0], weapons[0], weapons[1]], 10000, duration=5)

# 场景2：僵尸群
print("\n" + "="*50)
print("场景2：僵尸群（HP 500，无护甲）")
simulate_battle([weapons[3], weapons[4], weapons[0]], 500, duration=5)
