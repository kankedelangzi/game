一、CC套（宫廷套装）基础属性总览

📦 CC套 = 宫廷套装（Costume Set）

CC套是DNF 70版本中一套以基础属性加成为核心的时装套装，不包含技能等级加成，而是直接提供面板属性的大幅提升。其核心特点是6件阈值机制：仅当装备6件时套装效果才生效，1~5件仅享受单件属性。

🔑 阈值机制说明：CC套是DNF中少有的"6件阈值"时装套装。装备1~5件时，仅享受单件属性（力量约260、物理攻击约90、独立攻击约100、暴击约2.5%），套装加成完全不生效。只有装备6件时，套装效果才完整激活，额外提供力量+310、物理攻击+110、独立攻击+120、暴击+3%的加成。

二、伤害公式回顾与收益计算模型

2.1 固伤技能公式（狂战士核心）

伤害 = 技能基数 × (1 + 独立攻击 / 250) × 防御减伤 × 暴击期望
来源：DNF 70版本伤害公式体系 · 任务11-15验证

固伤技能不受力量、物理攻击影响，仅受独立攻击、技能倍率、E2/E3加成、防御减伤、暴击影响。

2.2 百分比技能公式（剑魂核心）

伤害 = 物理攻击 × 技能倍率 × (1 + 力量 / 250) × 防御减伤 × 暴击期望
来源：DNF 70版本伤害公式体系 · 任务17验证

2.3 CC套收益计算模型

固伤收益（独立攻击）：
收益比 = (1 + (独立 + 120) / 250) / (1 + 独立 / 250) - 1

百分比收益（力量）：
收益比 = (1 + (力量 + 310) / 250) / (1 + 力量 / 250) - 1

百分比收益（物理攻击）：
收益比 = (1 + (物理攻击 + 110) / 2500) / (1 + 物理攻击 / 2500) - 1
注：DNF 70版本物理攻击系数分母标准值为2500，本分析采用2500计算

暴击收益：
收益比 = [(1 - 暴击率_new) + 暴击率_new × 1.5] / [(1 - 暴击率_old) + 暴击率_old × 1.5] - 1
其中暴击率_new = 暴击率_old + 3%，暴击伤害倍率 = 1.5（DNF 70标准）

4.3 破极兵刃协同效应

🔥 破极兵刃 × CC套 协同分析

破极兵刃提供物理攻击+30%（Lv10），CC套提供物理攻击+110。两者在物理攻击乘区叠加：

  • 破极兵刃：2000 → 2600（+30%）
  • CC套：2600 → 2710（+2.2%，使用完整系数比公式）
  • 协同总物理攻击：2710（相比无CC套的2600，实际提升+2.2%）

注意：CC套的物理攻击加成是固定数值，在破极兵刃的百分比加成之后叠加，因此实际收益略低于基础面板下的收益。

📌 核心结论

  1. CC套6件套阈值机制：仅6件激活套装效果，1~5件仅享受单件属性
  2. 6件套属性：力量+310、物理攻击+110、独立攻击+120、暴击+3%
  3. 狂战士收益：固伤+9.3%（独立+暴击），百分比+36.50%（力量+物理攻击+暴击）
  4. 剑魂收益：百分比+41.1%（力量+物理攻击+暴击），破极兵刃协同额外+2.2%物理攻击
  5. 最优搭配：E2 6件 + CC套2件（上衣+下装技能加成），兼顾技能等级与套装效果
  6. 收益差异：CC套对百分比职业（剑魂+41.1%）收益远高于固伤职业（狂战士固伤+9.3%），差距约4.4倍
