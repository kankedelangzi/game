## 三、关键公式验证

### 3.1 独立攻击收益公式
```
收益比 = (1 + (独立+120)/250) / (1 + 独立/250) - 1
      = (1 + 1370/250) / (1 + 1250/250) - 1
      = 6.48 / 6.0 - 1
      = +8.00%
```

### 3.2 暴击期望收益公式
```
暴击期望 = (1 - 暴击率) + 暴击率 × 1.5
收益比 = 暴击期望_new / 暴击期望_old - 1
      = 1.29 / 1.275 - 1
      = +1.18%
```

### 3.3 百分比力量收益公式
```
收益比 = (1 + (力量+310)/250) / (1 + 力量/250) - 1
      = (1 + 1329.2/250) / (1 + 1019.2/250) - 1
      = 6.3168 / 5.0768 - 1
      = +24.42%
```

### 3.4 物理攻击收益公式（v199修正）
```
收益比 = (物理攻击+110) / 物理攻击 - 1
      = 2110 / 2000 - 1
      = +5.50%
```
