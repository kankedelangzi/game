📐 宠物加成计算公式

宠物力量加成 = 基础力量 × (1 + 宠物力量%)

示例（面板力量728，宠物力量+10%）：
  宠物后力量 = 728 × 1.10 = 800.8
  净增 = 800.8 - 728 = 72.8点

百分比技能伤害增幅：
  增幅 = (1 + 800.8/250) / (1 + 728/250) - 1
  = 4.203 / 3.912 - 1
  = +7.4%

固伤技能：不受宠物力量影响 ❌

📐 称号加成计算

称号全属性加成 = 基础属性 × (1 + 全属性%)

示例（面板力量728，称号全属性+3%）：
  称号后力量 = 728 × 1.03 = 749.8
  净增 = 749.8 - 728 = 21.8点

百分比技能伤害增幅：
  增幅 = (1 + 749.8/250) / (1 + 728/250) - 1
  = 3.999 / 3.912 - 1
  = +2.2%

称号属强加成（独立乘区）：
  增幅 = (220 + 新增属强) / (220 + 原属强) - 1
  示例：属强+15 → (220+15)/(220+0) - 1 = +6.8%

六、加成乘区验证

📐 完整伤害公式（含宠物/称号/勋章）

百分比最终伤害 = 
  武器物理攻击 × 强化系数 × 精通系数 × 
  破极兵刃系数 × 
  (1 + 力量/250) × 
  (1 + 属强/220) × 
  防御减伤 × 暴击期望

其中：
  力量 = 基础力量 + 装备力量 + 宝石力量 + 时装力量 + 
         宠物力量 + 称号力量 + 勋章力量
  
  属强 = 基础属强 + 宝石属强 + 宠物属强 + 称号属强

示例（剑魂巨剑流，破极兵刃中）：
  基础物理攻击 = 712 × 1.36 × 1.10 = 1065
  破极兵刃后 = 1065 × 1.30 = 1385
  力量 = 600 + 200 + 80 + 44 + 20 + 20 + 111 = 1075
  属强 = 0 + 60 + 10 + 15 = 85
  
  最终倍率 = 1385 × (1 + 1075/250) × (1 + 85/220)
           = 1385 × 5.30 × 1.386
           = 10,186（崩山裂地斩Lv20，750%倍率）
