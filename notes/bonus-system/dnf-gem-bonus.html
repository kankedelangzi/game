1.3 属性石收益分析

📐 收益计算公式
力量收益（百分比技能）：
伤害增幅 = (1 + 新增力量/250) / (1 + 原力量/250) - 1

属性强化收益：
伤害增幅 = (220 + 新增属强) / (220 + 原属强) - 1

暴击率收益：
暴击期望 = 暴击率 × 1.5 + (1 - 暴击率) × 1.0 = 1 + 0.5 × 暴击率

2.3 强化收益曲线分析

📈 边际收益递减

强化等级的边际收益呈递减趋势：

每级强化收益：
+1~+5：每级 +2%~10%，成本极低，必做
+6~+7：每级 +12%~15%，成功率下降，性价比仍高
+8~+9：每级 +18%~22%，成功率60~70%，性价比中等
+10~+13：每级 +26%~42%，成功率骤降至20~50%，性价比急剧下降

结论：+7 是性价比拐点，+9 是风险拐点

4.2 边际收益分析

关键发现：
1. +0→+7 强化：每级平均 +2.4% 伤害，成本极低，性价比最高
2. +7→+9 强化：每级平均 +3.5% 伤害，成本中等，性价比中等
3. +9→+13 强化：每级平均 +4% 伤害，成本极高，性价比低
4. 宝石收益呈线性增长，无递减效应
5. 最优策略：优先满级宝石 + +7武器，再考虑更高强化

🏆 五、总结与建议

📌 宝石优先级

1. 属性强化宝石收益最高
2. 力量/智力宝石稳定收益
3. 暴击宝石条件收益
4. 其他宝石收益有限

📌 强化优先级

1. +0→+5必做（100%成功）
2. +5→+7建议（成功率>80%）
3. +7→+9视经济情况
4. +9→+13高风险高回报

📌 职业差异

百分比职业强化收益高
固伤职业强化收益低
混合职业平衡配置
属强收益全职业通用
