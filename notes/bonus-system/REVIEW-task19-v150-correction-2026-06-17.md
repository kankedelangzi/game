### ❌ 严重错误：狂战士百分比技能计算基准错误

**问题描述**：
HTML报告中狂战士百分比技能（崩山裂地斩、嗜魂封魔斩）的力量收益计算使用了错误的基础力量值。

**错误计算**：
```
力量收益 = (1 + 1038/250) / (1 + 728/250) - 1 = 31.70%
```
使用基础力量728作为基准。

**正确计算**：
```
力量收益 = (1 + 1329.2/250) / (1 + 1019.2/250) - 1 = 24.42%
```
使用暴走后力量1019.2作为基准（728 × 1.40）。

**影响分析**：
- 狂战士百分比力量收益：31.70% → 24.42%（差异-7.53pp）
- 狂战士百分比综合收益：36.50% → 28.97%（差异-7.53pp）
- 原因：崩山裂地斩和嗜魂封魔斩是百分比技能，在暴走状态下使用

**修正状态**：✅ 已修正
