## 🔍 详细分析

### 一、berserker-bleed-mechanism.html — 🔴 3项严重问题持续未修正

**问题1：出血等级Lv23 → 应为Lv21**

| 项目 | bleed-mechanism.html（错误） | 正确值（BERSERKER.md） | 差异 |
|------|----------------------------|----------------------|------|
| 出血等级 | Lv23 | **Lv21** | -2级 |
| 基础出血/秒 | 4,200 | **3,800** | +10.5% |
| 无Buff出血/秒 | 5,460 | **4,940** | +10.5% |
| E2出血/秒 | 5,880 | **5,320** | +10.5% |
| E2+E3出血/秒 | 6,510 | **5,890** | +10.5% |
| 满配总伤害 | 32,550 | **29,450** | +10.3% |

**问题2：出血等级计算逻辑错误**

文件声称（第106行）：
> "Lv23出血 = 技能Lv13（基础3,640）+ 血气分流Lv10（+5级）= Lv23"

**错误分析**：
- 血气分流是**暴击率被动技能**，满级Lv10增加暴击率+20%，**不增加出血等级**
- 出血等级来源：十字斩等级 + 出血精通 + 血气爆发
- 正确计算：十字斩Lv18 + 出血精通(+2) + 血气爆发(+1) = **Lv21**
- E2血之狂暴+3作用于十字斩等级：Lv18 + 3 = Lv21（出血等级同步为Lv21）

**问题3：数学计算错误**

第52行核心结论速览：
```
Lv23出血/秒伤害 = 5544
```

验算：
- 基础出血/秒 = 4,200
- +出血精通Lv10 = 4,200 × 1.10 = 4,620
- +出血精通 + 血气爆发 = 4,200 × 1.30 = **5,460**
- 文件显示5,544 → **计算错误**（偏差+84，+1.5%）

### 二、berserker-e2-bonus.html — 🟡 出血数据不一致 + 速览标注问题

**出血计算示例（第125-135行）**：
```
出血/秒 = 4,200 × (1 + 10% + 20% + 10% + 15%) = 4,200 × 1.55 = 6,510/秒
5秒总伤害 = 6,510 × 5 = 32,550
```

**应为**：
```
出血/秒 = 3,800 × (1 + 10% + 20% + 10% + 15%) = 3,800 × 1.55 = 5,890/秒
5秒总伤害 = 5,890 × 5 = 29,450
```

该文件与BERSERKER.md存在跨文件数据矛盾。

**核心结论速览标注问题（第25行）**：
```
+40% 出血伤害加成
```

此标注有歧义：
- E2单项出血加成 = **+10%**
- 出血精通(+10%) + 血气爆发(+20%) + E2(+10%) = **+40%**（累计值）
- 速览应标注E2单项贡献（+10%），而非累计值（+40%），否则易误导读者认为E2单独提供+40%出血加成

### 三、berserker-weapon-comparison.html — 🟡 百分比技能标注不清

百分比技能表格显示的值（如嗜魂封魔斩46,000/49,800）被标注为"基础伤害"，但连招总伤害使用的是暴走后值（64,400/76,335）。表格值与连招值在同一页面混用，易造成读者混淆。

### 四、数据来源标注 — 🟡 可追溯性不足
