```html
<td>650 <span class="source">[来源]</span></td>
```

## 🔍 审核机制（2026-05-29 新增）

### 审核流程

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  主Agent    │────▶│  审核Agent  │────▶│  修正/确认  │
│  (写作)     │     │ (交叉验证)  │     │             │
└─────────────┘     └─────────────┘     └─────────────┘
     │                    │
     │  输出HTML/MD       │  独立数据源验证
     │  (标注来源)        │  (另一套来源)
     ▼                    ▼
  用户可见              审核报告
```

### 审核Agent配置

| 项目 | 说明 |
|------|------|
| 触发方式 | Cron定时任务（每完成3个任务触发一次） |
| 审核范围 | 最近3个任务的输出文件 |
| 数据源 | 与主Agent不同的独立来源 |
| 输出 | 审核报告（标注问题/确认无误） |

### 审核数据源（独立于主Agent）

| 主Agent来源 | 审核Agent来源 |
|-------------|---------------|
| DNF Wiki | NGA DNF专区精品帖 |
| 社区数据库 | 游戏内截图验证 |
| 玩家攻略 | 官方公告/更新日志 |
