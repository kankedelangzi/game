## ⚠️ 数据来源强制规则（2026-05-29 新增）

**因发现杜撰数据问题，即日起严格执行以下规则：**

### 📌 规则1：所有数据必须标注来源

| 数据类型 | 要求 |
|----------|------|
| 武器属性（攻击/独立/力量等） | 必须标注来源（Wiki/NGA/游戏内截图） |
| 技能倍率/伤害数值 | 必须标注来源，禁止估算 |
| 套装加成数值 | 必须标注来源 |
| 公式推导 | 必须标注公式来源（官方文档/社区验证） |

### 📌 规则2：引用格式

在HTML/Markdown中，所有数据必须附带引用标记：

```markdown
| 属性 | 数值 | 来源 |
|------|------|------|
| 物理攻击 | 650 | [DNF Wiki](https://dnf.wiki/...) |
```

或HTML中：
```html
<td>650 <span class="source">[来源]</span></td>
```

### 📌 规则3：禁止行为

| 禁止行为 | 后果 |
|----------|------|
| 编造武器/技能数值 | ❌ 任务标记为"需重做" |
| 估算伤害数值 | ❌ 必须查证或标注"待核实" |
| 无来源的公式推导 | ❌ 必须补充来源 |

### 📌 规则4：自我审核清单

每次输出前必须检查：
- [ ] 所有数值都有来源标注
- [ ] 武器名称与类型匹配（如"屠戮"是太刀不是钝器）
- [ ] 职业定位描述准确（如红眼是混合流派）
- [ ] 公式推导有出处

## 🔍 审核机制（2026-05-29 新增）

### 审核流程

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  主Agent    │────▶│  审核Agent  │────▶│  修正/确认  │
│  (写作)     │     │ (交叉验证)  │     │             │
└─────────────┘     └─────────────┘     └─────────────┘
     │                    │
     │  输出HTML/MD       │  独立数据源验证
     │  (标注来源)        │  (另一套来源)
     ▼                    ▼
  用户可见              审核报告
```

### 审核Agent配置

| 项目 | 说明 |
|------|------|
| 触发方式 | Cron定时任务（每完成3个任务触发一次） |
| 审核范围 | 最近3个任务的输出文件 |
| 数据源 | 与主Agent不同的独立来源 |
| 输出 | 审核报告（标注问题/确认无误） |

### 审核数据源（独立于主Agent）

| 主Agent来源 | 审核Agent来源 |
|-------------|---------------|
| DNF Wiki | NGA DNF专区精品帖 |
| 社区数据库 | 游戏内截图验证 |
| 玩家攻略 | 官方公告/更新日志 |

### 审核报告格式

```markdown
## 审核报告

**审核时间**: 2026-05-29
**审核任务**: 任务15 - 狂战士武器对比

### ✅ 通过项
- 暴走加成数值 +40% — 来源一致 ✅
- E2套装阈值机制 — 来源一致 ✅

### ❌ 问题项
- 武器「屠戮」类型标注为钝器 — 实际为55级SS太刀 ❌
- 技能伤害数值无来源标注 — 需补充 ❌

### 建议修正
1. 修正武器类型
2. 补充所有数值来源
3. 重新计算伤害对比
```
