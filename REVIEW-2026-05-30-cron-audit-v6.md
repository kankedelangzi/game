## 🔧 修正建议

### 1. 🔴 武器强化+10系数修正

**位置**: `notes/02-swordsman-damage-calculation.html` 中"武器强化收益"表格

**当前错误**:
```
| +10 | 26% | +26% |
```

**应修正为**:
```
| +10 | 30% | +30% |
```

**影响**: 该错误导致所有基于+10强化等级的伤害计算偏低约4个百分点。

### 2. 🔴 武器精通倍率加成删除

**位置**: `notes/02-swordsman-damage-calculation.html` 中四种武器精通表格

**当前错误**: 巨剑精通标注"技能倍率加成+10%"，短剑/太刀/手剑标注"+5%"

**应修正为**: 删除"技能倍率加成"行，改为说明"武器精通仅提供武器基础物理攻击力差异"

**依据**: SWORDSMAN.md中明确标注"⚠️ 重要：DNF 70版本武器精通仅提供武器基础物理攻击力差异，不存在统一的'+10%/+5%技能倍率加成'"

**影响**: 虚构的倍率加成会导致伤害计算出现系统性偏差。

### 3. 🔴 极·鬼剑术（暴风式）分类修正

**位置**: `notes/02-swordsman-damage-calculation.html` 核心技能分类表

**当前错误**:
```
极·鬼剑术（暴风式）| 觉醒（被动）| 多段攻击
```

**应修正为**:
```
极·鬼剑术（暴风式）| 主动觉醒 | 多段攻击
```

**依据**: SWORDSMAN.md中已正确标注为"主动觉醒"，NGA DNF专区精品帖确认70版本觉醒技能为主动释放。

**影响**: 分类错误可能导致玩家误解技能使用方式（被动自动触发 vs 主动释放）。

### 4. 🟡 内部数据一致性修正

**问题**: 同一知识库中武器强化+10系数存在两个不同值（26%和30%）

**建议**: 统一所有文件中武器强化+10系数为30%，确保数据一致性。
