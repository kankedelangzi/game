### 📐 正确固伤公式

```
伤害 = 技能基数 × (1 + 独立攻击/250) × E2加成 × 防御减伤 × 暴击期望
```

**重要原则**：
- ✅ 固伤技能受：独立攻击、技能倍率、E2/E3加成、防御减伤、暴击
- ❌ 固伤技能**不受**：力量、物理攻击、属性强化
- ✅ 百分比技能受：力量、物理攻击、武器强化、属性强化

## 出血机制（70版本特色）
- **穿透防御**：出血伤害不受目标物理防御减伤影响（70版本核心机制，80版本后削弱）
- **持续伤害**：出血持续5秒，每秒触发一次，最多叠加5层
- **独立计算**：出血伤害基于出血等级，与面板独立攻击/力量/暴击/属强均无关
- **基础公式**：出血/秒 = 出血基础 × (1 + 出血精通 + 血气爆发 + E2 + E3)
- **出血等级计算**：出血等级 = 十字斩Lv18 + 出血精通(+2) + 血气爆发(+1) = **Lv21**（E2血之狂暴+3作用于十字斩等级，不直接作用于出血等级）
- **Lv21出血**：基础3,800/秒（十字斩Lv18 + E2+3 = Lv21）
  - 无加成：3,800 × 1.0 = 3,800/秒 → 19,000总伤害
  - +出血精通Lv10：3,800 × 1.10 = 4,180/秒 → 20,900
  - +出血精通 + 血气爆发：3,800 × 1.30 = 4,940/秒 → 24,700
  - +E2血之狂暴6件：3,800 × 1.40 = 5,320/秒 → 26,600
  - +E2 + E3血染江山6件：3,800 × 1.55 = 5,890/秒 → 29,450
- **加成来源**：出血精通(+10%) + 血气爆发(+20%) + E2(+10%) + E3(+15%) = 最高+55%
- **穿透验证**：防御10,000时，物理技能减伤2.9%，出血伤害不变，出血/物理比≈2.8倍
- **详细报告**：`notes/berserker-bleed-mechanism.html`（10.9KB）

## 暴走技能深度分析（任务11完成 ✅）
- **加成构成**：Lv10技能 +35% + E2血之狂暴6件 +5% = 总计 +40%
- **等级递增**：Lv1→Lv9 每级+2%，Lv10跳跃至+35%（满级奖励，单级+18%）
- **E2阈值机制**：仅6件激活+5%，1~5件完全不生效
- **固伤影响**：❌ 零影响 — 固伤伤害基于固定数值，与面板力量无关
- **百分比影响**：✅ 约+29.7%（以728力量面板计算：(1+1019/250)/(1+728/250)-1）
- **出血影响**：❌ 零影响 — 出血独立计算，穿透防御
- **整体收益**：约+10~15%（固伤占比约70%，百分比约20%，出血约10%）
- **CD机制**：300秒CD / 30秒持续 = 10%覆盖率，实际有效约5%，需卡CD使用
- **流派差异**：巨剑流（百分比为主）受益+18~25%，钝器流（固伤为主）受益+8~12%
- **力量计算公式**：最终力量 = (基础力量 + 装备力量) × (1 + 暴走加成)
- **版本对比**：60版+25% → 70版+40% → 80版+45% → 85版+50%
- **输出实例**：暴走前力量728 → 暴走后1019，净增291点
- **连招对比**：暴走前连招总伤害≈18.1万 → 暴走后≈21.3万，整体增幅≈+17.7%
- **详细报告**：`notes/berserker-burst-mechanism.html`（12.9KB）

## 异界E2套装（血之狂暴）加成分析（任务13完成 ✅）
- **套装机制**：阈值型加成，仅6件激活，1~5件完全不生效
- **技能加成**：所有核心技能统一+35%（十字斩/崩山击/血气之刃/怒气爆发/嗜魂之手/崩山裂地斩/嗜魂封魔斩）
- **出血加成**：+10%，与出血精通(+10%)、血气爆发(+20%)、E3(+15%)加法叠加，理论最大+55%
- **力量加成**：+5%，与暴走乘法叠加（暴走后982.8 → E2后1032，额外+49.2点）
- **套装属性**：物理攻击+120，力量+290，独立攻击+80，暴击率+4%
- **E2 vs E3对比**：E2技能+35% > E3技能+30%，E2是首选；E3出血+15% > E2出血+10%
- **搭配限制**：E2/E3不跨套叠加，必须集齐同套6件
- **实战实例**：十字斩10,030→13,541（+3511），崩山裂地斩85,000→114,750（+29,750）
- **详细报告**：`notes/berserker-e2-bonus.html`（12.4KB）
