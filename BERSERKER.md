## 正确固伤公式

```
伤害 = 技能基数 × (1 + 独立攻击/250) × E2加成 × 防御减伤 × 暴击期望
```

**重要原则**：
- ✅ 固伤技能受：独立攻击、技能倍率、E2/E3加成、防御减伤、暴击
- ❌ 固伤技能**不受**：力量、物理攻击、属性强化

## 出血机制（70版本特色）
- **穿透防御**：出血伤害不受目标物理防御减伤影响
- **出血等级**：十字斩Lv18 + E2血之狂暴+3 = **Lv21**
- **Lv21出血**：基础3,800/秒，最高加成+55%（出血精通+10% + 血气爆发+20% + E2+10% + E3+15%）
- **穿透验证**：防御10,000时，物理技能减伤2.9%，出血伤害不变
- **详细报告**：`notes/berserker-bleed-mechanism.html`

## 暴走技能深度分析（任务11完成 ✅）
- **加成构成**：Lv10技能 +35% + E2血之狂暴6件 +5% = 总计 +40%
- **等级递增**：Lv1→Lv9 每级+2%，Lv10跳跃至+35%（满级奖励）
- **E2阈值机制**：仅6件激活+5%，1~5件完全不生效
- **固伤影响**：❌ 零影响 — 固伤伤害基于固定数值
- **百分比影响**：✅ 约+29.7%（728力量面板计算）
- **整体收益**：约+10~15%（固伤占比约70%，百分比约20%，出血约10%）
- **详细报告**：`notes/berserker-burst-mechanism.html`

## 装备搭配
- **钝器流（推荐）**：70粉钝器「嗜魂」+12/+13 + E2血之狂暴6件 + 力量首饰
- **巨剑流**：70粉巨剑+12 + 高物理攻击装备，百分比技能收益高
- **毕业面板参考**：力量728（暴走后982.8，E2后1032），独立1250，物理攻击2000，暴击55%

### ⚠️ 重要修正
**「屠戮」是55级SS太刀，不是钝器！**

**核心机制澄清**：
- ✅ 固伤技能：使用**面板独立攻击**，钝器流与巨剑流使用同一面板值 → **固伤伤害相同**
- ✅ 百分比技能：使用力量+物理攻击+武器精通，**不受独立攻击影响**

### 流派差异
| 对比项 | 钝器流 | 巨剑流 |
|--------|--------|--------|
| 核心机制 | 固伤 + 百分比 | 物理攻击 + 武器精通 |
| 固伤技能差异 | 无差异 | 无差异 |
| 百分比技能差异 | 物理攻击低、精通低 | 物理攻击高、精通高 |
| 暴走收益 | +8~12% | +18~25% |
| 强化收益 | 低 | 高 |

### 伤害计算实例（E2 6件，暴走中，面板独立1250）
- **固伤技能**：两者持平（395,702）
- **百分比技能**：巨剑流领先 +6.4%（207,209 vs 220,538）
- **连招总伤害**：巨剑流 616,240 vs 钝器流 602,911 → **+2.2%**

### 最终推荐
**巨剑流（E2 + 裁决）综合略优于钝器流，伤害比 ≈ 1 : 1.022**
- 毕业配置：E2血之狂暴6件 + 70粉巨剑「裁决」+12/+13 + 力量首饰

> 📄 详细对比数据见：`archive/berserker-equipment-v15.md`

## 经典连招
```
嗜魂之手 → 十字斩 → 崩山击 → 血气之刃
总伤害约18.1万（暴击期望），连招时间8-10秒
```

## 属性优先级
- **固伤技能**：独立攻击 > 暴击 > 技能等级
- **百分比技能**：力量 > 物理攻击 > 武器强化 > 暴击
- **通用**：力量 > 独立攻击 > 暴击 > 物理攻击（仅百分比）
