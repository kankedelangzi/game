## 📊 核心技能：暴走（Berserk）

### 基础参数

| 属性 | 数值 |
|------|------|
| 技能类型 | 被动增益（Buff） |
| 最大等级 | Lv10 |
| 持续时间 | 30秒 |
| 冷却时间 | 300秒（5分钟） |
| SP消耗 | 每级30 SP，Lv10共300 SP |
| 前置条件 | 血气唤醒 Lv5 |

### 力量加成曲线

| 等级 | 力量加成 | 单级增幅 | 备注 |
|------|----------|----------|------|
| Lv1 | +2% | +2% | 基础起步 |
| Lv2 | +4% | +2% | 线性增长 |
| Lv3 | +6% | +2% | 线性增长 |
| Lv4 | +8% | +2% | 线性增长 |
| Lv5 | +10% | +2% | 线性增长 |
| Lv6 | +12% | +2% | 线性增长 |
| Lv7 | +14% | +2% | 线性增长 |
| Lv8 | +16% | +2% | 线性增长 |
| Lv9 | +17% | +1% | 增幅收窄 |
| **Lv10** | **+35%** | **+18%** | **满级跳跃奖励** |

> ⚠️ Lv1→Lv9 每级+2%，Lv9→Lv10 跳跃+18%，这是70版本暴走的核心设计——鼓励玩家务必点满Lv10

### 总加成公式

```
暴走总加成 = 技能等级加成 + E2套装加成
         = 35%（Lv10） + 5%（E2 6件）
         = 40%
```

### E2套装阈值机制

| E2血之狂暴件数 | 力量加成 | 触发条件 |
|----------------|----------|----------|
| 1~5件 | +0% | ❌ 不生效 |
| **6件** | **+5%** | ✅ 激活 |

> 🔑 E2套装的+5%力量加成是**阈值型**——只有集齐6件才激活，1~5件完全不生效。

## 💪 力量计算

### 暴走前后对比（728力量面板）

| 项目 | 暴走前 | 暴走后 | 变化 |
|------|--------|--------|------|
| 基础力量 | 350 | 350 | — |
| 装备力量 | 378 | 378 | — |
| 面板总力量 | 728 | 728 | — |
| **暴走加成** | — | **+40%** | **+40%** |
| **最终力量** | **728** | **1019** | **+291** |

```
最终力量 = (基础力量 + 装备力量) × (1 + 暴走加成)
         = (350 + 378) × 1.40
         = 728 × 1.40
         = 1019.2 ≈ 1019
```

## ⚔️ 对不同伤害类型的影响

### 百分比技能（巨剑流核心）

```
物理攻击 = 武器攻击 + (力量 × 武器攻击系数 / 250)

暴走前：728 力量 → 物理攻击 ≈ 2000
暴走后：1019 力量 → 物理攻击 ≈ 2342

物理攻击增幅 = (2342 - 2000) / 2000 ≈ +17.1%
```

**精确收益计算：**
```
百分比技能收益 = (1 + 暴走后力量/250) / (1 + 暴走前力量/250) - 1
              = (1 + 1019/250) / (1 + 728/250) - 1
              = 5.076 / 3.912 - 1
              = +29.7%
```

> 📌 这是**力量面板的边际收益递减**效应——力量越高，每点力量带来的百分比技能增幅越小。

### 固伤技能（钝器流核心）

| 技能 | 类型 | 暴走影响 | 原因 |
|------|------|----------|------|
| 十字斩 | 物理固伤 | ❌ 零收益 | 固伤基于固定数值 |
| 崩山击 | 物理固伤 | ❌ 零收益 | 固伤基于固定数值 |
| 血气之刃 | 物理固伤 | ❌ 零收益 | 固伤基于固定数值 |
| 怒气爆发 | 物理固伤 | ❌ 零收益 | 固伤基于固定数值 |
| 嗜魂之手 | 物理固伤 | ❌ 零收益 | 固伤基于固定数值 |

```
固伤伤害 = 基础数值 × (1 + 独立攻击/250) × 技能倍率

⚠️ 公式中不含"力量"变量，因此暴走对固伤技能完全无效
```

### 出血伤害

| 项目 | 数值 | 暴走影响 |
|------|------|----------|
| 出血基础伤害 | 4200/秒 | ❌ 零影响 |
| 出血精通加成 | +10% | ❌ 零影响 |
| 血气爆发加成 | +20% | ❌ 零影响 |
| **Lv23出血总伤害** | **5544/秒** | **❌ 零影响** |
| 出血持续时间 | 5秒 | — |
| **总出血伤害** | **27,720** | **❌ 零影响** |

> 🔑 出血伤害**穿透物理防御**且**独立计算**，与面板力量、独立攻击均无关。暴走对其无任何影响。

## 🗡️ 流派差异：巨剑流 vs 钝器流

| 指标 | 钝器流（固伤为主） | 巨剑流（百分比为主） |
|------|-------------------|---------------------|
| 核心输出技能 | 十字斩、崩山击、血气之刃 | 崩山裂地斩、嗜魂封魔斩 |
| 固伤技能占比 | 约 70% | 约 30% |
| 百分比技能占比 | 约 20% | 约 60% |
| 出血伤害占比 | 约 10% | 约 10% |
| **暴走总收益** | **+8 ~ +12%** | **+18 ~ +25%** |

> 📊 巨剑流受益于暴走的幅度是钝器流的**近2倍**。这也是为什么70版本末期高端玩家更倾向巨剑流的原因之一。

## ⏱️ CD覆盖率与实战策略

| 参数 | 数值 |
|------|------|
| 持续时间 | 30秒 |
| 冷却时间 | 300秒（5分钟） |
| 理论覆盖率 | 30 / 300 = **10%** |
| 实际有效覆盖率 | 约 **5%**（考虑换怪/跑图损耗） |

### 使用策略建议

- **BOSS战**：开场即开暴走，覆盖BOSS血量最厚阶段
- **小怪图**：卡CD使用，确保每波小怪都有暴走覆盖
- **副本竞速**：与队友协调，确保关键BOSS阶段暴走可用
- **不要浪费**：暴走30秒内尽量打出所有高伤害技能（血气之刃、怒气爆发等）
