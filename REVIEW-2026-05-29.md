## 🔧 修正建议

### 1. 崩山裂地斩伤害计算（严重）

**问题**: `02-berserker-damage-calculation.html` 中崩山裂地斩暴击期望仅24,858，但武器对比文件中为82,000~88,500，相差约3.5倍。

**原因分析**:
- 02文件计算中：防御减伤=81,216×0.231=18,761
- 此计算错误地将"百分比技能受防御减伤"与"固伤穿透防御"混淆
- 崩山裂地斩是**百分比技能**，确实受防御减伤，但计算过程有误

**修正建议**:
```
崩山裂地斩正确计算：
基础伤害 = 武器物理攻击 × 倍率 = 2000 × 800% = 16,000
力量加成 = 16,000 × (1 + 1019/250) = 16,000 × 5.076 = 81,216
防御减伤（假设怪物防御3000，防御系数≈0.23）= 81,216 × 0.77 = 62,536
暴击期望 = 62,536 × 1.325 = 82,860

修正后数值应与武器对比文件一致（约82,000~88,500）
```

### 2. 连招总伤害一致性（严重）

- 建议重新计算并统一两文件数值

### 3. 固伤技能伤害差异（严重）

**问题**: 武器对比文件中固伤技能钝器vs巨剑差异标注为+11.1%，但实际计算：
- 钝器独立1200 → 系数6.0
- 巨剑独立1180 → 系数5.92
- 差异 = 6.0/5.92 - 1 = +1.3%，非+11.1%

**修正建议**:
```
固伤技能伤害差异应修正为：
十字斩：钝器60,180 vs 巨剑59,350 → 差异+1.4%
崩山击：钝器43,896 vs 巨剑43,294 → 差异+1.4%
...
差异约+1.3%~+1.4%，非+11.1%
```

### 4. 宝石属强收益计算（严重）

**问题**: 四级属强宝石×4显示收益+26.7%，但正确计算：
- 四级宝石每个+40属强，4个=+160属强
- 收益 = (220+160)/(220+0) - 1 = 380/220 - 1 = +72.7%

**修正建议**:
```
修正属强收益计算：
- 四级属强×4：+160属强，收益约+72.7%（无属强基础）
- 若有基础属强（如+50），收益 = (220+210)/(220+50) - 1 = +59.3%
```

### 5. 暴击宝石收益计算（严重）

**问题**: 四级暴击×4显示收益+8%，但实际：
- 从55%暴击率增加到71%
- 暴击期望从1.275增加到1.355
- 增幅 = 1.355/1.275 - 1 = +6.3%

**修正建议**:
```
修正暴击收益计算：
- 四级暴击×4：+16%暴击率
- 55%→71%，期望伤害增幅约+6.3%
- 若从30%→46%，增幅约+12%
```

### 6. 数据来源标注（中等）
